## 這是什麼？

這是一份給**沒有人資背景的人**看的術語翻譯手冊。

如果你是剛出社會的新鮮人、轉職中的上班族，或任何聽到 PIP、Onboarding、RSU 就一頭霧水的人——這裡就是你的救星。

每個術語都有：
- **白話解釋**：不用任何專業背景就能懂
- **生活比喻**：用你熟悉的事物來對照
- **造句示範**：讓你知道這個詞在什麼情境下會出現

## 招募與入職類

> 從看到職缺、收到錄取、報到第一天到試用期——找工作必懂的詞。

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| JD（Job Description） | 職缺說明書，寫清楚這個工作要做什麼 | 餐廳徵人啟事上寫「要會炒菜、會切菜」 | 求職網站、HR 開缺 | [詳細](terms/jd.md) |
| Offer Letter（錄取通知） | 公司正式跟你說「你被錄取了」的那封信 | 大學放榜通知書 | 面試結束後、談薪階段 | [詳細](terms/offer-letter.md) |
| Onboarding（入職引導） | 新人報到後帶你認識環境的流程 | 轉學第一天有同學帶你認識校園 | 報到第一週、HR 安排 | [詳細](terms/onboarding.md) |
| Probation（試用期） | 正式錄取前先試做幾個月看合不合 | 交往先試試看合不合，再決定在一起 | Offer Letter 裡、入職說明 | [詳細](terms/probation.md) |

## 績效與管理類

> 在公司裡待久了就會碰到的詞——跟表現、主管、團隊有關。

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| 1-on-1（一對一面談） | 主管跟你私下聊工作狀況的定期會議 | 導師約你喝咖啡聊近況 | 每週/每兩週的行事曆 | [詳細](terms/one-on-one.md) |
| PIP（績效改善計畫） | 表現不好時公司給你的最後通牒改善方案 | 考試不及格被要求補考 | HR 通知、績效面談 | [詳細](terms/pip.md) |
| Turnover Rate（離職率） | 一段時間內有多少人離開公司 | 餐廳服務生換了幾輪 | HR 報表、管理會議 | [詳細](terms/turnover-rate.md) |
| Headcount（編制人數） | 公司目前有多少人、還能再招多少人 | 教室的座位數——滿了就不能再收學生 | 年度預算、招募計畫 | [詳細](terms/headcount.md) |

## 薪酬與工作型態類

> 關於薪水怎麼算、在哪上班、怎麼工作——影響你生活品質的詞。

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| RSU / Stock Option | 公司用股票當獎勵，但要待滿一段時間才能拿 | 集點卡集滿才能換禮物 | 薪資談判、Offer Letter | [詳細](terms/rsu-stock-option.md) |
| Burnout（職業倦怠） | 工作太累太久，身心都燒乾了 | 手機電池用太久開始膨脹 | 同事閒聊、健康檢查 | [詳細](terms/burnout.md) |
| Remote / Hybrid（遠端 / 混合工作） | 不用每天到辦公室，在家也能上班 | 有時在家煮飯，有時出去吃 | 職缺描述、面試問答 | [詳細](terms/remote-hybrid.md) |
| Freelancer / Contractor（自由工作者 / 約聘） | 不是正職，按專案或時間收費的工作者 | 叫外送 vs 請專屬廚師 | 外包合作、人力規劃 | [詳細](terms/freelancer-contractor.md) |

## 比喻對照總表

> 一次看完所有比喻，讓你對人資術語建立直覺。

### 招募與入職類

| 術語 | 比喻 |
|------|------|
| JD | 餐廳徵人啟事，上面寫「要會炒菜、會切菜、月休八天」 |
| Offer Letter | 大學放榜通知書，告訴你錄取了、什麼時候報到 |
| Onboarding | 轉學第一天有同學帶你認識校園，告訴你福利社在哪 |
| Probation | 交往先試試看合不合，再決定要不要正式在一起 |

### 績效與管理類

| 術語 | 比喻 |
|------|------|
| 1-on-1 | 導師約你喝咖啡聊近況，不是考試，是關心你 |
| PIP | 考試不及格被要求補考，補考沒過就真的被當了 |
| Turnover Rate | 餐廳服務生半年換了三批，代表工作環境可能有問題 |
| Headcount | 教室座位數——40 個位子坐滿了，就不能再收學生 |

### 薪酬與工作型態類

| 術語 | 比喻 |
|------|------|
| RSU / Stock Option | 集點卡集滿才能換禮物，中途離開就白集了 |
| Burnout | 手機電池用太久膨脹——不是手機壞了，是逼太久沒休息 |
| Remote / Hybrid | 有時在家煮飯、有時出去吃，看心情和需要 |
| Freelancer / Contractor | 叫外送（Freelancer）vs 請一個短期廚師幫忙（Contractor） |
