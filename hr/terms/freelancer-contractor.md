# Freelancer / Contractor（自由工作者 / 約聘）— 不是公司正職，按專案或時間收費的工作者

> **白話說：** Freelancer 和 Contractor 就像**叫外送 vs 請專屬廚師**——你不用養一個全職員工，有需要的時候才找人來做。

---

## 它到底是什麼？

這兩個詞都是指「不是公司正職」的工作型態，但有點不一樣：

### Freelancer（自由工作者 / 自由接案者）

- 自己當老闆，同時接好幾個客戶的案子
- 通常按專案收費（例如：做一個網站 5 萬元）
- 自己決定工作時間和地點
- 沒有固定薪水、沒有勞健保（要自己處理）
- 常見的 Freelancer 類型：設計師、攝影師、寫手、翻譯、工程師

### Contractor（約聘 / 承攬人員）

- 跟一間公司簽約，但不是正職員工
- 通常按時間收費（例如：時薪 800 元）或按固定期間（例如：半年合約）
- 有些 Contractor 會到公司上班，外表看起來跟正職差不多
- 合約到期就結束，不一定會續約
- 在台灣常聽到的「派遣」也是類似的概念

簡單的差異：

| | Freelancer | Contractor |
|---|---|---|
| 工作對象 | 同時接很多客戶 | 通常專注一間公司 |
| 收費方式 | 按專案 | 按時間或固定合約 |
| 工作地點 | 自己決定 | 常常在客戶公司 |
| 感覺像 | 開自己的小店 | 被借調過去的員工 |

## 生活比喻 / 實際例子

- **正職員工** = 你家請了一個專屬廚師，每天煮三餐，付月薪
- **Freelancer** = 你需要辦派對時找外燴，做完就走，下次有需要再找
- **Contractor** = 你家裝潢，請水電師傅來做三個月，做完合約就結束

**造句**：「我們這個專案的設計是外包給一個 **Freelancer** 做的，他同時還在接其他公司的案子。」

## 為什麼要知道這個詞？

- 越來越多人選擇當 Freelancer，享受時間自由但要自負盈虧
- 公司用 Contractor 可以彈性調整人力，不用承擔長期的人事成本
- 求職時要搞清楚職位是正職還是約聘——薪資福利差很多
- 如果公司大量使用 Contractor 取代正職，可能是在控制人事成本
