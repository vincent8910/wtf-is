# Remote / Hybrid（遠端 / 混合工作）— 不用每天到辦公室，在家或咖啡廳也能上班

> **白話說：** Remote 就像**在家煮飯**，Hybrid 就像**有時在家煮、有時出去吃**——工作不一定要在辦公室完成。

---

## 它到底是什麼？

這兩個詞描述的是不同的「上班地點」模式：

### Remote（遠端工作）

完全不用進辦公室，在哪裡工作都可以：

- 在家、在咖啡廳、在共享空間，甚至在另一個城市
- 所有溝通都靠線上工具（Slack、Zoom、Email）
- 有些公司是 Full Remote（全遠端），根本沒有實體辦公室

### Hybrid（混合工作）

一部分時間在辦公室，一部分時間遠端：

- 最常見的是「每週進辦公室 2-3 天，其他天在家」
- 有些公司是固定哪幾天要到（例如週二、週四必須進辦公室）
- 有些公司是彈性的，自己決定哪天要進去

疫情之後，Hybrid 成為最多公司採用的模式。完全 Remote 的機會雖然有，但很多公司還是希望員工偶爾能面對面互動。

也有一種叫 **Remote-First**，意思是雖然有辦公室，但公司的工作流程是以遠端為主設計的，進不進辦公室都可以。

## 生活比喻 / 實際例子

想像你吃飯的方式：

- **到辦公室上班**（On-site）= 每天去餐廳吃，不能外帶
- **Remote** = 在家煮飯或叫外送，不用出門
- **Hybrid** = 有時候去餐廳、有時候在家煮——看心情和需要

**造句**：「這間公司是 **Hybrid** 制，每週三和週五要進辦公室，其他天可以 **Remote**，我覺得滿彈性的。」

## 為什麼要知道這個詞？

- 找工作時，工作模式（Remote / Hybrid / On-site）是很多人最在意的條件之一
- 不同模式各有優缺點：Remote 自由但容易孤獨、Hybrid 彈性但通勤麻煩
- 面試時一定要問清楚：「是真的 Hybrid 還是名義上 Hybrid 但主管希望你天天來？」
- 如果是 Remote 職位，要確認是否有時區限制（有些公司要求跟總部同時區上班）
