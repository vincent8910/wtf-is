# Working Capital（營運資金） — 公司手上可以馬上拿來用的錢，扣掉短期要還的債

> **白話說：** 就像你的**零用金**——你口袋裡有多少現金，扣掉這個月一定要付的帳單（房租、水電、信用卡），剩下的才是你「可以靈活運用」的錢。

---

## 它到底是什麼？

**Working Capital（營運資金）** 的算法很簡單：

```
營運資金 = 流動資產 - 流動負債
```

| 流動資產（短期能變現的） | 流動負債（短期要付的） |
|---|---|
| 現金 | 應付帳款 |
| 應收帳款 | 短期借款 |
| 存貨 | 要付的稅 |
| 短期投資 | 一年內到期的長期負債 |

營運資金 > 0 → 公司短期內不會缺錢，還得出來
營運資金 < 0 → 危險！可能會周轉不靈

## 生活比喻 / 實際例子

你的「個人營運資金」：

```
你口袋裡有：
├─ 活存     50,000
├─ 別人欠你  10,000
└─ 合計流動資產  60,000

這個月一定要付的：
├─ 房租     12,000
├─ 信用卡    8,000
├─ 水電瓦斯   3,000
└─ 合計流動負債  23,000

你的營運資金 = 60,000 - 23,000 = 37,000
→ 還有 37,000 可以靈活運用，OK！
```

但如果你口袋只有 20,000，要付的帳單卻有 23,000 → 營運資金是 -3,000，你這個月會周轉不靈。

**造句**：
- 「**Working Capital** 不夠，先去跟銀行借週轉金」
- 「[應收帳款](accounts-receivable-payable.md)收太慢，**營運資金** 壓力很大」
- 「做好 **營運資金** 管理，公司才不會突然倒閉」

## 為什麼要知道這個詞？

- 很多公司不是「賠錢倒閉」而是「周轉不靈倒閉」——帳面上有賺，但手上沒現金
- 營運資金管理的核心：趕快把[應收帳款](accounts-receivable-payable.md)收回來、盡量延後應付帳款的付款時間
- [現金流量表](cash-flow-statement.md)上會看到營運資金的變化
- 跟[損益平衡點](break-even-point.md)的關係：即使已經損益兩平，營運資金不足一樣會出事
