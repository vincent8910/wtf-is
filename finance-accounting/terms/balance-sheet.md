# Balance Sheet（資產負債表） — 某個時間點，公司有什麼、欠什麼、股東剩多少

> **白話說：** 就像你打開記帳 App 看「我現在有多少存款、欠多少卡債、真正的淨資產是多少」——資產負債表就是公司版的這張快照。

---

## 它到底是什麼？

**Balance Sheet（資產負債表）** 是[三大報表](financial-statement.md)之一，它拍的是公司在**某一天**的財務狀態。

核心公式超簡單：

```
資產 = 負債 + 股東權益
（我有的東西 = 我借別人的 + 真正屬於我的）
```

| 項目 | 白話說 | 例子 |
|------|--------|------|
| 資產（Assets） | 公司擁有的所有值錢的東西 | 現金、辦公室、應收帳款、設備 |
| 負債（Liabilities） | 公司欠別人的錢 | 銀行貸款、應付帳款、預收款項 |
| 股東權益（Equity） | 扣掉欠的，真正屬於股東的 | 資本額 + 歷年累積賺的錢 |

## 生活比喻 / 實際例子

想像你自己的財務狀況：

```
你的「個人資產負債表」：

資產（你有的）：
├─ 銀行存款      300,000
├─ 機車          50,000
├─ 股票          100,000
└─ 合計          450,000

負債（你欠的）：
├─ 學貸          150,000
├─ 信用卡帳單     20,000
└─ 合計          170,000

淨資產（真正屬於你的）：
└─ 450,000 - 170,000 = 280,000
```

**造句**：
- 「看一下 **Balance Sheet**，公司帳上現金夠不夠」
- 「**資產負債表** 上負債太高了，要注意」
- 「投資人會看 **Balance Sheet** 來判斷公司的財務體質」

## 為什麼要知道這個詞？

- 跟[損益表](income-statement.md)不同，資產負債表看的是「某一天的狀態」而不是「一段時間的流量」
- 想知道公司是不是「資不抵債」（欠的比有的多），看這張表就知道
- 銀行決定要不要借錢給公司，第一個就看 Balance Sheet
- 跟[現金流量表](cash-flow-statement.md)搭配看，才能完整了解公司的財務狀況
