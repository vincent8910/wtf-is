# Break-even Point（損益平衡點） — 收入剛好等於支出，不賺不賠的那個點

> **白話說：** 就像你擺攤賣雞排，攤位費一天 2,000 元——你要賣到第 40 份（每份賺 50 元）才打平，第 41 份開始才是真的在賺錢。那個「第 40 份」就是損益平衡點。

---

## 它到底是什麼？

**Break-even Point（損益平衡點、BEP）** 是讓[營收](revenue.md)剛好等於總成本的那個銷售量或金額。

```
Break-even Point = 固定成本 ÷ 每單位貢獻利潤

固定成本：不管賣不賣得出去都要付的（租金、薪水）
變動成本：賣越多花越多的（原料、包裝）
貢獻利潤：售價 - 單位變動成本
```

| 狀態 | 意思 |
|------|------|
| 營收 < 損益平衡點 | 還在虧損 |
| 營收 = 損益平衡點 | 剛好打平，不賺不賠 |
| 營收 > 損益平衡點 | 開始賺錢了 |

## 生活比喻 / 實際例子

你開了一間手搖飲料店：

```
固定成本（每月不管賣多少都要付的）：
├─ 房租          30,000
├─ 員工薪水       60,000
├─ 水電雜費       10,000
└─ 合計          100,000

每杯飲料：
├─ 售價 60 元
├─ 材料成本 20 元（變動成本）
└─ 每杯賺 40 元（貢獻利潤）

Break-even Point = 100,000 ÷ 40 = 2,500 杯

→ 每個月要賣 2,500 杯才打平
→ 每天大約要賣 83 杯（假設一個月 30 天）
→ 第 2,501 杯開始才是真正的獲利
```

**造句**：
- 「我們的 **Break-even Point** 大概在月營收 300 萬」
- 「新產品預估半年可以到 **損益平衡**」
- 「還沒達到 **BEP**，我們還在燒錢」

## 為什麼要知道這個詞？

- 開公司或開店之前，一定要先算損益平衡點——知道要賣多少才不虧
- 投資人會問「你什麼時候能達到 break-even」——意思是「你什麼時候不再虧錢」
- 定價策略的重要參考：如果售價太低，可能永遠到不了損益平衡
- 跟[營運資金](working-capital.md)的關係：在達到損益平衡之前，你需要足夠的營運資金撐過去
