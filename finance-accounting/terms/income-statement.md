# Income Statement（損益表） — 一段時間內，公司賺了多少、花了多少、最後賺還是賠

> **白話說：** 就像你的**記帳 App 月報表**——這個月薪水進來多少、房租水電吃飯花多少、最後剩多少或透支多少，一目了然。

---

## 它到底是什麼？

**Income Statement（損益表）** 是[三大報表](financial-statement.md)之一，也常被叫做 P&L（Profit and Loss Statement）。

它記錄的是**一段時間內**公司的經營成果：

```
營收（Revenue）        ← 總共賣了多少
- 銷貨成本（COGS）     ← 做產品直接花的錢
= 毛利                 ← 賣價扣成本
- 營業費用（OPEX）     ← 租金、薪水、行銷等
= 營業利益             ← 本業到底賺不賺
- 稅金和其他
= 淨利（Net Income）   ← 最後真正賺到的
```

## 生活比喻 / 實際例子

假設你開了一家鹹酥雞攤，一個月的損益表：

| 項目 | 金額 |
|------|------|
| [營收](revenue.md)（這個月賣了多少） | +200,000 |
| [銷貨成本](cogs.md)（雞肉、油、粉） | -80,000 |
| = 毛利 | 120,000 |
| [營業費用](opex.md)（攤位租金、工讀薪水、瓦斯） | -60,000 |
| = 營業利益 | 60,000 |
| 稅金 | -6,000 |
| = [淨利](net-income.md)（最後真正賺到的） | **54,000** |

**造句**：
- 「看一下這季的 **Income Statement**，毛利率有沒有改善」
- 「**損益表** 上面 [OPEX](opex.md) 太高了，要砍預算」
- 「這個月的 **P&L** 是正的，恭喜賺錢了」

## 為什麼要知道這個詞？

- 想知道公司有沒有賺錢，第一個看的就是損益表
- 跟[資產負債表](balance-sheet.md)不同，損益表看的是「一段時間的流量」而非「某天的狀態」
- 公司的季報、年報裡，損益表通常是投資人最先看的
- 搭配[現金流量表](cash-flow-statement.md)一起看，才知道「帳面賺的錢」有沒有真的收回來
