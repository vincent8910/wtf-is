# OPEX（營業費用） — 經營公司日常要花的錢，不包含做產品的成本

> **白話說：** 就像你開店除了材料費以外的**日常開銷**——店租、員工薪水、水電費、廣告費⋯⋯這些不是直接花在做產品上，但沒有它們公司就運轉不了。

---

## 它到底是什麼？

**OPEX** 是 **Operating Expenses（營業費用）** 的縮寫。

跟 [COGS（銷貨成本）](cogs.md) 的差別：
- COGS = 直接跟「做產品」有關的成本
- OPEX = 跟「經營公司」有關的日常支出

常見的 OPEX 項目：

| 項目 | 例子 |
|------|------|
| 薪資費用 | 行政人員、業務、主管的薪水 |
| 租金 | 辦公室、倉庫的租金 |
| 行銷費用 | 廣告、活動、促銷 |
| 研發費用 | 工程師薪水、開發工具 |
| 管理費用 | 會計、法務、IT 設備 |
| [折舊](depreciation.md) / [攤銷](amortization.md) | 設備和無形資產的價值分攤 |

## 生活比喻 / 實際例子

你開了一間咖啡廳：

```
每個月的 OPEX（營業費用）：
├─ 店面租金      50,000
├─ 員工薪水      120,000
├─ 水電瓦斯       15,000
├─ WiFi + POS 系統  3,000
├─ IG 廣告費      10,000
├─ 設備折舊        5,000
└─ 合計 OPEX     203,000

這些錢不管你賣不賣得出去，每個月都要付。
咖啡豆和鮮奶的成本（COGS）是另外算的。
```

**造句**：
- 「公司 **OPEX** 太高了，要想辦法瘦身」
- 「行銷的 **營業費用** 佔比有點多，要調整」
- 「SaaS 公司的 **OPEX** 裡最大塊是人事費用」

## 為什麼要知道這個詞？

- [損益表](income-statement.md)上，毛利減掉 OPEX 就是營業利益——OPEX 控制不好，毛利再高也白搭
- 老闆要「砍成本」的時候，通常砍的就是 OPEX（裁員、搬辦公室、砍廣告預算）
- OPEX 跟 CAPEX（資本支出）不同：OPEX 是日常花費，CAPEX 是買設備、蓋廠房的大筆支出
- 軟體公司特別愛講「OPEX 佔營收比例」，因為這反映了經營效率
