# Net Income（淨利） — 扣掉所有成本、費用、稅之後，公司真正賺到的錢

> **白話說：** 就像你的薪水條上的**實領金額**——底薪 4 萬，扣掉勞健保、所得稅之後，實際匯進帳戶的才是你「真正拿到的」。淨利就是公司的「實領金額」。

---

## 它到底是什麼？

**Net Income（淨利）** 也叫稅後淨利、淨利潤、Bottom Line（因為它在[損益表](income-statement.md)的最下面一行）。

它是經過層層扣除後的最終結果：

```
營收（Revenue）          1,000 萬
- 銷貨成本（COGS）        400 萬
= 毛利                    600 萬
- 營業費用（OPEX）        300 萬
= 營業利益                300 萬
- 利息支出                 30 萬
- 稅金                     50 萬
= 淨利（Net Income）     220 萬  ← 最後真正賺到的
```

淨利是正的 → 公司有賺錢（盈餘）
淨利是負的 → 公司在虧損（虧損）

## 生活比喻 / 實際例子

你的月收入就像一張迷你損益表：

| 項目 | 金額 |
|------|------|
| 月薪（營收） | 50,000 |
| 勞健保（必要費用） | -3,000 |
| 所得稅（稅） | -2,500 |
| 房租（營業費用） | -12,000 |
| 吃飯交通（其他費用） | -15,000 |
| = **淨利（這個月真正存下來的）** | **17,500** |

如果你的「個人淨利」每個月都是正的，你的存款就會增加。公司也一樣——淨利為正，公司的資產就在累積。

**造句**：
- 「公司今年的 **Net Income** 成長了 15%」
- 「看 **淨利** 比看[營收](revenue.md)更重要，營收高不一定賺錢」
- 「**Bottom Line** 是負的，我們還在虧損」

## 為什麼要知道這個詞？

- 淨利是判斷公司「到底有沒有賺錢」最直接的指標
- EPS（每股盈餘）就是用淨利除以股數算出來的——投資人超級在意
- 淨利跟 [EBITDA](ebitda.md) 不同：淨利已經扣掉稅和利息了，EBITDA 沒有
- 有些公司「營收很高但淨利很低」（高成本產業），有些「營收不高但淨利率很好」（SaaS）
