# EBITDA — 不看稅、利息、折舊攤銷，公司本業真正的現金賺錢能力

> **白話說：** 就像你計算自己「工作真正賺了多少」——不管房貸利息、保險、分期付款，純看你的**工作收入扣掉日常開銷**後剩多少。EBITDA 就是公司「本業經營」的真實獲利能力。

---

## 它到底是什麼？

**EBITDA** 念作「yi-bit-dah」，全名是 **Earnings Before Interest, Taxes, Depreciation, and Amortization**，中文叫「稅息折舊攤銷前利潤」。

聽起來很複雜，其實就是把[損益表](income-statement.md)上的[淨利](net-income.md)加回四個東西：

```
EBITDA = 淨利 + 利息 + 稅 + 折舊 + 攤銷

為什麼要加回來？因為：
├─ 利息 → 跟你怎麼借錢有關，不影響本業經營能力
├─ 稅   → 跟稅法有關，各國不同，不影響本業
├─ 折舊 → 是帳上的費用，不是真的花了錢（非現金費用）
└─ 攤銷 → 同上，也是非現金費用
```

扣掉這些「干擾」後，EBITDA 更能看出公司**經營本業到底賺不賺錢**。

## 生活比喻 / 實際例子

你跟朋友各自經營一間飲料店，想比較誰經營得好：

| | 你的店 | 朋友的店 |
|---|---|---|
| 營收 | 200 萬 | 200 萬 |
| 營業費用 | 120 萬 | 130 萬 |
| 銀行貸款利息 | 20 萬 | 0（他沒借錢） |
| 淨利 | 50 萬 | 60 萬 |
| EBITDA | 80 萬 | 70 萬 |

看淨利，朋友贏（60 > 50）。但看 EBITDA，你贏（80 > 70）——你的店經營得比較好，只是借了比較多錢所以利息支出高。

**造句**：
- 「我們的 **EBITDA** margin 是 30%」
- 「投資人在看 **EBITDA**，不是只看淨利」
- 「這間公司 **EBITDA** 很漂亮，但現金流有問題」

## 為什麼要知道這個詞？

- EBITDA 是投資人最常用的獲利指標之一，尤其是在比較不同公司的時候
- 不同公司的借錢方式、稅率、設備折舊方式都不一樣，用 EBITDA 才能「站在同一條線上比較」
- 併購案中，公司估值常用「營收的幾倍」或「EBITDA 的幾倍」來算
- 但 EBITDA 也有缺點：它忽略了[折舊](depreciation.md)和[攤銷](amortization.md)，對於設備密集的公司來說，這些是真實的成本
