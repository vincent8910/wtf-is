# Audit（審計 / 稽核） — 找專業的人來檢查公司的帳有沒有問題

> **白話說：** 就像老師改考卷、或是衛生局來餐廳**抽查**——不是你自己說「我有 100 分」就算，要有第三方來驗證你的帳是不是真的。

---

## 它到底是什麼？

**Audit（審計）** 就是請獨立的專業人士（通常是會計師事務所）來檢查公司的[財務報表](financial-statement.md)，確認帳目是否正確、有沒有造假。

| 類型 | 誰來查 | 目的 |
|------|--------|------|
| 外部審計 | 四大會計師事務所（KPMG、Deloitte、PwC、EY） | 出具「查核報告」，證明財報可信 |
| 內部稽核 | 公司自己的稽核部門 | 內部控制、防弊、流程改善 |
| 稅務審計 | 國稅局 | 查你有沒有逃漏[稅](tax.md) |

審計的結果通常有幾種意見：

```
✅ 無保留意見 → 帳沒問題，一切 OK（最好的結果）
⚠️ 保留意見 → 大部分沒問題，但有些地方有疑慮
❌ 否定意見 → 帳有重大問題
❌ 無法表示意見 → 會計師拿不到足夠資料，沒辦法判斷
```

## 生活比喻 / 實際例子

想像你跟同學合資做園遊會擺攤：

- 你負責管錢，賣了多少、花了多少都由你記錄
- 活動結束後，大家不太確定你記的對不對
- 於是找一位沒有參與的同學來**審查**你的帳本
- 他核對收據、清點現金、比對帳目
- 最後他說：「帳目正確，沒有問題。」→ 這就是審計

在公司裡：
- 上市公司每年都**必須**請會計師來做外部審計
- 審計完成後，會計師會出具「查核報告」
- 如果查核報告有問題，公司的股價可能會大跌

**造句**：
- 「**Audit** 快到了，趕快把帳目整理好」
- 「今年的 **審計** 是找安永（EY）做的」
- 「內部 **稽核** 發現這筆費用報銷有問題」

## 為什麼要知道這個詞？

- 上市公司的財報必須經過審計，投資人才敢相信數字是真的
- 公司想要上市（IPO）之前，要先通過好幾年的審計
- 審計不只查帳——還會評估公司的內部控制制度夠不夠好
- 台灣的[會計年度](fiscal-year.md)結束後，通常在隔年 3 月前要完成審計
