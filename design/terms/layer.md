# 圖層（Layer）— 像疊透明片一樣，每一層放不同的東西

> **白話說：** 圖層就像一疊透明投影片——底下那張放背景、中間放照片、最上面放文字。每一張透明片可以獨立移動、修改，不會影響其他張。

---

## 它到底是什麼？

**圖層（Layer）** 是設計軟體裡最基本的概念。一個設計稿是由很多層「透明片」疊起來的，每一層放不同的元素。

```
最上面 → 📝 文字層：「限時特價」
中間   → 🖼️ 圖片層：產品照片
中間   → 🎨 形狀層：紅色圓形標籤
最下面 → 🏔️ 背景層：白色底色
```

疊起來看，就變成「白色底上有一張產品照片，上面有個紅色標籤寫著限時特價」。

圖層的好處：
- **獨立修改**：想改文字就只動文字那層，不會動到圖片
- **自由移動**：把文字層從左邊拖到右邊，背景不會跟著動
- **開關顯示**：不想看到某個元素？把那一層關掉就好
- **調整順序**：想讓文字在圖片後面？把文字層拖到圖片層下面就好

## 生活比喻 / 實際例子

想像你在做一本手工剪貼簿：

- 先在底紙上貼一張風景照（背景層）
- 再貼一張人物的去背照片（圖片層）
- 最後貼上用彩色紙剪的標題文字（文字層）

每一張貼上去的東西都是一個「圖層」。如果你想換標題，只要撕下最上面那張重貼就好，不用動到底下的風景照。

實際場景：
- 設計師說：「我把按鈕放在獨立的**圖層**上，方便之後調整。」
- PM 問：「可以把 Logo 移到右邊嗎？」設計師說：「可以，我移那個**圖層**就好。」
- 設計師抱怨：「前一個設計師的檔案**圖層**完全沒整理，一百多層全叫『圖層 1 副本 副本 副本』。」

## 為什麼要知道這個詞？

- 圖層是所有設計軟體（Photoshop、[Figma](figma.md)、Illustrator）的核心概念
- 知道圖層的概念，你就能理解設計師為什麼可以「只改文字不動圖片」
- 當設計師說「這個要分圖層做」，表示他在為未來的修改做準備
- 圖層整理得好，設計檔案就好維護；整理得差，接手的人會崩潰
