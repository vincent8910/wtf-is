# 遮罩（Mask）— 用一個形狀遮住圖片的一部分

> **白話說：** 遮罩就像用一張有洞的紙蓋在照片上——只有洞的地方看得到照片，其他地方被遮住了。圖片沒有被剪掉，只是被「藏起來」。

---

## 它到底是什麼？

**遮罩（Mask）** 是設計裡「控制圖片哪些部分要顯示、哪些部分要隱藏」的工具。

重點：遮罩**不會真的刪掉**圖片的一部分，它只是把不要的地方「蓋住」。隨時可以調整遮罩的形狀，露出不同的部分。

常見的遮罩用法：
- 把方形的大頭照變成圓形（社群頭像）
- 讓照片的邊緣漸漸淡出變透明
- 把照片裁成星星、愛心等特殊形狀
- 只露出照片的某一個區域

```
原始照片（方形）         遮罩（圓形）           結果
┌──────────┐         ┌──────────┐         ┌──────────┐
│ 🌸🌸🌸🌸 │         │   ╭──╮   │         │   ╭──╮   │
│ 🌸😊🌸🌸 │    +    │  │    │  │    =    │  │😊│  │
│ 🌸🌸🌸🌸 │         │   ╰──╯   │         │   ╰──╯   │
│ 🌸🌸🌸🌸 │         │          │         │          │
└──────────┘         └──────────┘         └──────────┘
（照片還在，只是圓形以外的部分被遮住了）
```

## 生活比喻 / 實際例子

想像你拿一張黑色卡紙，在中間剪一個圓洞，然後蓋在一張照片上：

- 透過圓洞你只看到照片的一小部分
- 照片本身完好無缺，沒有被剪壞
- 你隨時可以移動黑色卡紙，露出不同的部分
- 你也可以把圓洞改成方洞或心形

這就是遮罩的原理。

實際場景：
- 設計師說：「頭像我用圓形**遮罩**處理，不用真的把照片裁成圓的。」
- PM 問：「可以讓這張圖的邊緣淡出嗎？」設計師說：「加個漸層**遮罩**就好。」
- 工程師說：「CSS 也有 mask 的功能，可以做一樣的效果。」

## 為什麼要知道這個詞？

- 遮罩是設計師天天在用的基本功能
- 知道遮罩的概念，你就能理解為什麼設計師可以「隨時換露出的區域」而不需要重新裁圖
- 跟 [去背（Clipping）](clipping.md) 不同：遮罩是「蓋住不要的部分」，去背是「移除背景」
- 很多你看到的圓形頭像、異形裁切的圖片，都是用遮罩做的
