# 去背 — 把圖片的背景移除，只留下主體

> **白話說：** 去背就像把貼紙從底紙上撕下來——只留下貼紙本身，背後那張紙不要了。把產品照片的背景去掉，只留下產品本身。

---

## 它到底是什麼？

**去背（Clipping / Background Removal）** 就是把圖片中不要的背景移除，只保留主要的物件。

去背前 vs 去背後：

```
去背前：                    去背後：
┌──────────────┐           ┌──────────────┐
│ 🌤️           │           │              │
│    📱        │    →      │    📱        │
│       🌳    │           │              │
│ ░░░░░░░░░░ │           │              │
└──────────────┘           └──────────────┘
（有天空、有樹、有地面）     （只剩手機，背景透明）
```

去背後的圖片通常存成 **PNG 格式**（支援透明背景），這樣貼到任何背景上都不會有白邊。

常見的去背用途：
- 電商產品照片（純白底或透明底）
- 人物照片放到不同背景上
- Logo 放到各種顏色的底上
- 合成海報、Banner

## 生活比喻 / 實際例子

想像你在做美勞作業：

- 你有一張雜誌上的明星全身照
- 你沿著明星的輪廓**小心地剪下來**
- 然後把剪好的明星貼到另一張風景圖上

這就是去背——把主體「摳」出來，去掉原來的背景。

以前設計師要用 Photoshop 一筆一筆去背，現在有 AI 工具（像 remove.bg）一秒就搞定了。

實際場景：
- 設計師說：「這張產品照要先**去背**，才能放到 Banner 上。」
- 行銷說：「電商上架的圖片要求白底，幫忙**去背**一下。」
- PM 問：「為什麼這張圖放上去邊邊有白邊？」——因為去背沒去乾淨，或是存成 JPG（不支援透明）。

## 為什麼要知道這個詞？

- 電商、行銷、社群圖片，去背是最常見的需求之一
- 知道去背的概念，你就知道為什麼設計師要 PNG 格式而不是 JPG
- 跟 [遮罩（Mask）](mask.md) 的差別：遮罩是「蓋住」，去背是真的「移除」
- 現在 AI 去背工具很方便，簡單的圖片不一定要找設計師處理
