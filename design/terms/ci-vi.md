# CI / VI — 企業識別系統，規定品牌的 Logo、字體、顏色怎麼用

> **白話說：** CI / VI 就像連鎖店的「裝潢手冊」——規定了招牌的字體、牆壁的顏色、員工制服的樣式。這樣不管開到哪裡，客人一眼就認出是同一家店。

---

## 它到底是什麼？

- **CI**（Corporate Identity）：企業識別 → 品牌整體形象的完整規劃，包括視覺、理念、行為
- **VI**（Visual Identity）：視覺識別 → CI 裡面「看得到的部分」，是 CI 的一部分

日常溝通中，大家常把 CI 和 VI 混著說，你只要記住：**它就是一本「品牌長什麼樣子的規則手冊」**。

VI 手冊裡通常規定：

| 項目 | 規定什麼 |
|------|---------|
| **Logo** | 標準版、單色版、反白版、最小使用尺寸、周圍要留多少空間 |
| **色彩** | 品牌主色、輔助色、[Pantone](pantone.md) 色號、[RGB](rgb.md) / [CMYK](cmyk.md) 色碼 |
| **字體** | 標題用什麼字體、內文用什麼字體、中英文分別用什麼 |
| **圖示風格** | 線條圖示還是填滿圖示、圓角還是直角 |
| **應用規範** | 名片、信封、簡報模板、社群頭像⋯⋯每一個都有標準版面 |

## 生活比喻 / 實際例子

想想 7-ELEVEN：

- 不管你在台北或高雄，招牌都是那個綠橘紅配色
- 店員穿的圍裙一樣、收銀台的位置差不多
- 集點卡、購物袋上的 Logo 大小和擺放位置都一致

這些就是 CI / VI 的成果。如果沒有這套規範，每家分店自己亂搞，客人根本認不出這是同一個品牌。

實際場景：
- 設計師說：「新產品的包裝要照 **VI** 規範來做，顏色不能亂改。」
- 行銷說：「社群貼圖的 Logo 用法有 **CI** 規範，不能隨便壓縮變形。」
- 老闆說：「我們要做一套 **CI**，品牌形象要統一。」
- 廠商問：「有沒有 **VI** 手冊？我們做贈品需要 Logo 和色號。」

## 為什麼要知道這個詞？

- 任何正式的品牌都應該有 CI / VI，它是品牌一致性的基礎
- 你在製作任何品牌相關的物件（名片、簡報、社群圖）時，都要遵守 VI 規範
- 跟 [設計系統（Design System）](design-system.md) 的關係：CI/VI 管的是「品牌」層面，Design System 管的是「產品/介面」層面
- 當設計師說「這不符合 CI」，意思是你偏離了品牌的視覺規範
