# 出血（Bleed）— 印刷品多留一圈邊，裁切時才不會露白邊

> **白話說：** 出血就像你剪照片要沿著框線剪——如果圖片剛好印到邊邊，手一抖剪歪了就會露出白邊。所以印的時候故意多印出去一圈，剪歪一點也不怕。

---

## 它到底是什麼？

**出血（Bleed）** 是印刷的一個基本概念：把圖案或顏色**故意延伸到裁切線之外**，通常多留 3mm。

為什麼要這樣做？因為印刷廠裁切紙張時，刀不可能 100% 精準。如果你的設計剛好畫到紙張邊緣，裁切稍微偏一點，就會在邊緣露出一條白邊，看起來很不專業。

```
                    ← 出血區域（會被裁掉）
┌─ ─ ─ ─ ─ ─ ─ ─ ─┐
│ ┌───────────────┐ │
│ │               │ │  ← 裁切線（最終成品的邊界）
│ │   你的設計     │ │
│ │               │ │
│ └───────────────┘ │
└─ ─ ─ ─ ─ ─ ─ ─ ─┘
```

出血區域最後會被裁掉，但它確保了即使裁切有誤差，成品邊緣都不會有白邊。

## 生活比喻 / 實際例子

想像你在貼手機保護貼：

- 保護貼做得**剛剛好跟螢幕一樣大**：貼歪一點點就會露出邊，看起來很醜
- 保護貼做得**比螢幕大一圈**：就算貼歪了一點，邊緣也能蓋住

出血就是這個「多留的一圈」。

實際場景：
- 設計師說：「這個名片的設計記得留 **出血** 3mm。」
- 印刷廠退件：「你的背景顏色沒有延伸到**出血**區，請修改。」
- PM 問：「為什麼設計稿比名片尺寸大一圈？」——那就是出血區域。

## 為什麼要知道這個詞？

- 任何要印刷的東西都需要出血，這是印刷的基本規則
- 如果你負責審核印刷品的設計稿，要確認有沒有留出血
- 沒留出血的設計稿送印，印刷廠一定會退件或通知你修改
- 一般標準是留 3mm，有些大型印刷品可能要留更多
