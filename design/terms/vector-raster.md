# 向量圖 / 點陣圖 — 放大不會糊 vs 放大會變馬賽克

> **白話說：** 向量圖像用數學公式畫的圖，不管放多大都清清楚楚；點陣圖像用小格子拼的圖，放大到一定程度就會看到一格一格的馬賽克。

---

## 它到底是什麼？

這是兩種完全不同的圖片格式：

| | 向量圖（Vector） | 點陣圖（Raster） |
|--|-----------------|-----------------|
| 原理 | 用數學公式記錄形狀和顏色 | 用一格一格的小點（像素）組成 |
| 放大 | 怎麼放大都清晰 | 放大會模糊、出現馬賽克 |
| 適合 | Logo、icon、插畫、文字 | 照片、複雜的圖片 |
| 常見格式 | .ai、.svg、.eps | .jpg、.png、.gif |
| 檔案大小 | 通常比較小 | 通常比較大 |

簡單的判斷方式：
- **形狀簡單、線條明確的圖**（Logo、圖示）→ 應該用向量
- **照片、複雜畫面的圖** → 只能用點陣

## 生活比喻 / 實際例子

- **向量圖** 就像用尺和圓規畫的圖：不管你把紙放大幾倍重畫，線條都一樣清楚，因為你知道「從 A 點畫直線到 B 點」這個規則
- **點陣圖** 就像用馬賽克磁磚拼的圖：遠看沒問題，但走近看就是一格一格的磁磚

再想想 LINE 貼圖：如果原始檔是向量圖，印在小小的手機螢幕或超大的看板上都很清楚。但如果只有一張小小的點陣圖，放大印到看板上就會糊成一團。

實際場景：
- 設計師說：「Logo 一定要保留**向量檔**，以後不管放多大都不會糊。」
- 印刷廠說：「你這個 Logo 是**點陣圖**，放大會糊，有沒有**向量檔**？」
- PM 問：「為什麼 Logo 放在名片上很清楚，放到帆布看板就糊了？」——因為用的是點陣圖，不是向量圖。

## 為什麼要知道這個詞？

- Logo 一定要有向量檔（.ai 或 .svg），這是設計的基本規則
- 當印刷廠或廠商跟你要「向量檔」，就是要放大不會糊的原始設計檔
- 手機拍的照片都是點陣圖，無法無限放大
- 搞懂這個，就不會再把一張小小的 .jpg Logo 丟給廠商，然後印出來一團糊
