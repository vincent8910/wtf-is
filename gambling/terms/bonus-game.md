# 獎勵遊戲（Bonus Game） — 觸發特殊條件後進入的額外遊戲

> **白話說：** 就像超級瑪利歐裡的隱藏關卡——打到某個磚塊後，進入一個特別的場景，可以大撈一筆。

---

## 它到底是什麼？

Bonus Game 是老虎機裡的「加場表演」。在正常的旋轉過程中，如果你觸發了特定條件（比如出現 3 個特殊符號），遊戲會切換到一個完全不同的畫面和玩法。

常見的 Bonus Game 類型：

- **選擇類**：畫面上出現好幾個寶箱或門，你選一個打開看裡面是多少獎金
- **轉盤類**：轉一個大轉盤，指到哪裡就拿多少
- **多層級類**：一關一關往上闖，每過一關獎金越大，但也可能在某一關結束
- **技巧類**：需要玩家做一些簡單操作（比如射擊、翻牌）

Bonus Game 通常是一台老虎機裡獎金最豐厚的環節，也是遊戲設計師展現創意的地方。一台老虎機好不好玩，Bonus Game 的設計佔了很大一部分。

## 生活比喻 / 實際例子

想像你在看綜藝節目，來賓正常玩了幾輪遊戲之後，主持人突然說：「恭喜進入 Bonus 關卡！」畫面切到一個金碧輝煌的場景，有三道門讓你選，每道門後面藏著不同的獎金。

這就是 Bonus Game——正常遊戲之外的額外關卡，通常獎勵比正常遊戲好很多。

**造句**：「進 Bonus Game 可以選寶箱，我上次選到最大獎 800 倍！」

## 為什麼要知道這個詞？

- Bonus Game 是區分老虎機好不好玩的重要因素
- 玩家常常是為了觸發 Bonus Game 才持續遊玩的
- 遊戲設計和美術團隊花最多心思在 Bonus Game 的表現上
- [Free Spin](free-spin.md) 其實也算是一種 Bonus Game，是最常見的形式
