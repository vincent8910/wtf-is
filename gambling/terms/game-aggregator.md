# 遊戲聚合商（Game Aggregator） — 把多家遊戲供應商的產品整合到一個 API 接口

> **白話說：** 像外送平台（UberEats、foodpanda）——不用一家一家餐廳跑，一個 App 就能點到所有餐廳的菜。

---

## 它到底是什麼？

Game Aggregator（遊戲聚合商）是博弈產業裡的「中間商」。他們把幾十家甚至上百家 [遊戲供應商](game-provider.md) 的遊戲整合到同一個技術接口（API），讓博弈平台只需要對接一次，就能上架所有遊戲。

如果沒有聚合商，一家博弈平台想要提供 50 家遊戲供應商的遊戲，就需要分別跟 50 家公司談合約、串接 50 套不同的 API、處理 50 種不同的技術規格。光是技術對接就可能花上一整年。

有了聚合商，平台只需要串接聚合商的一套 API，所有遊戲就都能用了。聚合商負責處理跟各家遊戲供應商的技術對接、結算、報表等繁瑣工作。

知名的遊戲聚合商有：SoftSwiss、EveryMatrix、SoftGamings、Hub88 等。他們的商業模式通常是抽取交易佣金（遊戲營收的一定比例）或收取月費。

對小型博弈平台來說，聚合商幾乎是必備的——因為他們沒有足夠的技術團隊去一個一個對接遊戲商。

## 生活比喻 / 實際例子

想像你要辦一場園遊會，想要找 50 家攤販來擺攤。你可以一家一家去談（累死），也可以找一個「園遊會仲介公司」——你只跟仲介說你要什麼類型的攤販，仲介幫你全部搞定。Game Aggregator 就是博弈業的「園遊會仲介」。

**造句**：「接了這家 Game Aggregator，一次就能上線 50 家遊戲供應商的遊戲。」

## 為什麼要知道這個詞？

- 聚合商是博弈產業生態系中很重要的一環，連接了遊戲供應商和平台營運商
- 對想要快速上線的新平台來說，選對聚合商能省下大量時間和技術成本
- 聚合商的品質直接影響遊戲的載入速度、穩定性和玩家體驗
- 理解聚合商的角色才能看懂博弈產業的商業生態和合作關係
