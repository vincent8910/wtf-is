# 遊戲供應商（Game Provider） — 專門開發博弈遊戲的公司

> **白話說：** 像電影公司——迪士尼、漫威、皮克斯各拍各的電影，戲院（博弈平台）再把它們放上去給觀眾看。

---

## 它到底是什麼？

Game Provider（遊戲供應商）就是專門設計和開發博弈遊戲的公司。他們負責做出 [老虎機](slot-machine.md)、[百家樂](baccarat.md)、[輪盤](roulette.md) 等各種遊戲，然後授權給博弈平台上架。

就像你在 Netflix 上看的電影來自不同的製片公司，博弈平台上的遊戲也來自不同的供應商。每家供應商都有自己的特色：

- **PG Soft**：亞洲市場很紅，畫面精緻、動畫流暢
- **Pragmatic Play**：遊戲量大、種類多，是目前全球最活躍的供應商之一
- **NetEnt**：老牌瑞典公司，經典遊戲很多
- **Microgaming**：業界元老，擁有最早的線上老虎機
- **Evolution Gaming**：[真人荷官](live-dealer.md)遊戲的霸主

遊戲在上架之前，必須通過獨立的第三方認證機構（如 GLI、BMM、eCOGRA）的測試，確認 [RNG](rng.md)（隨機數產生器）是公平的、[RTP](rtp.md) 符合宣稱的數值。

遊戲供應商通常不直接面對玩家，而是透過 [遊戲聚合商](game-aggregator.md) 或直接與博弈平台合作，把遊戲送到玩家面前。

## 生活比喻 / 實際例子

想像夜市有很多不同的攤販。有人專做雞排、有人專做珍珠奶茶、有人專做彈珠台。每個攤販（遊戲供應商）有自己的招牌和風格。夜市管理處（博弈平台）負責提供場地和人流，攤販負責提供好吃好玩的東西。兩邊合作，大家都賺錢。

**造句**：「PG Soft 是亞洲很紅的 Game Provider，他們的遊戲畫面很精緻。」

## 為什麼要知道這個詞？

- 不同供應商的遊戲風格、[波動度](volatility.md)、RTP 差異很大，了解供應商可以幫你選遊戲
- 對博弈平台來說，合作的供應商陣容直接影響玩家數量和營收
- 遊戲供應商的認證和合規狀況決定了遊戲的公平性和合法性
- 這是博弈產業裡規模最大、最核心的一個環節
