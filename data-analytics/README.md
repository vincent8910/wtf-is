## 基礎概念類

> 數據分析的入門詞彙——什麼是大數據？資料怎麼整理？報表怎麼看？

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| 大數據（Big Data） | 大到傳統工具處理不了的海量資料 | 整個城市的垃圾量，用掃把掃不完 | 「我們要用大數據來分析用戶行為」 | [→ 詳細](terms/big-data.md) |
| 資料清洗（Data Cleaning） | 把髒亂的原始資料整理乾淨 | 菜市場買回來的菜要洗過才能煮 | 「資料要先清洗，不然分析會不準」 | [→ 詳細](terms/data-cleaning.md) |
| 資料模型（Data Model） | 規劃資料要怎麼組織、分類、關聯的藍圖 | 圖書館的分類系統 | 「先把資料模型建好再開始做」 | [→ 詳細](terms/data-model.md) |
| 資料視覺化（Visualization） | 把數字變成圖表讓人一眼看懂 | 天氣預報的那張氣象圖 | 「這份報告需要做資料視覺化」 | [→ 詳細](terms/visualization.md) |
| BI（商業智慧） | 把公司資料變成老闆看得懂的報表和儀表板 | 公司的健檢報告 | 「BI 團隊幫我拉一下上個月的數據」 | [→ 詳細](terms/bi.md) |
| 指標 vs 維度 | 指標是「多少」，維度是「按什麼分」 | 考試分數是指標，科目是維度 | 「這個報表的指標和維度要怎麼設？」 | [→ 詳細](terms/metric-dimension.md) |

## 統計方法類

> 數據分析背後的統計學——怎麼找規律？怎麼判斷結論可不可信？

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| 迴歸分析（Regression） | 用已知資料預測未來趨勢的統計方法 | 看過去的月薪成長速度，預測明年會賺多少 | 「跑個迴歸分析看看哪個因素影響最大」 | [→ 詳細](terms/regression-analysis.md) |
| 相關性 vs 因果關係 | 兩件事一起發生，不代表 A 造成 B | 冰淇淋銷量和溺水人數都在夏天增加，但吃冰不會導致溺水 | 「這只是相關性，不是因果關係」 | [→ 詳細](terms/correlation-causation.md) |
| 離群值（Outlier） | 跟其他數據差很遠的異常值 | 全班考 70 分，有個人考 5 分 | 「先把離群值處理掉，不然平均值會被拉歪」 | [→ 詳細](terms/outlier.md) |
| 抽樣偏差（Sampling Bias） | 樣本不夠有代表性，導致結論歪掉 | 只問台北人喜歡什麼，就說全台灣都喜歡 | 「這個調查有抽樣偏差，樣本都是年輕人」 | [→ 詳細](terms/sampling-bias.md) |
| P 值（P-value） | 判斷實驗結果是真的有差還是碰巧的指標 | 丟硬幣連出 10 次正面，是作弊還是運氣好？ | 「P 值小於 0.05，結果有統計顯著性」 | [→ 詳細](terms/p-value.md) |
| 常態分佈（Normal Distribution） | 鐘形曲線，大部分數據集中在中間 | 全班成績分佈：大多數人在中間，特別好和特別差的很少 | 「這個資料呈常態分佈」 | [→ 詳細](terms/normal-distribution.md) |
| 描述性 vs 預測性分析 | 回頭看發生了什麼 vs 預測接下來會怎樣 | 看體檢報告 vs 預測未來健康狀況 | 「我們不只要描述性分析，還要做預測」 | [→ 詳細](terms/descriptive-predictive.md) |

## 進階分析類

> 更進階的數據分析技巧——從海量資料中挖寶、讓機器幫你判斷。

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| 資料探勘（Data Mining） | 從大量資料中挖掘出隱藏的規律 | 淘金：從一堆沙子裡找出金子 | 「用資料探勘看看有什麼隱藏的消費模式」 | [→ 詳細](terms/data-mining.md) |
| 機器學習（Machine Learning） | 讓電腦自己從資料中學會判斷 | 教小孩認水果：看多了自己就會分辨 | 「我們用機器學習來預測客戶流失」 | [→ 詳細](terms/machine-learning.md) |
| 世代分析（Cohort Analysis） | 把同一時期進來的用戶分成一組來追蹤 | 追蹤同一屆畢業生的就業狀況 | 「做個世代分析看哪一批用戶留存最好」 | [→ 詳細](terms/cohort-analysis.md) |
| 分群分析（Cluster Analysis） | 把相似的東西自動分成幾群 | 自助餐把菜自動分成主食、肉類、青菜 | 「分群分析把用戶分成了五種類型」 | [→ 詳細](terms/cluster-analysis.md) |
| 情感分析（Sentiment Analysis） | 用程式判斷文字是正面還是負面 | 看餐廳評論，一則一則判斷是好評還是差評 | 「跑個情感分析看看網友對這次活動的反應」 | [→ 詳細](terms/sentiment-analysis.md) |
| 樞紐分析表（Pivot Table） | 把一大堆資料快速分類彙總的工具 | 把一整箱發票快速按月份和類別分堆統計 | 「用樞紐分析表交叉比對一下」 | [→ 詳細](terms/pivot-table.md) |

## 商業指標類

> 老闆最愛問的那些數字——多少人在用？留住幾成？每人賺多少？

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| DAU / MAU（日活 / 月活） | 每天 / 每月有多少人在用你的產品 | 早餐店每天來幾個客人 / 每月來幾個客人 | 「我們的 DAU 有 50 萬」 | [→ 詳細](terms/dau-mau.md) |
| 留存率（Retention Rate） | 用了一次之後，有多少人還會回來繼續用 | 新開的餐廳，一個月後還有多少客人回訪 | 「7 日留存率只有 20%，要想辦法提升」 | [→ 詳細](terms/retention.md) |
| 流失率（Churn Rate） | 有多少用戶離開不再使用 | 健身房每個月有多少會員退出 | 「這個月的流失率太高了」 | [→ 詳細](terms/churn-rate.md) |
| LTV（用戶終身價值） | 一個用戶從開始到離開，總共帶來多少收入 | 一個常客從第一次來到不來了，總共在你店裡消費多少 | 「LTV 要大於 CAC 才划算」 | [→ 詳細](terms/ltv.md) |
| ARPU（每用戶平均收入） | 平均每個用戶幫你賺多少錢 | 餐廳的平均客單價 | 「ARPU 這季提升了 15%」 | [→ 詳細](terms/arpu.md) |

## 比喻對照總表

> 每個數據分析術語，用一句生活比喻讓你秒懂。

### 基礎概念類

| 術語 | 比喻 |
|------|------|
| 大數據（Big Data） | 整個城市的**垃圾量**——用掃把掃不完，得出動垃圾車和清潔隊 |
| 資料清洗（Data Cleaning） | **菜市場買回來的菜**——要洗乾淨、挑掉爛葉子才能煮 |
| 資料模型（Data Model） | **圖書館的分類系統**——決定書要怎麼分類、放在哪個書架 |
| 資料視覺化（Visualization） | **天氣預報的氣象圖**——不用看一堆數字，看圖就知道明天會不會下雨 |
| BI（商業智慧） | **公司的健檢報告**——把各種數據整理成老闆一眼就看懂的報告 |
| 指標 vs 維度 | **考試分數**是指標，**科目**是維度——分數告訴你「多少」，科目告訴你「哪一科」 |

### 統計方法類

| 術語 | 比喻 |
|------|------|
| 迴歸分析（Regression） | 看過去的**月薪成長速度**——用趨勢線預測明年會賺多少 |
| 相關性 vs 因果關係 | **冰淇淋銷量和溺水人數**——夏天都增加，但吃冰不會導致溺水 |
| 離群值（Outlier） | **全班考 70 分，有個人考 5 分**——太偏離正常的那個值 |
| 抽樣偏差（Sampling Bias） | **只問台北人就說代表全台灣**——樣本不夠廣，結論就歪了 |
| P 值（P-value） | **丟硬幣連出 10 次正面**——是真的不公平，還是純粹運氣好？ |
| 常態分佈（Normal Distribution） | **全班成績的鐘形曲線**——大多數人在中間，頭尾很少 |
| 描述性 vs 預測性分析 | **看體檢報告** vs **預測未來健康**——一個回頭看，一個往前看 |

### 進階分析類

| 術語 | 比喻 |
|------|------|
| 資料探勘（Data Mining） | **淘金**——從一堆沙子裡篩出有價值的金子 |
| 機器學習（Machine Learning） | **教小孩認水果**——看多了自己就學會分辨蘋果和橘子 |
| 世代分析（Cohort Analysis） | **追蹤同一屆畢業生**——看同一批人後來的發展如何 |
| 分群分析（Cluster Analysis） | **自助餐分區**——自動把相似的菜分成主食、肉類、青菜 |

| 樞紐分析表（Pivot Table） | **一箱發票快速分堆統計**——按月份、類別交叉整理 |

### 商業指標類

| 術語 | 比喻 |
|------|------|
