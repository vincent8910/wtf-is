# 散戶 — 用自己的錢在股市裡買賣的一般投資人

> **白話說：** 就像**夜市裡的散客**——你自己逛、自己挑、自己買，不是批發商也不是店家，只是一個人去買東西的普通消費者。在股市裡，散戶就是我們這些普通人。

---

## 它到底是什麼？

散戶（Retail Investor）就是**用自己的錢、自己決定買賣的一般投資人**。

跟散戶相對的是[法人](institutional-investors.md)——法人是拿別人的錢來投資的機構（像基金公司、外資、壽險公司），資金動輒幾十億、幾百億。散戶通常就是你、我、隔壁鄰居、辦公室同事，拿自己的薪水存款在買股票的人。

散戶的特色：

```
散戶 vs 法人：

散戶：
├─ 資金：幾萬到幾百萬（跟法人比是小蝦米）
├─ 資訊：看新聞、看 YouTube、聽朋友說
├─ 決策：容易追高殺低、受情緒影響
└─ 優勢：船小好調頭，想買就買、想賣就賣

法人：
├─ 資金：幾十億到幾千億
├─ 資訊：有專業研究團隊、法說會第一手消息
├─ 決策：有紀律、有策略
└─ 劣勢：資金太大，進出場不方便
```

散戶在台灣股市是非常重要的力量。台股散戶的交易量佔了將近六成，也就是說超過一半的交易都是散戶在做的。

## 生活比喻 / 實際例子

想像股市就是一個超大型的菜市場：

- **散戶** = 提著菜籃來買菜的婆婆媽媽，一次買個幾百塊
- **法人** = 開著大貨車來批貨的餐廳老闆，一次買幾十萬
- 散客多的時候市場熱鬧，散客都不來的時候市場冷清

散戶的經典行為：
- 看到股票漲了才衝進去買 → 結果買在最高點
- 看到股票跌了就嚇到賣掉 → 結果賣在最低點
- 所以才有那句名言：「散戶永遠是最後知道消息的人」

**造句**：
- 「這波下跌就是**散戶**恐慌殺出來的」
- 「**散戶**都在追這支股票，法人已經在倒貨了」
- 「我就是個小**散戶**，被割韭菜割習慣了」
- 「這支股票**散戶**比例很高，籌碼比較凌亂」

## 為什麼要知道這個詞？

- 如果你有買股票，你就是散戶——了解自己的定位很重要
- 知道散戶的弱點（追高殺低、資訊落後），才能避免犯同樣的錯
- 看[籌碼分析](chip-analysis.md)的時候，「散戶買超」和「散戶賣超」是重要的判斷指標
- 財經節目常說「散戶指標」——當散戶都衝進來的時候，有時反而是該小心的訊號
