# 資產配置 (Asset Allocation) — 把錢分散放在不同類型的資產

> **白話說：** 就像**不要把所有食物都放同一個冰箱**——萬一那個冰箱壞了，你的食物全沒了。把錢分散放在股票、債券、現金等不同地方，就算其中一個出問題，其他的還在。

---

## 它到底是什麼？

資產配置就是把你的投資資金**分配到不同類型的資產**，主要的資產類型有：

| 資產類型 | 特性 | 比喻 |
|---------|------|------|
| **股票** | 報酬高但波動大 | 雲霄飛車，刺激但可能暈 |
| **債券** | 報酬穩定但偏低 | 公車，慢但穩 |
| **現金/定存** | 幾乎不賠但報酬最低 | 走路，最安全但最慢 |
| **不動產** | 保值但流動性低 | 腳踏車，需要維護但實用 |
| **黃金/原物料** | 避險但不生利息 | 急救包，平時用不到但危急時很重要 |

資產配置的核心觀念是：**不同資產在不同時期的表現不一樣**。

- 經濟好的時候，股票通常漲比較多
- 經濟差的時候，債券和黃金通常比較抗跌
- 如果你全部押在同一種資產，等於把命運完全綁在一起

常見的資產配置比例：

```
積極型（年輕、能承受風險）：股票 80% + 債券 15% + 現金 5%
穩健型（中年、平衡風險）：股票 60% + 債券 30% + 現金 10%
保守型（退休、保本為主）：股票 30% + 債券 50% + 現金 20%
```

## 生活比喻 / 實際例子

想像你的收入來源如果只有一份工作：
- 公司裁員 → 你就零收入了

但如果你同時有：
- 一份正職薪水（穩定，像「債券」）
- 一個網拍副業（賺多賺少不一定，像「股票」）
- 一間出租套房（每月穩定收租，像「不動產」）
- 銀行活存（隨時可用，像「現金」）

就算其中一個出問題，你的生活不會馬上垮掉。**資產配置就是把你的財務風險分散開來**。

實際會聽到的話：
- 「年輕的時候**資產配置**可以積極一點，股票比例高一些」
- 「不要 100% 都買股票，要做好**資產配置**才安全」
- 「**股債配置 7:3** 是很經典的入門比例」
- 「快退休了，**資產配置**要慢慢調成保守型」
- 「資產配置比選股更重要——研究顯示，**90% 的投資績效來自資產配置**」

## 為什麼要知道這個詞？

資產配置是投資理財最重要的觀念之一，甚至比「選哪支股票」更重要。因為再好的股票也可能暴跌，但如果你做好資產配置，整體資產不會跟著一起崩。

跟[分散投資](diversification.md)的概念相輔相成：資產配置是「不同類型的資產要分開放」，分散投資是「同一類型裡面也不要只買一支」。兩個一起做，你的投資就有了雙重保護。

搭配[定期定額](dca.md)的方式來執行資產配置，就是一個簡單又有效的長期投資計畫。
