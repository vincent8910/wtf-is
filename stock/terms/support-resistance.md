# 支撐 / 壓力 — 股價跌到某處就跌不下去 / 漲到某處就漲不上去

> **白話說：** 就像大樓的**地板和天花板**——股價掉到地板會彈起來，漲到天花板會被擋住。

---

## 它到底是什麼？

在股票市場裡，股價常常會在某些價位反覆「卡住」：

- **支撐（Support）**：股價跌到這個價位附近就會止跌反彈，像有一隻手在下面接住它
- **壓力（Resistance）**：股價漲到這個價位附近就會停住下跌，像有一面牆擋在上面

為什麼會這樣？因為很多人在同一個價位做出同樣的決定：

- **支撐形成的原因**：很多人覺得「跌到這個價格很便宜，我要買」→ 買盤湧入 → 跌不下去
- **壓力形成的原因**：很多人覺得「漲到這個價格夠了，我要賣」→ 賣壓湧出 → 漲不上去

```
        壓力 ─ ─ ─ ─ ─ ─ ─ ─ ─  ← 股價到這裡就被擋住
       /    \      /    \
      /      \    /      \
     /        \  /        \
    /          \/          \
─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  ← 股價到這裡就被接住
        支撐
```

## 生活比喻 / 實際例子

想像你在蝦皮賣二手 iPhone：

- **支撐**：你賣 15,000 元沒人買，降到 12,000 元馬上很多人搶 → 12,000 就是這支手機的「支撐價」
- **壓力**：你想賣 18,000 元，但同樣的手機別人都賣 16,000，你的永遠賣不出去 → 16,000 就是「壓力價」

在股市裡也一樣，有些價格會形成大家「心中的底價」或「心中的天花板」。

實際會聽到的話：
- 「台積電在 500 元有很強的**支撐**，跌到那邊都會彈」
- 「這支股票 200 元是很大的**壓力**，已經試了三次都過不去」
- 「**支撐一旦跌破**，就會變成新的壓力」

## 為什麼要知道這個詞？

支撐和壓力是技術分析裡最直覺、最常用的觀念。

不管你看什麼分析師的報告，幾乎都會提到「支撐在哪、壓力在哪」——這就像看地圖時知道「紅線是不能走的、綠線是可以走的」。它幫你判斷：股價可能在哪裡停下來、在哪裡反彈。

搭配 [K 線](k-line.md) 和 [成交量](volume.md) 一起看，你就能更準確地判斷支撐壓力有沒有被突破。
