# 財報季（Earnings Season） — 美股公司集中公布財報的時段

> **白話說：** 就像**期末考週，所有公司集中交成績單**——每季一次，投資人盯著看誰考得好、誰考砸了，股價會跟著成績大幅波動。

---

## 它到底是什麼？

美國上市公司每年要公布 **4 次**財務報告（每季一次），而大部分公司都集中在差不多的時段公布，這段密集公布財報的期間就叫**財報季**。

```
Q1 財報季：4 月中 ～ 5 月中（公布 1-3 月的成績）
Q2 財報季：7 月中 ～ 8 月中（公布 4-6 月的成績）
Q3 財報季：10 月中 ～ 11 月中（公布 7-9 月的成績）
Q4 財報季：1 月中 ～ 2 月中（公布 10-12 月的成績）
```

財報裡面最重要的數字：

| 指標 | 白話 | 為什麼重要 |
|------|------|-----------|
| 營收（Revenue） | 公司賺了多少錢 | 看生意好不好 |
| 每股盈餘（[EPS](eps.md)） | 每股賺多少 | 最核心的獲利指標 |
| 展望（Guidance） | 公司對下季的預測 | 比現在的數字更重要！ |

財報季最刺激的地方在於：**市場會預估一個數字，然後跟公司公布的實際數字比較**。

```
分析師預期 EPS = 1.5 元

公司公布 EPS = 1.8 元 → 「優於預期」→ 股價噴漲 🚀
公司公布 EPS = 1.2 元 → 「不如預期」→ 股價暴跌 📉
公司公布 EPS = 1.5 元 → 「符合預期」→ 可能小漲或沒反應
```

注意：**就算公司賺錢，只要沒有「超出預期」，股價也可能跌**。因為市場看的不是絕對好壞，而是「有沒有超過大家的期待」。

## 生活比喻 / 實際例子

想像一所學校的期末考：

- **財報季** = 學校的**成績公布週**
- **分析師預期** = 家長的期望值。媽媽覺得你這次應該考 85 分
- **公司公布財報** = 你的成績單出來了

結果：
- 你考了 **95 分**（超過預期）→ 媽媽超開心，獎勵你（股價漲）
- 你考了 **70 分**（低於預期）→ 媽媽很失望，罰你（股價跌）
- 你考了 **85 分**（符合預期）→ 媽媽覺得還好，沒什麼反應

最慘的情況：你考了 80 分，其實不差，但媽媽期望你考 90 分 → 還是會被罵（**「優等生」的壓力**，就像蘋果、輝達這種公司，大家期望超高）

實際會聽到的話：
- 「下週開始**財報季**，科技股要注意」
- 「蘋果**財報優於預期**，盤後大漲 5%」
- 「特斯拉**財報不如預期**，股價崩了」
- 「**財報季**是美股最刺激的時候，每天都有大波動」
- 「最重要的不是數字本身，而是有沒有超過**市場預期**」

## 為什麼要知道這個詞？

如果你投資美股，財報季是一年四次的「超級盃」，每次持續約一個月。這段期間股價波動特別大，一份財報可以讓一支股票在[盤後交易](pre-after-market.md)就漲跌 10% 以上。

即使你沒有直接投資美股，台灣很多公司（尤其是電子供應鏈）的股價也會受到美股財報影響。比如蘋果財報好 → 蘋果供應商台積電、鴻海、大立光也會跟著漲。

財報季的生存法則：
- **不確定就不要在財報公布前重押單一股票**——賭對了大賺，賭錯了大賠
- **關注「展望」比「數字」重要**——公司對未來的看法往往比現在的成績更影響股價
- **注意「已經反映在股價裡了」**——如果大家早就預期很好，公布後反而可能下跌（利多出盡）
