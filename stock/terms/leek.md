# 韭菜 — 被反覆收割的散戶

> **白話說：** 就像韭菜田裡的韭菜——**割了又長、長了又割**。韭菜指的是在股市裡一直被當肥羊宰、反覆賠錢卻不學教訓的[散戶](retail-investor.md)。

---

## 它到底是什麼？

「韭菜」是股市裡的網路用語，形容那些**反覆被收割（賠錢）的散戶投資人**。

這個詞來自韭菜的特性：韭菜割掉之後會再長出來，然後又可以再割。就像股市裡永遠有新的散戶進場，被割了一批，又來一批。

```
韭菜的經典循環：

1. 聽到大家都在賺錢 → 「我也要買！」→ 衝進股市
2. 追高買進（買在山頂）
3. 股價開始跌 → 「沒關係，會漲回來的」→ 凹單
4. 跌到受不了 → 「不行了，賣掉！」→ 殺在谷底
5. 賣掉之後股價反彈 → 「……」
6. 過一陣子又聽到誰在賺錢 → 回到第 1 步

這個循環就叫做「韭菜收割」
```

韭菜的常見特徵：
- **追高殺低**：漲的時候追著買，跌的時候嚇到賣
- **聽明牌操作**：別人報什麼就買什麼，沒有自己的判斷
- **沒有[停損](stop-loss-profit.md)紀律**：該停損的時候[凹單](holding-loss.md)，該抱住的時候卻早早賣掉
- **重複犯同樣的錯**：被割了好幾次還是不改

## 生活比喻 / 實際例子

想像股市就是一座牧場：

- **大戶 / 主力** = 牧場主人，知道什麼時候該播種、什麼時候該收割
- **韭菜** = 韭菜田裡的韭菜，努力長高（存錢投入），結果被割掉（賠錢）。過一陣子又長出來（又存了一筆錢），然後又被割

或者想像夜市的套圈圈遊戲：
- 你花了 500 元去套圈圈，一個都沒套到
- 你覺得「下次一定可以」，又花了 500 元
- 老闆笑嘻嘻地看著你一直投錢
- 你就是韭菜，老闆就是主力

實際會聽到的話：
- 「我就是**韭菜**命，買什麼跌什麼」
- 「主力又在**割韭菜**了，散戶慘兮兮」
- 「一看就知道是**韭菜**，追高追成那樣」
- 「新一批**韭菜**進場了，行情要準備反轉了」
- 「我以前是**韭菜**，後來學會紀律投資才開始賺錢」

## 為什麼要知道這個詞？

「韭菜」這個詞雖然是自嘲的玩笑話，但背後反映的是散戶在股市裡的真實困境：**資訊落後、情緒化操作、缺乏紀律**。

了解「韭菜」的行為模式，最大的意義是**避免自己成為韭菜**：
- 不追高殺低，有自己的買賣策略
- 不聽明牌，做好功課再進場
- 設好[停損停利](stop-loss-profit.md)，嚴格執行
- 每次賠錢後檢討原因，不要重複犯錯

記住：每個投資高手都曾經是韭菜，差別在於有沒有從被收割的經驗中學到東西。
