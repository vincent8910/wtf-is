# 存股 — 每個月固定買股票，長期持有不賣

> **白話說：** 就像**每個月固定存錢到定存**——只是你存的不是現金，而是股票。不管股價高低，固定時間就買，長期累積，靠[配息](dividend-fill.md)和股價成長來賺錢。

---

## 它到底是什麼？

存股就是一種**長期投資策略**：你挑好一支股票（或 [ETF](etf.md)），然後每個月固定買一定金額或一定股數，不管股價是漲是跌，持續買、不要賣。

存股的核心觀念：

```
存股 vs 短線交易：

存股：
├─ 每個月固定買 5,000 元的 0056
├─ 股價高就少買幾股，股價低就多買幾股（定期定額）
├─ 持有 10 年、20 年
├─ 靠配息 + 股價長期成長來獲利
└─ 像種樹，慢慢長大、年年結果

短線交易：
├─ 今天買、明天賣
├─ 猜漲跌賺價差
├─ 需要盯盤、技術分析
└─ 像釣魚，有時大豐收、有時空手回
```

存股最重要的兩件事：

1. **選對標的**：要選穩定獲利、持續配息的公司或 ETF。不是隨便一支都適合存
2. **堅持不賣**：跌的時候不恐慌殺出，漲的時候不貪心獲利了結。時間是存股族最大的朋友

台灣最多人存的標的：

| 標的 | 為什麼適合存 |
|------|-------------|
| 0050 | 跟著台灣大盤走，長期向上 |
| 0056 | 高股息，每年配息穩定 |
| 00878 | 高股息 + ESG，季配息 |
| 中華電（2412） | 像公用事業，獲利穩定 |
| 台積電（2330） | 護國神山，長期成長 |

## 生活比喻 / 實際例子

想像你每個月存 5,000 元到郵局定存：

- **銀行定存** = 每年利率 1.5%，10 年後你的錢多了一點點
- **存股** = 每年股息殖利率 5%，加上股價可能成長 5%，10 年後你的資產可能翻倍

但存股的風險是：股票可能跌。定存不會虧損，存股可能短期帳面虧錢。所以你要有耐心，撐過去才會看到成果。

**造句**：
- 「我每個月**存股** 0056，已經存三年了」
- 「**存股**不要看短線漲跌，重點是長期」
- 「**存股族**最怕的就是中途受不了賣掉」
- 「年輕人可以用**定期定額存股**，慢慢累積資產」

## 為什麼要知道這個詞？

- 存股是目前台灣最流行的投資方式之一，身邊一定有人在存股
- 很多券商都推出了「定期定額」功能，每月最低 1,000 元就能開始
- 適合不想花時間研究股票、不想盯盤的上班族和投資新手
- 存股搭配[配息](dividend-fill.md)的概念，是理解被動收入的基礎
