# ADR（美國存託憑證） — 外國公司在美國掛牌交易的方式

> **白話說：** 就像台灣的珍珠奶茶品牌在**美國開授權分店**——你不用飛去台灣，在美國就能買到同樣的東西。

---

## 它到底是什麼？

ADR 的全名是 **American Depositary Receipt（美國存託憑證）**。

簡單來說：一間外國公司（例如台積電）想讓美國投資人也能買它的股票，但美國人沒辦法直接買台灣的股票。怎麼辦？

做法是這樣的：

```
1. 台積電把一些股票交給美國的銀行（存託銀行）
2. 美國的銀行根據這些股票，發行「憑證」（ADR）
3. 美國投資人買賣這些 ADR，就等於間接持有台積電的股票
4. ADR 用美金計價，在美國股市交易
```

所以 ADR 就是一張「證明你擁有某間外國公司股票」的憑證，讓你不用開外國的證券帳戶，在美國就能投資外國公司。

台灣最有名的 ADR 就是「台積電 ADR」（代號 TSM），在美國紐約證券交易所掛牌。很多台灣投資人會關注台積電 ADR 的漲跌，因為它在[盤前盤後](pre-after-market.md)交易時段的表現，常常預告隔天台灣台積電的走勢。

## 生活比喻 / 實際例子

想像你超愛吃台南的某家牛肉湯，但你住在台北：

- **沒有 ADR 的世界**：你每次想喝牛肉湯都要自己坐高鐵去台南，很麻煩
- **有 ADR 的世界**：那家店在台北開了授權分店，用的是同樣的食材和做法。你在台北就能喝到，不用跑去台南

ADR 就是那個「授權分店」——賣的是同樣的東西（同一間公司的股票），只是開在不同的地方（美國市場），用不同的幣別（美金）。

實際會聽到的話：
- 「昨晚**台積電 ADR** 漲了 3%，今天台積電應該也會漲」
- 「日月光的 **ADR** 跌了，要注意一下」
- 「看 **ADR** 走勢可以預判台股明天的開盤方向」
- 「美國投資人透過 **ADR** 買台灣公司的股票」

## 為什麼要知道這個詞？

如果你投資台灣的大型公司（尤其是台積電），ADR 是一個超重要的參考指標。

因為美股交易時間是台灣的晚上到半夜，所以「昨晚台積電 ADR 漲跌多少」常常能預測今天台灣台積電的開盤走勢。很多投資人睡前都會看一下 ADR 的表現，決定隔天要不要操作。

另外，如果你想投資美股，很多外國公司（像阿里巴巴、三星）也是透過 ADR 的方式在美國掛牌，讓你用美國的證券帳戶就能投資全世界的公司。
