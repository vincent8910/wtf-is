# 黑天鵝 — 無法預料的重大事件

> **白話說：** 以前歐洲人覺得天鵝都是白色的，直到有人在澳洲發現了**黑天鵝**——原來不可能的事真的會發生。在股市裡，黑天鵝指的是沒有人預料到、卻造成巨大衝擊的事件。

---

## 它到底是什麼？

「黑天鵝」（Black Swan）這個詞是由風險學家 Nassim Nicholas Taleb（塔雷伯）在 2007 年的同名書中提出的，用來形容**符合三個條件的事件**：

```
黑天鵝事件的三個特徵：

1. 事前無法預測 → 沒有人料到它會發生
2. 衝擊力極大　 → 發生之後造成巨大影響
3. 事後合理化　 → 發生之後大家都說「早就知道了」

歷史上的黑天鵝事件：
├─ 2001 年 911 恐怖攻擊 → 美股暫停交易，全球股市崩跌
├─ 2008 年金融海嘯　　　→ 雷曼兄弟倒閉，全球經濟衰退
├─ 2020 年新冠疫情　　　→ 全球股市暴跌，各國封城
├─ 2022 年俄烏戰爭　　　→ 能源價格飆升，歐洲經濟受創
└─ 每次都是事前沒人想到，事後大家都說「早該預防」
```

黑天鵝事件之所以可怕，不是因為它發生的機率高，而是因為**你完全無法預測它什麼時候會來，而且一來就是天翻地覆**。

## 生活比喻 / 實際例子

想像你每天走同一條路上班，走了 10 年都平安無事。你會覺得「這條路很安全」。但某天這條路突然發生了大地震、路面崩塌——這就是黑天鵝。

或者想像你養了一隻雞：
- 雞每天被餵食，覺得人類對牠很好
- 第 364 天牠還是很開心地吃著飼料
- 第 365 天是感恩節，雞被宰了
- 對雞來說，這就是一隻黑天鵝

重點是：**過去的經驗不代表未來不會發生意外**。

實際會聽到的話：
- 「這次疫情就是典型的**黑天鵝**事件，完全無法預測」
- 「投資要留一些現金，以防**黑天鵝**來的時候有錢可以撿便宜」
- 「你不能假設**黑天鵝**不會發生，要做好風險控管」
- 「每次**黑天鵝**過後，市場最終都會恢復，問題是你撐不撐得過去」
- 「[VIX](vix.md) 飆到 80，標準的**黑天鵝**行情」

## 為什麼要知道這個詞？

黑天鵝事件告訴我們一件殘酷的事：**你無法預測所有風險**。不管你做了多完美的分析、看了多少技術指標，總有一些事情是你完全想不到的。

對投資人來說，黑天鵝的啟示是：

1. **永遠不要 All In**：不要把所有的錢都投入市場，留一些現金當安全墊
2. **分散投資**：不要把雞蛋都放在同一個籃子裡，分散到不同的股票、不同的市場、甚至不同的資產類型（股票 + 債券 + 現金）
3. **設好[停損](stop-loss-profit.md)**：黑天鵝來的時候，有停損的人可以把傷害降到最低
4. **危機就是轉機**：每次黑天鵝事件造成的大跌，長期來看都是進場的好機會（前提是你手上還有錢）

股市裡有一句話：「牛市賺多少不重要，黑天鵝來的時候你還剩多少才重要。」風險管理的核心不是追求最高報酬，而是確保你在最壞的情況下還能活下來，這就是[風險報酬比](risk-reward.md)的概念。
