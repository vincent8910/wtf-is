# 景氣循環股 (Cyclical) — 跟著景氣好壞一起漲跌的股票

> **白話說：** 就像**冰淇淋店夏天生意好、冬天生意差**——有些公司的業績會跟著經濟景氣一起上上下下，景氣好的時候賺很多，景氣差的時候可能虧錢。

---

## 它到底是什麼？

景氣循環股指的是那些**業績跟整體經濟景氣高度連動**的股票。當經濟好的時候，大家有錢就會多花、多買；經濟差的時候，大家就節省、延後消費。

常見的景氣循環產業：

| 產業 | 為什麼受景氣影響 | 台灣例子 |
|------|-----------------|---------|
| **航運** | 景氣好 → 貿易量大 → 運費高 | 長榮、陽明 |
| **鋼鐵** | 景氣好 → 建設多 → 鋼鐵需求大 | 中鋼 |
| **石化** | 景氣好 → 工業生產多 → 原料需求大 | 台塑 |
| **汽車** | 景氣好 → 大家敢買車 | 裕隆 |
| **半導體**（部分）| 景氣好 → 消費性電子需求大 | 聯電、力積電 |
| **營建** | 景氣好 → 房市熱 → 蓋房子 | 興富發 |
| **觀光旅遊** | 景氣好 → 大家願意花錢旅遊 | 雄獅 |

景氣循環股的特色：
- **景氣好時**：獲利暴增、股價飆漲、[EPS](eps.md) 很高
- **景氣差時**：獲利暴跌甚至虧損、股價大跌
- **週期性**：會反覆經歷高峰和低谷

## 生活比喻 / 實際例子

想像夜市裡的兩種攤位：

- **冰淇淋攤**（景氣循環）：夏天的時候大排長龍、日營收 2 萬；冬天的時候門可羅雀、日營收 3 千。生意好壞完全看「季節」（景氣）。

- **滷肉飯攤**（[防禦型](defensive-stock.md)）：不管夏天冬天，每天都穩穩賣個 8 千。因為大家不管景氣好壞都要吃飯。

景氣循環股就像冰淇淋攤——你要在「冬天」（景氣最差、大家不看好的時候）買進，在「夏天」（景氣最好、大家搶著買的時候）賣出。

⚠️ 一個常見的陷阱：景氣循環股在[本益比](pe-ratio.md)最低的時候，可能反而是最危險的時候（因為 EPS 在高峰，之後可能暴跌）。

實際會聽到的話：
- 「航運股是典型的**景氣循環股**，不能只看本益比」
- 「**景氣循環股要反著買**——大家最悲觀的時候進場」
- 「現在景氣開始回溫，**循環股**可以留意了」
- 「不要在景氣高峰的時候追買循環股，小心**買在山頂**」

## 為什麼要知道這個詞？

投資景氣循環股的邏輯跟一般股票不太一樣。一般股票是「[EPS](eps.md) 好就買」，但循環股可能要「EPS 最差的時候才是最好的買點」——因為谷底之後就是回升。

了解一支股票是不是景氣循環股，可以幫你避免用錯分析框架。搭配了解[防禦型股票](defensive-stock.md)，你就能在[資產配置](asset-allocation.md)時做出更好的搭配——景氣循環股追求超額報酬，防禦型股票穩定軍心。
