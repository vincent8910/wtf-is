# MACD — 看趨勢是在走強還是走弱的指標

> **白話說：** 就像車子的**加速度儀表板**——不只告訴你車在前進還是後退，還告訴你「加速中」還是「減速中」。

---

## 它到底是什麼？

MACD 的全名是 **Moving Average Convergence Divergence（移動平均收斂擴散指標）**。名字很嚇人，但概念不難：

它用兩條[均線](ma.md)之間的「差距」來判斷趨勢的強弱。

MACD 圖上你會看到三樣東西：
- **MACD 線（快線）**：短期均線和長期均線的差距
- **訊號線（慢線）**：MACD 線的平均值
- **柱狀圖**：快線和慢線之間的差距，畫成直條

簡單判讀：

| 觀察 | 意思 |
|------|------|
| MACD 線在 0 以上 | 短期趨勢比長期強 → 偏多（偏漲） |
| MACD 線在 0 以下 | 短期趨勢比長期弱 → 偏空（偏跌） |
| 柱狀圖越來越高 | 上漲的力道在增強 |
| 柱狀圖越來越矮 | 上漲的力道在減弱（雖然還在漲，但快沒力了） |
| 快線往上穿越慢線 | 黃金交叉 → 可能要漲 |
| 快線往下穿越慢線 | 死亡交叉 → 可能要跌 |

## 生活比喻 / 實際例子

想像你在騎腳踏車上坡：

- **MACD 在 0 以上且柱狀圖越來越高**：你正在上坡，而且踩得越來越快 → 強勢上漲
- **MACD 在 0 以上但柱狀圖越來越矮**：你還在上坡，但開始踩不動了 → 漲勢趨緩，小心
- **MACD 在 0 以下且柱狀圖越來越長**：你正在下坡，而且越滑越快 → 跌勢加速
- **MACD 在 0 以下但柱狀圖越來越短**：你還在下坡，但快到平地了 → 跌勢趨緩，可能要止跌

MACD 的精髓就是：**不只看方向，更看力道的變化。**

實際會聽到的話：
- 「**MACD 黃金交叉了**，趨勢轉多」
- 「雖然還在漲，但 **MACD 柱狀圖在縮小**，動能在減弱」
- 「**MACD 在零軸以下**，大趨勢還是偏空」

## 為什麼要知道這個詞？

MACD 是技術分析三大常用指標之一（另外兩個是 [KD](kd.md) 和 [RSI](rsi.md)）。

它最大的特點是能看出「趨勢轉折的前兆」——就像車子開始減速，雖然還在前進，但你知道快要停了。MACD 就是在幫你捕捉這個「快要變盤」的時機。
