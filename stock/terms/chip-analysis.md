# 籌碼 — 追蹤誰在買、誰在賣的分析方法

> **白話說：** 就像看**誰在掃貨、誰在出清**——不只看價格漲跌，更要看是哪些人在買、哪些人在賣，推測接下來會怎麼走。

---

## 它到底是什麼？

「籌碼」在股市裡指的是「股票握在誰手上」。籌碼分析就是研究「大戶、法人、散戶各持有多少股票，最近又在買還是在賣」。

核心觀念：**股票在誰手上，決定了股價的走向。**

```
情境 A：
[法人](institutional-investors.md) 持續買進 → 籌碼集中到大戶手上
→ 散戶手上的股票越來越少 → 賣壓變小 → 股價容易漲

情境 B：
法人持續賣出 → 籌碼分散到散戶手上
→ 散戶容易恐慌賣出 → 賣壓變大 → 股價容易跌
```

常見的籌碼指標：

| 指標 | 看什麼 | 怎麼解讀 |
|------|--------|---------|
| 三大法人買賣超 | [外資、投信、自營商](institutional-investors.md)的進出 | 法人買超 → 偏多 |
| [融資](margin-trading.md)餘額 | 散戶借錢買股的量 | 融資大增 → 散戶太樂觀 |
| [融券](short-selling.md)餘額 | 放空的量 | 融券大增 → 有人賭會跌 |
| 大戶持股比例 | 持有 400 張以上的人佔多少 | 大戶持股增加 → 籌碼集中 |
| 董監事持股 | 公司內部人有沒有在賣 | 內部人賣股 → 要小心 |

## 生活比喻 / 實際例子

想像你在夜市觀察一個攤位：

- **技術分析**（看 [K 線](k-line.md)）：看這攤的價格怎麼變化
- **基本面分析**（看 [EPS](eps.md)）：看這攤的食材品質好不好
- **籌碼分析**：看排隊的是什麼人——

  - 如果排隊的都是**美食部落客和餐廳主廚**（法人）→ 這攤大概真的好吃
  - 如果排隊的都是**看到排隊就跟著排的路人**（散戶跟風）→ 可能只是從眾效應
  - 如果美食家們開始**悄悄離開**去別攤（法人賣超）→ 可能這攤要退燒了

看「誰在排隊」比看「隊伍多長」更有資訊量。

實際會聽到的話：
- 「這支股票**籌碼很集中**，大戶在鎖貨」
- 「**外資買超**連續十天，籌碼面很強」
- 「**融資暴增**，散戶在追高，我反而要小心」
- 「看**籌碼面**比看技術面更準」

## 為什麼要知道這個詞？

籌碼分析是台灣散戶非常重視的分析方法（可能比其他國家更重視）。因為台灣股市有完整的法人買賣超、融資融券等公開資訊，讓你可以清楚看到「誰在買誰在賣」。

簡單的原則：
- **跟著法人走**比跟著散戶走更安全
- **籌碼集中**（大戶在買）的股票通常比較強勢
- **籌碼分散**（散戶在追）的股票通常快到頂了

搭配[成交量](volume.md)和[法人動向](institutional-investors.md)一起看，你就能對市場有更深入的理解。
