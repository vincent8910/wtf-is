# 藍籌股（Blue Chip） — 大型績優龍頭股

> **白話說：** 就是股市裡的**台積電、鴻海這種大咖**——公司規模大、賺錢穩定、名氣響亮，是市場裡最「安全」的選擇。

---

## 它到底是什麼？

藍籌股指的是那些**規模大、獲利穩、歷史久、品牌響**的龍頭公司股票。

「藍籌」這個名字來自賭場——在撲克牌賭局裡，**藍色籌碼是面額最大的**，所以拿來形容股市裡最有價值、最穩健的股票。

藍籌股的特徵：

```
✅ 市值大（通常是產業龍頭）
✅ 獲利穩定（不會大起大落）
✅ 長期經營（不是剛成立的新公司）
✅ 品牌知名（大家都聽過這間公司）
✅ 通常會配股息（穩定發錢給股東）
✅ 流動性好（買賣容易，不怕找不到人交易）
```

台灣常見的藍籌股：

| 產業 | 代表公司 | 為什麼是藍籌 |
|------|---------|-------------|
| 半導體 | 台積電 | 全球晶圓代工龍頭 |
| 電子代工 | 鴻海 | 全球最大電子代工廠 |
| 金融 | 國泰金、富邦金 | 台灣最大金控集團 |
| 電信 | 中華電信 | 全台最大電信業者 |
| 塑化 | 台塑 | 台灣石化龍頭 |

美國的藍籌股就是道瓊指數裡的那些公司：蘋果、微軟、可口可樂、嬌生、麥當勞等等。

不過要注意：藍籌股「穩」不代表「不會跌」。它只是相對於小型股來說，波動比較小、倒閉的風險比較低。

## 生活比喻 / 實際例子

想像你要選一家餐廳吃飯：

- **藍籌股** = 去**鼎泰豐**吃飯。品質穩定、不太會踩雷、價格不便宜但物有所值、不會突然倒閉。你帶客戶去不會丟臉
- **小型股** = 去巷口新開的小店吃飯。可能超好吃（大漲），也可能超難吃（大跌），甚至可能下個月就倒了（下市）

或者用便利商店比喻：
- **藍籌股** = 7-11、全家。到處都有、穩定經營、不會突然消失
- **小型股** = 社區裡的獨立小店。可能很有特色，也可能開不久就收了

實際會聽到的話：
- 「新手投資先從**藍籌股**下手，比較穩」
- 「台積電是台灣最大的**藍籌股**」
- 「我的投資組合 70% 放**藍籌股**，30% 買成長股」
- 「**藍籌股**雖然漲得慢，但也跌得少，適合長期持有」
- 「存股族最愛的就是**藍籌股**加高股息」

## 為什麼要知道這個詞？

藍籌股是投資的「基本款」，就像衣櫃裡的白襯衫——不花俏但很實用。

對投資新手來說，藍籌股是最安全的起點。你不需要會看[技術分析](k-line.md)、不需要追蹤[法人動向](institutional-investors.md)，只要選幾間你認識的大公司，長期持有，通常不會太差。

很多退休基金、保險公司的大筆資金，也都是放在藍籌股裡面。因為這些錢承受不起虧損，需要最穩健的投資標的。如果連藍籌股都崩了，那代表整個經濟都出大問題了。
