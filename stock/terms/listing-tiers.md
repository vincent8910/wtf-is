# 興櫃 / 上櫃 / 上市 — 台灣股票的三個交易等級

> **白話說：** 就像賣東西的場地有等級——**夜市攤位（興櫃）→ 商場櫃位（上櫃）→ 百貨公司專櫃（上市）**，等級越高、門檻越嚴格、能見度也越高。

---

## 它到底是什麼？

在台灣，一間公司要讓股票能夠被大家公開買賣，有三個不同等級的「舞台」：

```
興櫃（最初階）
  ↓ 升級
上櫃（中階）
  ↓ 升級
上市（最高階）
```

### 興櫃（Emerging Stock Market）

- **門檻最低**：公司只要設立滿一定年限、有兩家券商推薦就可以
- **交易方式**：透過[造市商](market-maker.md)議價交易（不像上市上櫃有集中撮合）
- **特色**：很多新創公司、準備上櫃上市前的「練習階段」
- **白話**：還在夜市擺攤的階段，客人比較少、資訊也比較不透明

### 上櫃（OTC Market）

- **門檻中等**：公司獲利要達標、財報要經會計師查核、資本額要夠
- **交易方式**：在證券櫃檯買賣中心（OTC）交易
- **特色**：很多中小型公司、科技新創公司
- **白話**：已經進到商場租櫃位了，有一定的品質保證

### 上市（Listed / TWSE）

- **門檻最高**：公司規模、獲利能力、股東人數等都有嚴格要求
- **交易方式**：在台灣證券交易所（TWSE）交易
- **特色**：大型企業、產業龍頭、法人最愛投資的標的
- **白話**：百貨公司專櫃等級，品質最有保障、最多人逛

三者比較：

| | 興櫃 | 上櫃 | 上市 |
|---|---|---|---|
| 門檻 | 低 | 中 | 高 |
| 公司規模 | 小～中 | 中 | 中～大 |
| 交易量 | 少 | 中 | 多 |
| 資訊透明度 | 較低 | 中 | 最高 |
| 風險 | 最高 | 中 | 相對較低 |
| 代表指數 | 無 | 櫃買指數（OTC） | 加權指數（TAIEX） |

## 生活比喻 / 實際例子

想像你是一個服飾品牌，想把衣服賣給更多人：

- **興櫃** = 你先在**蝦皮上開賣**。門檻低、很快就能開始賣，但買家比較少、信任度也比較低。客人會想「這牌子靠譜嗎？」
- **上櫃** = 你進到了**百貨公司的特賣會場**。有一定的品質篩選，客人比較多也比較信任，但還不是最好的位置
- **上市** = 你在**百貨公司一樓拿到專櫃**。門面最好、人流最大、品牌形象最高。客人看到你在一樓就知道你是大品牌

或者用學校來比喻：
- **興櫃** = 社團活動（參加門檻低，隨時可以加入）
- **上櫃** = 校隊（要通過選拔，有一定水準）
- **上市** = 國家代表隊（最高等級，代表最強的一群）

實際會聽到的話：
- 「那間公司現在還在**興櫃**，還沒**上市**」
- 「他們下個月要從**上櫃**轉**上市**了，算是升級」
- 「**興櫃**股票風險比較高，但如果選對了，等它**上市**就會大漲」
- 「我只買**上市**股票，比較安全透明」
- 「**櫃買指數**今天漲了，代表中小型股表現不錯」

## 為什麼要知道這個詞？

在台灣投資股票，你一定會遇到「這支是上市還是上櫃？」的問題。

對新手來說，**上市股票**是最安全的選擇——這些公司規模大、資訊透明、交易量大（容易買賣）。你在券商 APP 上看到的台灣前 50 大公司（0050 成分股），全部都是上市公司。

**上櫃股票**適合想找成長機會的投資人——很多科技新創、生技公司在上櫃市場，可能成長性更高但風險也更大。

**興櫃股票**風險最高，比較適合有經驗的投資人。但如果你看準了一間公司，在興櫃時就買進，等它一路升級到上市，股價可能翻好幾倍。就像在品牌還沒紅之前就先入手，等它變成名牌就賺到了。
