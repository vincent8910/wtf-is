# 配息 / 填息 — 公司發現金給你，然後股價要漲回來才算真的賺到

> **白話說：** **配息**就像公司發年終獎金——公司把賺的錢分一些給股東。**填息**就像年終發完後，你的薪水漲回原本的水準。配息不難懂，但要搞懂「為什麼配息後股價會跌」才是重點。

---

## 它到底是什麼？

### 配息（Dividend）

配息就是**公司把賺到的錢，分一部分現金給股東**。

例如某公司宣布配息 2 元，你手上有 1,000 股，你就會收到 2,000 元現金。就像公司說：「今年賺錢了，分紅給大家。」

### 但是！配息後股價會等值下跌

這是很多人搞不懂的地方：

```
配息前：
├─ 股價 50 元
├─ 公司宣布配息 2 元
│
配息後（除息日）：
├─ 你收到 2 元現金
├─ 但股價會從 50 元自動調整為 48 元
└─ 你的「總資產」其實沒變（48 元股票 + 2 元現金 = 50 元）
```

為什麼？因為公司把錢發出去了，公司的價值就變少了，股價自然要等值下降。所以配息其實是「左口袋換到右口袋」，你的總財富沒有增加。

### 填息（Dividend Fill）

那什麼時候才算真的賺到？就是**填息**——股價從除息後的低點漲回到除息前的價格。

```
除息前股價：50 元
配息 2 元後股價：48 元
─────────────────────
填息 = 股價從 48 元漲回 50 元
→ 這時候你才是真的多賺了 2 元！

如果股價一直沒漲回 50 元（叫「貼息」）
→ 你收到 2 元現金，但股票虧更多
→ 等於賠了
```

## 生活比喻 / 實際例子

想像你有一隻會下金蛋的雞：

- **配息** = 雞下了一顆金蛋（你拿到現金）
- **除息** = 雞下蛋後變瘦了（股價下跌，因為營養被蛋帶走了）
- **填息** = 雞又吃胖回來了（股價漲回原本的價格）
- **貼息** = 雞一直沒吃胖回來（股價跌了就沒漲回去）

所以重點不是「雞下了幾顆蛋」，而是「雞下完蛋之後還能不能吃回來」。

**造句**：
- 「0056 今年**配息** 2.2 元，殖利率大概 5%」
- 「除息完一個禮拜就**填息**了，算是蠻快的」
- 「這支股票**配息**很高，但年年**貼息**，等於白配」
- 「**存股族**最開心的就是**配息**領到手軟，而且都有**填息**」

## 為什麼要知道這個詞？

- 台灣的[存股](stock-accumulation.md)風氣很盛，大家最關心的就是配息多少
- 很多人以為「配息就是多賺的錢」——錯！要填息才算真的賺到
- 高股息 ETF（0056、00878、00940）每次配息都是大新聞，一定會看到這些詞
- 搞懂配息和填息的關係，你才不會被「高殖利率」的假象吸引，買到年年貼息的股票
