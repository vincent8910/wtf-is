# 分散投資 (Diversification) — 不要把雞蛋放在同一個籃子

> **白話說：** 就像**出門不要把證件和錢包都放同一個包**——萬一那個包弄丟了，你什麼都沒了。投資也一樣，分散放在不同地方，風險就分散了。

---

## 它到底是什麼？

分散投資的概念很直白：**不要把所有的錢都押在同一個地方**。具體來說，可以從好幾個層面來分散：

| 分散層面 | 白話意思 | 舉例 |
|---------|---------|------|
| **個股分散** | 不只買一支股票 | 台積電 + 中華電 + 統一超 |
| **產業分散** | 不只買同一個產業 | 科技 + 金融 + 民生消費 |
| **市場分散** | 不只買同一個國家 | 台股 + 美股 + 歐股 |
| **資產分散** | 不只買股票 | 股票 + 債券 + 現金（→ [資產配置](asset-allocation.md)） |
| **時間分散** | 不一次全部買進 | [定期定額](dca.md)分批進場 |

為什麼要分散？因為**你無法預測誰會出事**：
- 2020 年疫情 → 旅遊股暴跌，但科技股大漲
- 2022 年升息 → 科技股暴跌，但能源股大漲
- 如果你只持有一個產業，遇到那個產業的利空就完蛋

分散投資不會讓你賺最多（因為你不可能全部都壓在漲最多的那一支），但它能**保護你不會因為一次失誤就把錢全賠光**。

## 生活比喻 / 實際例子

想像你要從台北搭車去高雄，你有 4 個朋友，每人帶一部分行李：

- 如果所有行李都放在小明的車上 → 小明的車拋錨，**所有行李都拿不到**
- 如果行李分散放四台車 → 就算一台車拋錨，你還有 75% 的行李

投資也一樣：

```
❌ 不分散：100 萬全買台積電
   → 台積電跌 30%，你直接虧 30 萬

✅ 有分散：25 萬台積電 + 25 萬中華電 + 25 萬 0050 + 25 萬債券 ETF
   → 台積電跌 30%（虧 7.5 萬），但其他可能漲或持平
   → 整體可能只虧 5-10%
```

實際會聽到的話：
- 「不要把雞蛋放在同一個籃子裡——**分散投資**很重要」
- 「我的股票**分散在五個不同產業**，不怕單一利空」
- 「最簡單的分散方式就是買 [ETF](etf.md)，裡面已經幫你分散好了」
- 「**過度分散**也不好，持有太多支股票你根本顧不來」
- 「分散投資是**降低風險**，不是保證不賠錢」

## 為什麼要知道這個詞？

分散投資是投資界最基本也最重要的風險管理概念。諾貝爾經濟學獎得主馬可維茲（Markowitz）稱它為「投資中唯一的免費午餐」——你可以在不犧牲太多報酬的情況下降低風險。

對新手來說，最簡單的分散方式就是買 [ETF](etf.md)（例如 0050 裡面有台灣前 50 大公司），一次就分散到幾十支股票。再搭配[資產配置](asset-allocation.md)和[定期定額](dca.md)，你就有了一個有紀律又分散風險的投資組合。
