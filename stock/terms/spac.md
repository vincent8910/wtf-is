# SPAC（特殊目的收購公司） — 先募資上市再找公司併購的空殼

> **白話說：** 就像**先開一家空的百貨商場，再去找品牌進駐**——先把錢籌好、場地弄好，然後再去找適合的公司來合併上市。

---

## 它到底是什麼？

SPAC 的全名是 **Special Purpose Acquisition Company（特殊目的收購公司）**。

一般公司要上市的流程是：

```
傳統 IPO：
公司先經營好幾年 → 申請上市 → 通過審查 → 掛牌交易（過程 6-12 個月）

SPAC 的做法：
1. 有經驗的人（發起人）先成立一個「空殼公司」
2. 這個空殼公司先去上市募資（投資人只知道錢會拿去併購某間公司）
3. 募到錢後，有 2 年時間去尋找適合的公司
4. 找到後跟那間公司合併 → 那間公司就等於「借殼上市」了
```

白話翻譯：

```
SPAC 發起人：「大家先把錢給我，我保證會幫你們找一間好公司來投資」
投資人：「好，我信你，先給你 10 億」
（6 個月後）
SPAC 發起人：「找到了！是一家電動車新創公司，我們合併吧」
→ 那間電動車公司就直接上市了，不用走傳統 IPO 流程
```

SPAC 的優缺點：

| | 優點 | 缺點 |
|---|---|---|
| 對目標公司 | 上市速度快、確定性高 | 可能估值被壓低 |
| 對投資人 | 信任發起人的眼光、可投資未上市公司 | 不知道錢會投去哪、風險高 |
| 對發起人 | 有資金可以談併購 | 2 年內找不到就要退錢 |

SPAC 在 2020-2021 年超級火熱，但很多 SPAC 合併後的公司表現很差，所以後來 [SEC](sec-us.md) 加強了監管，市場也冷卻了不少。

## 生活比喻 / 實際例子

想像你跟朋友合資做生意：

- **傳統 IPO** = 你先開了一家滷肉飯店，生意做了 5 年很成功，然後申請到百貨美食街設櫃
- **SPAC** = 你先跟朋友說「大家先出錢，我在百貨美食街租好一個櫃位（空的），然後我會去找一家很棒的餐廳來進駐」
  - 找到好餐廳 → 大家開心，生意興隆
  - 找到爛餐廳 → 大家虧錢
  - 根本找不到 → 錢退給大家，櫃位收回

或者用更簡單的比喻：
- SPAC 就像一張**空白支票**——你相信某個人的眼光，先把錢給他，讓他去幫你挑投資標的

實際會聽到的話：
- 「那間公司不是傳統 IPO，是透過 **SPAC** 上市的」
- 「2021 年 **SPAC** 超夯，結果很多都爆掉了」
- 「這個 **SPAC** 的發起人是前 Google 高管，背景不錯」
- 「**SPAC** 合併後股價腰斬的案例太多了，要小心」
- 「**SEC** 對 **SPAC** 加強監管，現在沒以前那麼好搞了」

## 為什麼要知道這個詞？

SPAC 是美股特有的一種上市方式，在 2020-2021 年非常流行。很多電動車、太空科技、加密貨幣相關的新創公司都是透過 SPAC 上市的。

對一般投資人來說，SPAC 的風險比較高，因為：
- 你投資的時候不知道錢會被用來買哪間公司
- 合併的目標公司可能還沒有穩定獲利
- 發起人和投資人的利益不一定完全一致

如果你在美股看到有人說某間公司是「SPAC 上市」的，通常要多一份謹慎——不是說一定不好，但要更仔細研究那間被合併的公司本身的體質，而不是只看 SPAC 發起人的名氣。
