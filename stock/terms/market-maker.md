# 造市商（Market Maker） — 負責在市場上提供買賣報價的機構

> **白話說：** 就像機場的**外幣兌換所**——不管什麼時候去，都有人願意跟你買美金也願意賣美金給你，讓你隨時都能換到錢。

---

## 它到底是什麼？

股票市場要順利運作，需要有人在那邊「隨時準備買」也「隨時準備賣」，這個角色就是**造市商（Market Maker）**。

想像一個沒有造市商的世界：

```
你想賣 100 股台積電
但此刻沒有人想買 → 你賣不掉
你只能等......等到有買家出現（可能等很久）
```

有造市商的世界：

```
你想賣 100 股台積電
造市商：「我買！我出 595 元」→ 你馬上就賣掉了

有人想買 100 股台積電
造市商：「我賣！我要 596 元」→ 他馬上就買到了
```

造市商靠什麼賺錢？就是**買賣之間的價差**：

```
造市商用 595 元跟你買 → 再用 596 元賣給別人
→ 中間差 1 元就是造市商的利潤
→ 這就是「買賣價差」（Bid-Ask Spread）
```

造市商通常是大型金融機構（券商、投資銀行），它們有義務在市場上持續提供[買價和賣價](bid-ask.md)，確保投資人隨時都能交易。

造市商的功能：
- **提供流動性**：讓你想買就能買、想賣就能賣
- **縮小價差**：越多造市商競爭，買賣價差越小，對投資人越有利
- **穩定市場**：避免因為沒人交易而導致價格劇烈波動

## 生活比喻 / 實際例子

想像一個傳統市場：

**沒有造市商的市場** = 你去跳蚤市場賣舊手機。你舉著手機站在那邊，要碰巧遇到想買的人才能成交。可能站一整天都賣不掉。

**有造市商的市場** = 你去二手手機店。店家（造市商）用 8000 元跟你收購，然後用 8500 元賣給下一個客人。你不用等、馬上就能賣掉。店家賺中間的 500 元差價。

或者用更生活化的比喻：
- **造市商** = 7-11 的**ATM 提領/存款服務**
- 你想換錢？ATM 隨時都在（造市商隨時提供報價）
- 手續費就是造市商賺的價差

實際會聽到的話：
- 「這支股票的**造市商**報價很積極，買賣價差很小」
- 「冷門股沒什麼**造市商**願意做，所以流動性很差」
- 「**Market Maker** 在盤中持續掛買賣單，確保交易順暢」
- 「那斯達克的制度就是由**造市商**負責提供流動性」
- 「台灣的[興櫃](listing-tiers.md)股票也需要**造市商**來維持交易」

## 為什麼要知道這個詞？

造市商是股票市場能夠順暢運作的幕後功臣。你每次買賣股票能「秒成交」，很大一部分要感謝造市商在背後提供流動性。

對投資人來說，造市商的存在意味著：
- **熱門股票**（像台積電、蘋果）：造市商很多、競爭激烈，買賣價差極小，你交易幾乎不會被「吃價差」
- **冷門股票**：造市商少甚至沒有，買賣價差很大，你可能要用比較差的價格才能成交

所以在選擇投資標的時，除了看公司好不好，**流動性好不好**也很重要。流動性差的股票，就算你看對方向也可能賣不掉——而流動性的關鍵就是有沒有造市商在照顧。
