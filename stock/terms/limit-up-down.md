# 漲停 / 跌停 — 一天之內漲或跌到上限，不能再交易了

> **白話說：** 就像遊樂園的**身高限制**——不管你多想玩，超過那條線就不讓你玩了。股價一天最多漲或跌 10%，到了就鎖住。

---

## 它到底是什麼？

台灣股市規定：**一支股票一天最多只能漲 10% 或跌 10%**（以前一天收盤價為基準）。

- **漲停**：今天漲了 10%，到達上限，不能再往上了
- **跌停**：今天跌了 10%，到達下限，不能再往下了

```
昨天收盤：100 元
├─ 今天漲停價：110 元（+10%）← 最多只能漲到這裡
└─ 今天跌停價：90 元（-10%）← 最多只能跌到這裡
```

漲停和跌停鎖住之後的狀況：
- **漲停鎖死**：想買的人排隊等（掛買單），但沒人願意賣 → 買不到
- **跌停鎖死**：想賣的人排隊等（掛賣單），但沒人願意買 → 賣不掉

這個制度叫做「漲跌幅限制」，目的是防止股價在一天內暴漲暴跌，讓投資人冷靜一下。

## 生活比喻 / 實際例子

想像你在 Costco 搶限量商品：

- **漲停**：一款超夯的氣炸鍋，早上開門就被搶光了。你想買但完全買不到，只能排隊等明天 → 需求太大，供不應求
- **跌停**：一批快過期的食品大降價，但大家都怕過期不敢買，貨架上堆滿了都沒人拿 → 沒人要，供過於求

漲停就是「東西太搶手，搶不到」；跌停就是「東西太嚇人，沒人要」。

實際會聽到的話：
- 「台積電今天**漲停**了！直接鎖住！」
- 「這支股票連續三天**跌停**，超慘」
- 「**漲停打開了**——本來鎖住的，有人開始賣了」
- 「掛[漲停價](limit-order.md)買進」＝用今天最高允許價格去排隊

## 為什麼要知道這個詞？

漲停和跌停是台灣股市裡天天都在發生的事，你一定會在新聞裡看到「XX 股漲停」或「XX 股跌停」。

更重要的是，漲停和跌停代表的是**極端的市場情緒**：
- 漲停 ＝ 大家瘋狂看好，搶著買
- 跌停 ＝ 大家極度恐慌，搶著逃

搭配[成交量](volume.md)一起看：漲停卻沒什麼量（鎖死），代表根本沒人願意賣；漲停但量很大，代表有人在高點出貨了。
