# 利多 / 利空 — 影響股價的好消息和壞消息

> **白話說：** **利多**就像考前老師洩題——好消息讓大家信心大增，股價跟著漲。**利空**就像臨時宣布加考——壞消息讓大家嚇到，股價跟著跌。

---

## 它到底是什麼？

**利多（Bullish News）** 就是對股價有正面影響的好消息，會讓股價上漲。
**利空（Bearish News）** 就是對股價有負面影響的壞消息，會讓股價下跌。

這兩個詞可以用在個別公司，也可以用在整個市場。只要是會影響投資人信心的消息，不管大小，都可以分成利多和利空。

常見的利多消息：

```
公司層面：
├─ 營收創新高、EPS 成長
├─ 拿到大訂單
├─ 推出暢銷新產品
├─ 宣布配發高股息
└─ 被外資調升評等

大環境層面：
├─ 央行降息（借錢變便宜，資金往股市跑）
├─ 政府推出利多政策
├─ 國際景氣好轉
└─ 某產業需求大爆發
```

常見的利空消息：

```
公司層面：
├─ 獲利衰退、虧損
├─ 大客戶抽單
├─ 產品出問題、被罰款
├─ 董事長出事
└─ 被外資調降評等

大環境層面：
├─ 央行升息（借錢變貴，資金從股市跑掉）
├─ 戰爭、天災
├─ 全球景氣衰退
└─ 某產業被政策打壓
```

重點觀念：**利多出盡就是利空，利空出盡就是利多。** 意思是好消息如果都已經反映在股價上了，反而可能開始跌；壞消息如果已經跌夠了，反而可能開始漲。

## 生活比喻 / 實際例子

想像你在路上看到一間排隊名店：

- **利多** = 這間店被美食節目報導（好消息）→ 更多人來排隊 → 生意更好（股價漲）
- **利空** = 這間店被爆出食安問題（壞消息）→ 大家不敢吃 → 生意變差（股價跌）
- **利多出盡** = 這間店被報導太多次，所有人都知道了，該來的都來過了，反而開始退燒

**造句**：
- 「台積電拿到 AI 晶片大單，這是個大**利多**」
- 「美國又升息了，對股市是**利空**」
- 「這個**利空**消息已經反映在股價裡了，別再恐慌了」
- 「別追了，**利多出盡**就是利空，小心被套」

## 為什麼要知道這個詞？

- 這是財經新聞出現頻率最高的詞彙之一，幾乎每天都會看到
- 學會判斷利多和利空，你才能理解為什麼股價漲或跌
- 「利多出盡」和「利空出盡」的觀念很重要，讓你不會單純看消息好壞就追進去或殺出來
- 跟朋友聊股票時，說「這個消息是利多還是利空」是最基本的分析方式
