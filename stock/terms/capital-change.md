# 增資 / 減資 — 公司增加或減少股本

> **白話說：** 就像合資開店——**增資**是大家再掏錢出來擴大店面，**減資**是把一部分投資款退還給股東，把店面縮小。

---

## 它到底是什麼？

**增資**和**減資**是公司調整「股本」大小的動作。股本就是公司發行在外的股票總金額。

### 增資（增加股本）

公司需要更多錢的時候，會發行新股票來募資。增資有幾種方式：

```
現金增資：公司發行新股票，向股東或市場募集現金
           → 像是開店缺錢，找股東再投資一筆

盈餘轉增資：把賺到的利潤變成股票發給股東
           → 像是開店賺了錢，不發現金，而是把利潤變成更多股份

```

增資後股票總數會變多。如果公司拿到的錢能賺更多回來，對股東是好事；但如果只是亂花錢，那就是稀釋了你的股份價值。

### 減資（減少股本）

公司把一部分股票註銷掉，減資也有幾種類型：

```
現金減資：把錢退還給股東，同時註銷對應股票
         → 像是公司說「我們用不了這麼多錢，退一些給你」

彌補虧損減資：公司長年虧損，把帳面虧損打消
             → 像是記帳本太難看，重新開一本
             （這種減資股東拿不到錢，股票直接變少）
```

## 生活比喻 / 實際例子

想像你跟 4 個朋友合資開一間飲料店，每人出 100 萬，總股本 500 萬：

**增資的情況：**
- 生意太好想開分店，需要再 300 萬
- 5 個人每人再出 60 萬 → 股本從 500 萬變成 800 萬
- 如果分店也賺錢 → 大家開心，投資值得
- 如果分店賠錢 → 多投的錢都浪費了

**減資的情況：**
- 經營穩定後發現帳上多了 200 萬現金用不到
- 決定退還每人 40 萬 → 股本從 500 萬縮為 300 萬
- 店還是那間店，只是帳上的資本額變小了

實際會聽到的話：
- 「這家公司要**現金增資**了，要不要參加認購？」
- 「又**減資**了，上次虧損減資完股價也沒起色」
- 「大立光的**現金減資**是退錢給股東，跟虧損減資不一樣」
- 「**增資**太頻繁的公司要小心，可能一直在跟股東要錢」

## 為什麼要知道這個詞？

增資和減資會直接影響你手上股票的數量和價值。

現金增資的時候，公司會用低於市價的價格讓你認購新股，看起來很划算，但如果你不參加認購，你的持股比例就會被稀釋（像原本佔 10% 的店，增資後可能只剩 8%）。

減資的時候要看清楚是哪種：現金減資（退錢給你）通常是正面訊號，代表公司現金充裕；虧損減資（不退錢）通常是負面訊號，代表公司之前虧了很多需要美化帳面。看到[EPS](eps.md) 突然變好看，要先確認是不是因為減資導致分母變小。
