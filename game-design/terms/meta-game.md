# Meta Game（元遊戲）— 遊戲主要玩法之外的長期策略系統

> **白話說：** 就像便當店除了賣便當，還有集點換公仔、會員等級、限定套餐——這些「便當以外的東西」讓你一直回來光顧。

---

## 它到底是什麼？

Meta Game（元遊戲，或簡稱 Meta）是指遊戲核心玩法之外，那些讓你長期投入的系統。核心玩法可能是射擊、消除、賽車，但 Meta Game 是圍繞在外面的養成、收集、社交、賽季排名這些東西。

打個比方，一款射擊遊戲的核心玩法是「瞄準→開槍→擊倒敵人」，但你玩完一場之後為什麼還想再玩？因為你要升級角色、解鎖新武器造型、爬排行榜、完成每週任務、跟朋友組隊⋯⋯這些全部都是 Meta Game。核心玩法讓你覺得「這一場好玩」，Meta Game 讓你覺得「我明天還要上線」。

Meta Game 之所以重要，是因為核心玩法再好玩，玩了幾百場之後也會膩。Meta Game 提供了長期目標和變化，讓玩家有持續追求的動力。就像一家餐廳菜再好吃，也需要推新菜、辦活動、做會員制度來讓客人一直回來。

## 生活比喻 / 實際例子

想像你常去的早餐店。吃早餐（核心玩法）本身很滿足，但這家店還有：集滿十杯送一杯（收集系統）、LINE 群組每天猜謎送優惠（社交系統）、每月推出限定口味（賽季內容）。這些「吃早餐以外的事」就是 Meta Game，讓你不只是餓了才來，而是每天都想來。

再比如麥當勞的「大富翁」活動：你去吃麥當勞（核心行為），但撕貼紙、收集、看有沒有中獎（Meta Game）變成了你更期待的事。

**造句：**
- 「這款遊戲打起來很爽，但 Meta Game 太淺了，玩兩個禮拜就沒目標了。」
- 「他們最近更新了 Meta Game，加了公會戰和賽季排名，我又開始每天上線了。」
- 「好的 Meta Game 設計可以讓一款遊戲活好幾年。」

## 為什麼要知道這個詞？

- **你想理解為什麼有些遊戲能玩好幾年**：核心玩法讓人入門，Meta Game 讓人留下來
- **你是遊戲設計師或企劃**：Meta Game 是玩家長期留存和付費的關鍵，不能只把核心玩法做好就收工
- **你在做任何訂閱制或會員制產品**：Meta Game 的概念也適用於非遊戲產品，例如健身 App 的徽章系統、咖啡店的會員等級
- **你看到遊戲公司在討論「長線營運」**：長線營運的本質就是不斷更新和豐富 Meta Game
