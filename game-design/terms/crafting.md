# Crafting（合成 / 製作）— 把幾個材料組合在一起做出新道具

> **白話說：** 就像料理一樣——把蛋、麵粉、牛奶混在一起，就變成蛋糕了。

---

## 它到底是什麼？

Crafting 就是遊戲裡的「合成系統」。你在遊戲世界裡收集各種材料（木頭、鐵礦、草藥、皮革……），然後按照配方把它們組合在一起，就能做出新的道具。比如三塊木頭 + 兩塊鐵 = 一把劍，或是兩朵花 + 一瓶水 = 一瓶治療藥水。

Crafting 系統讓玩家有一種「自己動手做」的成就感。比起直接從商店買一把劍，自己收集材料、學會配方、親手打造出來的武器感覺就是不一樣。很多遊戲甚至讓你自己研發配方——把各種材料隨意組合看看會產生什麼東西，有點像在做實驗。

從簡單到複雜，Crafting 系統的差異很大。有些遊戲的合成很直覺：A + B = C，配方清楚列在那裡。有些遊戲的合成非常深入，你需要考慮材料品質、製作順序、甚至角色的製作等級，做出來的東西品質也會不同。

## 生活比喻 / 實際例子

最直觀的比喻就是做菜。你去超市買了蛋、培根、吐司、起司（收集材料），然後照食譜的步驟把它們組合起來（合成），最後做出一份早午餐三明治（新道具）。如果你技術好，做出來的三明治更好吃（品質更高）；如果你用高級食材，成品也會更好。

或是想想手搖飲料的客製化。基底茶 + 珍珠 + 半糖 + 少冰 = 你的專屬飲料。不同的組合會產生不同的結果，這就是一種日常生活中的 Crafting。

《Minecraft》（麥塊）大概是最知名的 Crafting 遊戲。整個遊戲的核心就是：挖材料 → 合成工具 → 用工具挖更好的材料 → 合成更好的工具。把木頭排成 T 字形就是鶴嘴鋤，排成一直線就是棍子。這種「排列組合」的樂趣讓全世界幾億人玩到停不下來。

**造句**：「我終於收集到所有材料了！可以 Craft 那把傳說級的武器了，花了我整整一個禮拜。」

## 為什麼要知道這個詞？

- 現在超多遊戲都有 Crafting 系統，從手遊到大型主機遊戲都有
- 「合成」「製作」「打造」在遊戲裡講的都是 Crafting
- 有些遊戲的 Crafting 是核心玩法（像 Minecraft），有些只是附加功能
- 理解 Crafting 機制能幫你更快上手有這個系統的遊戲
