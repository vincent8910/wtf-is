# Core Loop — 遊戲裡讓你一直重複做的那個循環

> **白話說：** 就像手搖飲的集點卡——買飲料→集點→換免費飲料→再買飲料，你就這樣一圈一圈轉下去，停不下來。

---

## 它到底是什麼？

Core Loop（核心循環）就是一款遊戲裡，玩家不斷重複做的那一組動作。每一圈做完，你會得到一點好處，讓你想再做一圈。這個循環設計得好不好，直接決定這款遊戲好不好玩、能不能讓人上癮。

最經典的例子就是：**打怪→掉寶→升級→打更強的怪**。你打完一隻怪，掉了一把新武器，角色變強了，就會想去打更難的怪。打完更難的怪又掉更好的寶，你又更強了⋯⋯就這樣一圈一圈停不下來。

不只是打怪遊戲，所有遊戲都有 Core Loop。手機遊戲「Candy Crush」的核心循環是：消糖果→過關→解鎖新關卡→繼續消。就連「開心農場」也有：種菜→等待→收成→賣錢→買更好的種子→種更多菜。只要你會一直重複做同樣的事，那就是 Core Loop。

## 生活比喻 / 實際例子

想像你去夜市玩彈珠台：投幣→彈彈珠→看結果→拿獎品→覺得差一點就中大獎→再投幣。這就是一個核心循環。

或者想想 7-11 的集點活動：消費→集點→看目錄想換什麼→差兩點快集滿了→再去消費。超商就是用這個循環讓你一直回來買東西。

**造句：**
- 「這款遊戲的 Core Loop 太無聊了，就只是一直點點點。」
- 「我們要先把 Core Loop 想清楚，確認玩家每一圈都有成就感，才開始做其他系統。」
- 「他們的核心循環設計得很好，我每次說再打一場就好，結果一打就是兩小時。」

## 為什麼要知道這個詞？

- **你是遊戲開發者**：Core Loop 是遊戲設計的地基，地基歪了整棟大樓都會倒。做任何功能之前，先把核心循環想通
- **你在玩遊戲時好奇為什麼停不下來**：因為遊戲設計師精心安排了這個循環，讓你每一圈都有「再一次就好」的感覺
- **你在做任何產品或服務**：其實不只遊戲，很多 App 也在設計自己的 Core Loop，像社群媒體的「滑→看→按讚→被通知→再回來滑」
- **你聽到遊戲公司在討論「留存率」**：留存率高不高，Core Loop 是最關鍵的因素
