# Loot Box（戰利品箱）— 遊戲裡用遊戲幣或現金買的隨機寶箱

> **白話說：** 就像百貨公司的福袋——付了錢，但打開之前你不知道裡面裝了什麼。可能賺到，也可能全是你不要的東西。

---

## 它到底是什麼？

Loot Box（戰利品箱、寶箱）是遊戲裡的一種隨機獎勵機制。你用遊戲幣或花真錢買一個箱子，打開之後隨機獲得道具、造型、裝備等等。你不能選你要什麼，開出來的東西完全靠運氣。

Loot Box 跟 Gacha（轉蛋）很像，概念都是「付錢→隨機結果」。差別在哪？主要是文化跟呈現方式不同。Gacha 源自日本，通常用「抽卡」的形式呈現（選一張牌翻過來看是什麼）；Loot Box 比較偏歐美遊戲，通常用「開箱子」的方式呈現（一個寶箱打開掉出東西來）。但本質上，兩者都是用真錢賭運氣。

Loot Box 引發了大量爭議。2017 年遊戲《星際大戰：乞力馬扎前線 2》因為 Loot Box 設計太過分（不課金幾乎沒法玩），引發全球玩家暴怒，甚至讓好幾個國家開始立法管制。比利時和荷蘭直接禁止了某些形式的 Loot Box，認為它本質上就是賭博。

## 生活比喻 / 實際例子

過年的時候百貨公司賣福袋，一袋兩千塊，裡面可能有價值五千的東西，也可能全是過季商品。你站在櫃檯前猶豫：「搞不好這袋就是大獎！」結果打開來是三雙襪子和一條你不會戴的圍巾。Loot Box 就是這個體驗的遊戲版。

或者想想夜市的戳戳樂——一面板子上面很多洞，每個洞裡面有一包東西，你戳一個不知道會拿到什麼。大獎可能是一台遊戲機，但大部分洞裡面只有糖果。遊戲的 Loot Box 也一樣，稀有好物的機率通常很低。

**造句：**
- 「這遊戲的 Loot Box 太坑了，我開了二十個箱子全是重複的貼紙。」
- 「歐洲好幾個國家已經禁止遊戲裡的 Loot Box 了。」
- 「他們改成直接購買造型，不再用 Loot Box 了，我覺得這樣比較好。」

## 為什麼要知道這個詞？

- **你有在玩歐美遊戲**：《CS2》《FIFA》《Overwatch》等大作都有（或曾經有）Loot Box 機制
- **你是家長**：了解 Loot Box 有助於判斷小孩玩的遊戲是不是在變相鼓勵賭博行為
- **你關注遊戲產業新聞**：Loot Box 的法律管制是近年遊戲產業最大的議題之一
- **你在設計任何隨機獎勵機制**：了解 Loot Box 的爭議可以幫你避開設計陷阱，也可以參考好的做法
