# WTF is...? 🤷

> **又有人在講天書了？這裡翻譯給你聽。**
>
> 工程師、分析師、設計師、行銷、管理層⋯⋯每個圈子都有自己的行話。
> 這裡用白話解釋、生活比喻、實際造句，讓你秒懂。

---

## 這是什麼？

**wtf-is** 是一份給**聽不懂專有名詞的人**看的術語翻譯手冊。

不只是軟體工程——從股票投資、電子產業、行銷企劃、美術設計到企業管理，各領域的天書都幫你翻譯好了。

每個術語都有：
- **白話解釋**：不用任何專業背景就能懂
- **生活比喻**：用你熟悉的事物來對照
- **造句示範**：讓你知道這個詞在什麼情境下會出現

---

## 怎麼用？

1. **找到你不懂的詞** → 看速查表的連結
2. **還是不懂？** → 點進去看完整頁面
3. **想深入某個領域？** → 點領域名稱進入專屬頁面

### 📱 加到手機桌面，像 App 一樣用

這個專案有一個[搜尋網頁](https://vincent8910.github.io/wtf-is/)，可以加到手機桌面當捷徑，開會時隨手查。

**iPhone / iPad（用 Safari 開啟）：**
1. 用 **Safari** 打開 [搜尋頁面](https://vincent8910.github.io/wtf-is/)
2. 點底部的**分享按鈕**（方框 + 向上箭頭 ↑）
3. 往下滑找到 **「加入主畫面」**，點它
4. 點右上角 **「新增」**，完成！

**Android（用 Chrome 開啟）：**
1. 用 **Chrome** 打開 [搜尋頁面](https://vincent8910.github.io/wtf-is/)
2. 點右上角 **⋮ 選單**
3. 點 **「加到主畫面」** 或 **「安裝應用程式」**
4. 確認後就會出現在桌面上了

> 加到桌面後，打開就是全螢幕（不會有瀏覽器網址列），而且支援離線使用。

## 如何貢獻

歡迎任何人幫忙讓這份手冊更好：

- **發現解釋不清楚？** 開一個 [Issue](https://github.com/vincent8910/wtf-is/issues) 告訴我們
- **想新增術語？** Fork 後在對應領域的 `terms/` 資料夾加一個 `.md` 檔，然後發 PR
- **比喻不夠生活化？** 直接建議更好的比喻

```markdown

> 一句話解釋

## 生活比喻
用大家都懂的事物來比喻
```
