# Fabless — 只設計不製造的晶片公司，像設計師不開工廠

> **白話說：** Fabless 就像**服裝設計師**——只負責畫設計圖，衣服交給成衣工廠去做。晶片設計好了，交給台積電去製造。

---

## 它到底是什麼？

**Fabless** 的 Fab 是 Fabrication（製造）的縮寫，-less 是「沒有」的意思。所以 Fabless 就是「**沒有工廠**」的意思。

Fabless 公司專注在 [IC 設計](ic-design.md)，設計好晶片之後，交給 [Foundry（代工廠）](foundry.md) 來製造。這樣做的好處是：

- **省下蓋工廠的錢**：一座先進製程的晶圓廠要花幾千億台幣
- **專注設計創新**：不用操心製造的事
- **彈性大**：可以根據需求選不同的代工廠

全球知名的 Fabless 公司：

```
輝達（NVIDIA）   → 設計 GPU，交給台積電做
高通（Qualcomm） → 設計手機晶片，交給台積電 / 三星做
聯發科（MediaTek）→ 設計手機 / 電視晶片，交給台積電做
AMD              → 設計 CPU 和 GPU，交給台積電做
蘋果（Apple）    → 設計 A 系列 / M 系列晶片，交給台積電做
```

相對的，像 Intel、三星，自己設計也自己製造，叫做 **IDM（整合元件製造商）**。

## 生活比喻 / 實際例子

想像餐飲業的分工：

- **Fabless** = 研發食譜的美食顧問（只設計菜色，不開餐廳）
- **[Foundry](foundry.md)** = 中央廚房（專門照食譜把菜做出來）
- **IDM** = 自己開餐廳又自己做菜的老闆（像鼎泰豐，研發到製作全包）

這就是為什麼台積電這麼重要——全世界大部分的 Fabless 公司，都把晶片交給台積電做。

實際你會聽到的說法：
- 新聞：「NVIDIA 是全球最大的 **Fabless** 晶片公司」
- 分析師：「**Fabless** 模式讓 IC 設計公司可以專注在創新」
- 投資人：「這家 **Fabless** 公司毛利率很高，因為不用養工廠」

## 為什麼要知道這個詞？

- 理解半導體產業鏈的分工方式，Fabless 是其中一個重要角色
- 台灣很多上市公司是 Fabless 模式（聯發科、瑞昱、聯詠⋯⋯）
- 搞懂 Fabless 和 [Foundry](foundry.md) 的關係，你就懂了半導體產業最核心的商業模式
