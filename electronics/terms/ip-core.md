# IP Core — 別人設計好的晶片「零件」，可以直接拿來用

> **白話說：** IP Core 就像 **IKEA 的系統廚具**——別人已經設計好、測試過的標準模組，你直接買來裝進自己的房子（晶片）裡就好，不用從零開始做。

---

## 它到底是什麼？

**IP Core** 的 IP 是 **Intellectual Property（智慧財產）** 的縮寫，Core 是「核心」的意思。

在晶片設計中，IP Core 是指**別人已經設計好、驗證過的電路功能區塊**，你可以花錢授權（買使用權）之後，直接放進自己的晶片設計裡。

為什麼需要 IP Core？因為從零開始設計一顆晶片的所有功能，既花時間又花錢。如果某些標準功能（像是 USB 介面、Wi-Fi 模組、記憶體控制器）已經有人做好了，直接買來用就好。

```
自己設計所有功能 → 花 3 年、花 30 億
用 IP Core 組裝   → 花 1 年、花 10 億
```

最有名的 IP Core 公司是 **Arm**——全世界大部分的手機晶片（包括蘋果、高通、聯發科），裡面的 CPU 核心都是跟 Arm 買授權的。

## 生活比喻 / 實際例子

想像你要裝潢一間新房子：

- **從零設計** = 自己找木工師傅，從一塊木頭開始做所有家具
- **用 IP Core** = 去 IKEA 買現成的系統櫃、系統廚具、浴室設備，直接裝上去
- IKEA（Arm）賣的是**設計授權**，你付錢就能用他們的標準產品
- 你可以在 IKEA 家具（IP Core）的基礎上，加上自己的特色設計

實際你會聽到的說法：
- 新聞：「Arm 的 CPU **IP** 授權費用持續成長」
- 法說會：「我們的新晶片整合了多個第三方 **IP Core**，加速開發時程」
- 分析師：「使用 **IP Core** 可以把 [Tape-out](tape-out.md) 風險降到最低」

## 為什麼要知道這個詞？

- Arm 是全球最重要的 IP Core 公司之一，它的商業模式就是賣 IP 授權
- 搞懂 IP Core，你就理解為什麼 Arm 自己不做晶片，卻能影響整個手機產業
- [IC 設計](ic-design.md) 公司的成本結構中，IP 授權費是重要的一環
- 也能理解為什麼美中科技戰中，IP 授權的限制是一個重要的制裁手段
