# Foundry（代工） — 專門幫別人製造晶片的工廠

> **白話說：** Foundry 就像**印刷廠**——你把書的稿子交過去，印刷廠幫你印出來。台積電就是全世界最大的晶片印刷廠。

---

## 它到底是什麼？

**Foundry（晶圓代工廠）** 是專門負責「製造晶片」的工廠。它自己不設計晶片，而是接受 [Fabless（IC 設計公司）](fabless.md) 的設計圖，用自己的 [製程技術](process-node.md) 把晶片做出來。

台積電（TSMC）就是全球最大的 Foundry，佔了全球晶圓代工市場超過一半的份額。

Foundry 的核心競爭力：

```
製程技術      → 能做到多小（3nm、5nm⋯⋯）
良率          → 做出來的合格率有多高
產能          → 每個月能做幾萬片晶圓
客戶信任      → 不會偷客戶的設計
```

## 生活比喻 / 實際例子

想像一個超大型的**代工便當工廠**：

- 各家餐廳（[Fabless 公司](fabless.md)）把自己的食譜和配方（晶片設計圖）交過來
- 便當工廠（Foundry）照著食譜，用自己的廚房設備（製程技術）把便當做出來
- 工廠保證不會把 A 餐廳的食譜偷偷給 B 餐廳
- 哪家工廠的設備最好、做出來品質最穩定，就最多人排隊下單

台積電就是那個設備最好、品質最穩、客戶最多的便當工廠。NVIDIA、Apple、AMD 全部排隊找它做。

實際你會聽到的說法：
- 新聞：「台積電是全球最大的晶圓**代工**廠」
- 法說會：「**Foundry** 產能利用率本季回升至 90%」
- 分析師：「三星 **Foundry** 的良率仍落後台積電」

## 為什麼要知道這個詞？

- 台積電是台灣最重要的公司之一，搞懂 Foundry 就搞懂台積電在做什麼
- [Fabless](fabless.md) + Foundry 的分工模式，是整個半導體產業的核心商業邏輯
- 看半導體新聞、法說會，Foundry 這個詞幾乎每次都會出現
