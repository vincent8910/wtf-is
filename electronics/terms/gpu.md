# GPU — 專門做大量計算的處理器，AI 訓練靠它

> **白話說：** GPU 就像一間有 **1000 個員工的工廠**——每個員工只會做簡單的工作，但 1000 個人同時做，速度超快。AI 訓練需要同時做大量相似的計算，所以 GPU 成了 AI 時代最搶手的晶片。

---

## 它到底是什麼？

**GPU** 是 **Graphics Processing Unit（圖形處理器）** 的縮寫。

GPU 最初是設計來處理「畫面」的——遊戲裡面的 3D 場景，每一幀畫面都需要同時計算幾百萬個像素的顏色和位置。GPU 擅長的就是「**同時做大量簡單的計算**」。

後來大家發現：AI 訓練也是這種工作模式——需要對大量資料做同樣的數學運算。所以 GPU 從「遊戲顯示卡」變成了「AI 訓練的核心」。

CPU 和 GPU 的差別：

```
CPU（中央處理器）         GPU（圖形處理器）
├── 4-16 個核心          ├── 幾千到幾萬個核心
├── 每個核心很強         ├── 每個核心比較弱
├── 像 1 個博士生        ├── 像 10000 個小學生
├── 適合複雜的任務       ├── 適合大量簡單的任務
└── 一次做一件大事       └── 同時做一萬件小事
```

**NVIDIA** 是 GPU 市場的霸主，尤其在 AI 領域，NVIDIA 的資料中心 GPU（H100、B100 系列）佔了超過 80% 的市場份額。

## 生活比喻 / 實際例子

想像你要數一整間倉庫裡的貨物：

- **CPU 的做法** = 派 1 個超強的倉管，一件一件仔細清點（準確但慢）
- **GPU 的做法** = 派 10000 個工讀生，每人負責數一小區（沒那麼精確但超快）

AI 訓練就是這種「大量重複計算」的工作，GPU 的人海戰術剛好完美勝任。

實際你會聽到的說法：
- 新聞：「NVIDIA **GPU** 供不應求，全球科技大廠搶購」
- 法說會：「我們已部署 10000 顆 **GPU** 用於 AI 訓練」
- 分析師：「**GPU** 需求帶動台積電先進製程和 [CoWoS](cowos.md) 封裝訂單」

## 為什麼要知道這個詞？

- GPU 是 AI 時代最核心的硬體，NVIDIA 因此成為全球市值最高的公司之一
- NVIDIA 的 GPU 由台積電代工、用 [HBM](hbm.md) 記憶體、用 [CoWoS](cowos.md) 封裝——整個供應鏈都受惠
- 很多公司開始自研 [ASIC](asic.md) 來跟 GPU 競爭（如 Google 的 TPU）
- 搞懂 GPU，你就理解為什麼 AI 產業的成長跟 NVIDIA 綁得這麼緊
