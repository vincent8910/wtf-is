# 良率（Yield） — 做出來的晶片裡面有多少比例是合格的

> **白話說：** 良率就像夜市射氣球的命中率——射了 100 發，中了 95 發，良率就是 95%。晶片工廠做了一片 [晶圓](wafer.md)，上面合格的晶片越多，良率越高。

---

## 它到底是什麼？

**良率（Yield）** 就是「合格率」——從一片 [晶圓](wafer.md) 上切下來的晶片裡面，有多少比例是可以正常使用的。

製造晶片的過程非常精密，任何一個微小的灰塵顆粒、任何一個步驟的偏差，都可能讓晶片報廢。所以不可能做到 100% 都合格。

良率的重要性：

```
良率高（>90%）  → 做 100 顆，90 顆以上能用 → 成本低，賺錢
良率低（<50%）  → 做 100 顆，只有不到 50 顆能用 → 成本超高，可能賠錢
```

良率直接影響成本和獲利。同樣一片晶圓的製造成本是固定的，良率越高，每顆合格晶片分攤到的成本越低。

## 生活比喻 / 實際例子

想像你在做**雞蛋糕**：

- 一盤模具可以做 30 顆雞蛋糕
- 做出來有 3 顆燒焦、2 顆沒熟 → 良率 = 25/30 = **83%**
- 如果你手藝越來越好，燒焦的越來越少 → 良率就越來越高
- 良率高 = 同樣的材料錢，做出更多能賣的雞蛋糕 = **更賺錢**

用 [製程節點](process-node.md) 的角度來說：
- 越先進的製程（3nm、5nm），電路越精密，越容易出錯 → **初期良率低**
- 經過不斷調整、改善 → 良率逐漸提升 → 這叫「良率爬坡」
- 良率爬坡成功 → 才能大量生產、才划算

實際你會聽到的說法：
- 法說會：「3 奈米**良率**已達量產水準」
- 新聞：「三星 3nm **良率**不佳，導致客戶轉單台積電」
- 分析師：「**良率**每提升 1%，毛利率就能改善 0.5%」

## 為什麼要知道這個詞？

- 良率是半導體製造最核心的指標之一
- 台積電之所以厲害，最大的護城河之一就是良率領先競爭對手
- 看半導體公司的財報、法說會，良率幾乎一定會被提到
- 良率低 → 成本高 → 晶片貴 → 最終消費者也會受影響（手機更貴）
