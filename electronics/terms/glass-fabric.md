# 玻纖布 — PCB 裡面用來加強結構的材料

> **白話說：** 玻纖布就像蓋房子用的**鋼筋**——[PCB 電路板](pcb.md) 的板子如果只有樹脂（膠），會軟趴趴的。加上玻纖布之後，板子才會硬挺、不容易彎折斷裂。

---

## 它到底是什麼？

**玻纖布（Glass Fabric / Fiberglass Cloth）** 是用極細的玻璃纖維編織而成的布料。

[PCB 電路板](pcb.md) 的製造過程是這樣的：

```
玻纖布 + 樹脂（環氧樹脂） → 浸泡、壓合 → 變成「基板材料（CCL）」
  ↓
在基板材料上面做銅線電路 → 變成 PCB 電路板
```

所以玻纖布就是 PCB 板子裡面的「骨架」，功能是：
- **加強硬度**：讓板子不會軟趴趴
- **耐熱**：焊接零件時溫度很高，不能融化
- **絕緣**：防止電路之間短路
- **穩定尺寸**：不會因為溫度變化而變形

台灣主要的玻纖布公司：台灣玻璃、富喬、德宏。

## 生活比喻 / 實際例子

想像你在做**玻璃纖維釣竿**：

- 如果只用樹脂（膠）做 → 太脆、容易斷
- 加上玻璃纖維的布 → 變得又輕又強韌
- **玻纖布** 就是那個讓東西「從脆弱變堅固」的材料

或者想像**鋼筋混凝土**：
- 混凝土（樹脂）本身雖然硬但容易裂
- 加上鋼筋（玻纖布）之後 → 既硬又有韌性
- PCB 裡面的玻纖布就扮演「鋼筋」的角色

實際你會聽到的說法：
- 新聞：「**玻纖布** 報價上漲，帶動 CCL 基板成本增加」
- 分析師：「高階 PCB 需要更細、更密的 **玻纖布**」
- 產業報告：「5G 和 AI 帶動高頻 **玻纖布** 需求」

## 為什麼要知道這個詞？

- 玻纖布是 PCB 的上游原材料，PCB 又是所有電子產品的基礎
- 玻纖布價格波動會影響整個 PCB 產業鏈的成本
- 高頻、高速的 PCB（用在 AI、5G）需要特殊的玻纖布，是新的成長領域
- 搞懂玻纖布，你就理解 PCB 產業鏈的上游是什麼
