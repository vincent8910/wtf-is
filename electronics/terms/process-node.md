# 製程節點（3nm / 5nm） — 數字越小代表晶片上的電路越精密，越省電越快

> **白話說：** 製程節點就像筆的粗細——**筆尖越細，同樣一張紙上就能寫越多字**。3nm 就是目前最細的筆，能在同樣大小的晶片上塞進更多電路。

---

## 它到底是什麼？

你常在新聞看到「台積電 3 奈米製程」、「蘋果 A17 用 3nm」——這個 **nm（奈米）** 數字，就是在講**製造晶片時能刻出多細的電路線條**。

1 奈米 = 十億分之一公尺，小到完全看不見。

重點觀念：

- **數字越小** → 電路越精密 → 同樣面積塞更多電晶體
- **電晶體越多** → 效能越強、越省電
- **但越難做** → 成本越高、技術門檻越高

目前的進度大概是這樣：

```
28nm → 比較舊，便宜，用在車用晶片、物聯網
 7nm → 幾年前的高階，現在變主流
 5nm → 目前的高階（iPhone 15 的晶片）
 3nm → 目前最先進（iPhone 16 的晶片）
 2nm → 正在開發中，下一代
```

## 生活比喻 / 實際例子

想像你在一張 A4 紙上寫字：

- **28nm 製程** = 用麥克筆寫，字很大，一頁寫不了幾個字
- **7nm 製程** = 用原子筆寫，字變小了，能寫更多
- **3nm 製程** = 用最細的代針筆寫，同一頁塞了超多內容
- **2nm 製程** = 用顯微鏡才看得到的筆在寫

紙張大小沒變（晶片面積差不多），但寫進去的內容（電路）越來越多。

實際你會聽到的說法：
- 法說會：「我們的 **3 奈米製程**良率已達量產水準」
- 新聞：「蘋果最新晶片採用台積電 **3nm 製程**，效能提升 15%」
- 投資人：「**先進製程**的毛利率比成熟製程高很多」

## 為什麼要知道這個詞？

- 這是判斷一家晶片公司技術水準的核心指標
- 台積電、三星、Intel 三家公司就是在比誰先做到更小的製程
- [IC 設計](ic-design.md)公司選用什麼製程，直接影響產品的效能和成本
- 投資人看半導體股票，製程節點是必懂的基本面
