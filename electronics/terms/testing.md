# 測試（Testing） — 檢查每顆晶片能不能正常運作

> **白話說：** 測試就像工廠的**品管關卡**——每顆晶片做好之後，都要經過一連串的檢查，確認沒有問題才能出貨。就像便利商店進貨前會檢查食品有沒有過期一樣。

---

## 它到底是什麼？

**測試（Testing）** 是半導體製造流程中不可或缺的環節。晶片從 [晶圓](wafer.md) 製造出來之後，在 [封裝](packaging.md) 前後都要做測試。

測試分成兩個階段：

```
晶圓測試（Wafer Test / CP Test）
├── 在晶圓還沒切割前，先用探針一顆顆戳，測每顆 Die 有沒有問題
├── 壞掉的做記號，之後不用浪費錢去封裝它
└── 又叫「中測」

最終測試（Final Test / FT）
├── 封裝完之後，再做一次全面測試
├── 測功能、測速度、測耐溫、測各種極端條件
└── 通過才能出貨
```

測試聽起來簡單，但其實非常複雜：
- 一顆晶片可能有幾十億個電晶體，每個都要驗證
- 測試設備（叫「測試機台」）非常昂貴
- 測試時間越長、成本越高，所以如何「又快又準」地測是一門學問

台灣主要的測試公司：日月光（ASE）、矽品、京元電子等。

## 生活比喻 / 實際例子

就像汽車出廠前的品質檢查：

- **晶圓測試** = 車子組裝到一半時先檢查引擎有沒有問題（壞的就不繼續裝了）
- **最終測試** = 車子全部組裝好之後，開到測試跑道上跑一圈
- 測煞車、測加速、測冷氣、測各種電子設備 → 全部通過才能賣給消費者
- 測試不合格的 = 報廢或降級（有些小問題的晶片會降級為低階版本賣）

實際你會聽到的說法：
- 新聞：「日月光的封裝**測試**產能滿載」
- 法說會：「AI 晶片的**測試**時間較長，影響整體產出」
- 分析師：「**測試**成本佔先進製程晶片總成本的比重越來越高」

## 為什麼要知道這個詞？

- 測試是半導體產業鏈中不可跳過的環節，直接影響 [良率](yield.md) 和成本
- 封裝測試（封測）公司是台股半導體族群的重要板塊
- AI 晶片因為更複雜，測試需求和成本都大幅增加
- 搞懂測試，你就理解為什麼有些晶片會有「降頻版」或「閹割版」——測試不完全通過的就降級賣
