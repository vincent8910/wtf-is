# DRAM — 電腦的短期記憶體，斷電就忘光

> **白話說：** DRAM 就像你的**工作桌面**——你正在處理的文件都攤在桌上，隨手就能拿到（速度快）。但下班把東西收走後，桌上就空了（斷電就消失）。

---

## 它到底是什麼？

**DRAM** 是 **Dynamic Random Access Memory（動態隨機存取記憶體）** 的縮寫。

它是電腦、手機裡面的「**短期記憶體**」——程式正在執行的時候，資料就暫時放在 DRAM 裡面。

DRAM 的特性：
- **速度快**：讀寫資料非常快，讓程式跑得順
- **斷電就消失**：一關機，DRAM 裡面的資料就全部不見了
- **需要不斷刷新**：「Dynamic（動態）」就是這個意思，資料會不斷流失，需要一直補充電力來維持

跟 [NAND Flash](nand-flash.md) 的區別：

```
DRAM          → 速度超快，但斷電就消失（像工作桌面）
NAND Flash    → 速度比較慢，但斷電不會消失（像抽屜/檔案櫃）
```

你手機的「8GB RAM」、電腦的「16GB 記憶體」——這裡的 RAM 就是 DRAM。

全球前三大 DRAM 公司：**三星、SK 海力士、美光（Micron）**，三家加起來佔了超過 90% 的市場。

## 生活比喻 / 實際例子

想像你在辦公：

- **DRAM（記憶體）** = 你的辦公桌面，正在處理的文件攤在桌上
- **[NAND Flash](nand-flash.md)（儲存空間）** = 旁邊的檔案櫃，存放暫時不用的文件
- 桌面越大（DRAM 越多）→ 可以同時攤開越多文件 → 工作效率越高
- 下班關燈（關機）→ 桌面清空（DRAM 資料消失），但檔案櫃還在（NAND 保留資料）

所以手機「記憶體不夠」的時候，就是桌面太小、文件攤不下，你切換 App 的時候就會很卡。

實際你會聽到的說法：
- 新聞：「**DRAM** 價格本季上漲 15%」
- 法說會：「AI 伺服器帶動 **DRAM** 需求強勁成長」
- 同事：「這台電腦 **RAM** 只有 8GB，跑起來很吃力」

## 為什麼要知道這個詞？

- DRAM 是半導體產業的三大核心產品之一（邏輯晶片、DRAM、NAND）
- DRAM 價格波動很大，是影響三星、SK 海力士、美光獲利的關鍵因素
- [HBM](hbm.md) 就是一種特殊的 DRAM，是 AI 晶片最需要的零件
- 投資人常常關注「DRAM 報價」來判斷記憶體產業的景氣好壞
