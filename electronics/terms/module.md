# 模組（Module） — 把多個零件組裝成一個功能單元

> **白話說：** 模組就像**樂高積木組合包**——裡面有好幾個零件，組合起來就是一個完整的功能。例如相機模組裡面有鏡頭、感光元件、馬達，組在一起就能拍照。

---

## 它到底是什麼？

**模組（Module）** 就是把多個零件預先組裝成一個「功能單元」，這樣下游的製造商就不用一個一個零件去組裝，直接拿一整個模組來用就好。

常見的模組種類：

| 模組名稱 | 裡面有什麼 | 用在哪裡 |
|---------|-----------|---------|
| 相機模組 | 鏡頭 + 感光元件 + 馬達 + 排線 | 手機 |
| Wi-Fi 模組 | 天線 + 無線晶片 + 電路 | 各種聯網裝置 |
| [散熱模組](thermal-module.md) | 散熱器 + 風扇 + 熱導管 | 電腦、伺服器 |
| 電源模組 | 變壓器 + 電容 + 穩壓晶片 | 所有電子設備 |
| 記憶體模組 | [DRAM](dram.md) 晶片 + [PCB](pcb.md) | 電腦 |
| 顯示模組 | 面板 + 背光 + 驅動 IC + 排線 | 手機、電視、筆電 |

為什麼要做成模組？

```
沒有模組的世界：
  手機廠要買 1000 種零件 → 自己一個個組裝 → 慢、容易出錯

有模組的世界：
  手機廠買 10 個模組 → 像拼樂高一樣組在一起 → 快、品質穩定
```

## 生活比喻 / 實際例子

就像你在 IKEA 買**組合式衣櫃**：

- 如果 IKEA 把木板、螺絲、滑軌全部散裝給你 → 你要自己量尺寸、鑽孔、組裝（超麻煩）
- 如果 IKEA 把「抽屜」做成一個模組、「掛衣桿區」做成另一個模組 → 你只要把模組卡進去就好

手機的製造也是這個道理：
- 相機模組 → 大立光、舜宇光學做好 → 直接裝進手機
- Wi-Fi 模組 → 做好 → 直接裝進手機
- 手機廠（像 Apple、Samsung）把這些模組組在一起 → 就變成一台手機

實際你會聽到的說法：
- 新聞：「大立光是全球最大的手機**相機模組**鏡頭供應商」
- 法說會：「我們的 **Wi-Fi 模組** 出貨量創新高」
- 分析師：「AI 伺服器的 **散熱模組** 規格要求比傳統伺服器高很多」

## 為什麼要知道這個詞？

- 電子產業很大一部分的工作就是做模組、賣模組
- 大立光（鏡頭模組）、台達電（電源模組）、雙鴻（散熱模組）都是模組廠商
- 看到某某公司是「XX 模組供應商」，你就知道他們在做什麼了
- 搞懂模組的概念，你就理解電子產業的分工方式——不是一家公司做完所有事
