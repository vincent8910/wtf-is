# PCB — 所有電子零件住的「公寓大樓」，就是那塊綠色的板子

> **白話說：** PCB 就像一棟**公寓大樓**——所有的電子零件（晶片、電容、電阻⋯⋯）都住在這塊板子上，裡面的銅線就像大樓的走廊和管線，讓住戶（零件）之間互相連通。

---

## 它到底是什麼？

**PCB** 是 **Printed Circuit Board（印刷電路板）** 的縮寫。

你有沒有拆開過壞掉的遙控器或舊手機？裡面那塊**綠色（或棕色）的硬板子**，上面佈滿密密麻麻的線路和焊接的小零件——那就是 PCB。

PCB 的功能：

```
1. 提供「地基」 → 讓所有零件有地方可以安裝
2. 提供「線路」 → 讓零件之間的電訊號可以互相傳遞
3. 提供「結構」 → 保護零件、維持形狀
```

PCB 有不同的複雜度：

| 類型 | 層數 | 用在哪裡 |
|------|------|---------|
| 單面板 | 1-2 層 | 遙控器、簡單電器 |
| 多層板 | 4-8 層 | 手機、電腦 |
| 高階板 | 10-20+ 層 | 伺服器、AI 設備、5G 基地台 |

層數越多，能塞的線路越多，功能越強大，但製造難度和成本也越高。

台灣是全球 PCB 產值最高的國家，主要公司：臻鼎、欣興、華通、健鼎等。

## 生活比喻 / 實際例子

就像一棟**公寓大樓**：

- **PCB 板子本身** = 大樓的結構和樓板
- **上面的電子零件** = 住在裡面的住戶
- **PCB 裡面的銅線** = 大樓裡的水管、電線、網路線
- **多層板** = 大樓有很多樓層，每層的管線不同
- **[玻纖布](glass-fabric.md)** = 大樓的鋼筋，讓板子更堅固

你身邊所有的電子產品——手機、電腦、電視、冷氣、汽車、冰箱——裡面都有 PCB。

實際你會聽到的說法：
- 新聞：「台灣是全球最大的 **PCB** 生產國」
- 法說會：「AI 伺服器用的高階 **PCB** 需求持續成長」
- 分析師：「**PCB** 是電子產業的基礎，景氣跟整體電子業連動」

## 為什麼要知道這個詞？

- PCB 是所有電子產品的基礎零件，不管什麼設備裡面都有
- 台灣是全球最大的 PCB 生產國，相關公司是台股重要族群
- AI 伺服器需要更高階的 PCB，帶動高階板的需求
- 搞懂 PCB，你就理解電子產品裡面最基本的結構是什麼
