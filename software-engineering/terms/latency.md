# Latency / Ping（延遲）— 打電話的回音延遲

> **白話說：** 你按下操作到伺服器回應之間的時間差，越低越好。

---

## 它到底是什麼？

**Latency**（延遲）就是你做了一個動作，到對方收到並回應，中間經過的時間。**Ping** 則是測量這個延遲的方式。

```
低延遲（Ping 10ms）：你說「喂？」→ 對方馬上回「喂！」→ 通話順暢
高延遲（Ping 500ms）：你說「喂？」→ 等了半秒 → 對方才回「喂！」→ 很卡
```

**就像打國際電話的回音延遲。** 如果對方在隔壁，幾乎沒延遲；如果對方在地球另一端，你說完話要等一下才聽到回應。

## 生活比喻 / 實際例子

你在打線上遊戲時，畫面右上角常常會看到一個數字（例如 30ms、100ms），那就是你的 Ping 值。

- **Ping 30ms**：操作超順，按了馬上有反應
- **Ping 100ms**：還可以，偶爾有一點點延遲感
- **Ping 300ms+**：明顯感覺角色動作慢半拍，玩射擊遊戲會被打爆

### 常見情境

- **玩家：**「我 ping 好高，角色一直瞬移！」
- **遊戲伺服器工程師：**「亞洲玩家的 latency 太高了，我們需要在日本加一個伺服器。」
- **PM：**「延遲超過多少要跳提示？」→ 通常 150ms 以上就算高延遲

## 為什麼要知道這個詞？

不只是遊戲，任何線上服務都有 latency 的問題。視訊會議的延遲、網頁載入的速度、App 回應的快慢——背後都跟 latency 有關。當你聽到工程師說「要降低 latency」，就是在說「要讓系統回應更快」。
