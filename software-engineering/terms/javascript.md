# JavaScript — 讓網頁「動起來」的程式語言

> **白話說：** 如果 [HTML](html.md) 是房子的結構、[CSS](css.md) 是裝潢，那 JavaScript 就是水電和電器——按按鈕會有反應、門鈴按了會響。

---

## 它到底是什麼？

JavaScript 是一種**程式語言**，也是目前全世界最多人使用的程式語言之一。

它最早是為了讓網頁「有互動」而誕生的。比如你在網頁上：
- 點一個按鈕，跳出一個視窗 → JavaScript
- 滑鼠移過去，圖片會放大 → JavaScript
- 表單送出前，檢查你的 Email 有沒有打對 → JavaScript

後來因為太好用了，大家開始拿它做越來越多事情。現在連[後端](frontend-backend.md)、手機 App、桌面軟體都可以用 JavaScript 來寫。[Node.js](nodejs.md) 就是讓 JavaScript 能在電腦上執行（不只在瀏覽器裡）的工具。

## 生活比喻 / 實際例子

想像你去一間餐廳：
- **[HTML](html.md)** 決定餐廳裡有哪些桌椅、牆壁、吧台（結構）
- **[CSS](css.md)** 決定桌椅的顏色、牆壁的風格、燈光的氣氛（外觀）
- **JavaScript** 決定你按門鈴會有人來開門、點菜系統會運作、叫號螢幕會更新（互動行為）

**你什麼時候會聽到：**
- 「我們[前端](frontend-backend.md)用的是 **JavaScript**。」
- 「這個動畫效果要用 **JavaScript** 來做。」
- 「[React](framework-library.md)、Vue 這些[框架](framework-library.md)都是 **JavaScript** 寫的。」

## 為什麼要知道這個詞？

- 它是目前網頁開發的核心語言，幾乎所有網站都用到它
- 當你聽到 React、Vue、Angular、[Node.js](nodejs.md)、Next.js 這些名詞，底層都是 JavaScript
- 知道這個詞能幫你理解工程師在說什麼：「前端用 JavaScript」= 「網頁的互動功能用 JavaScript 寫的」
- [JSON](json.md) 的全名裡就有 JavaScript（JavaScript Object Notation），但 JSON 本身已經獨立成通用格式了
