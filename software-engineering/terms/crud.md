# CRUD — Excel 的基本功

> **白話說：** 資料的四種基本操作：新增（Create）、讀取（Read）、修改（Update）、刪除（Delete）。

---

## 它到底是什麼？

**CRUD** 是四個英文字的縮寫，代表你對資料能做的四件最基本的事：

| 字母 | 全名 | 白話 | Excel 對照 |
|------|------|------|-----------|
| **C** | Create | 新增一筆資料 | 新增一列 |
| **R** | Read | 讀取 / 查看資料 | 打開試算表看 |
| **U** | Update | 修改已有的資料 | 改某一格的內容 |
| **D** | Delete | 刪除資料 | 刪掉一列 |

幾乎所有你用過的軟體都是在做 CRUD：

```
Line 的記事本：新增筆記、查看筆記、編輯筆記、刪除筆記
蝦皮的商品管理：上架商品、瀏覽商品、修改價格、下架商品
```

## 生活比喻 / 實際例子

**CRUD 就是 Excel 的基本功。** 你在 Excel 裡每天做的事——新增一列資料、查看現有資料、改某格數字、刪掉整列——就是 CRUD。

### 常見情境

- **後端工程師：**「這個功能就是基本的 CRUD，沒什麼複雜的。」→ 就是做新增、查詢、修改、刪除
- **PM：**「會員管理後台需要完整的 CRUD。」→ 要能新增會員、查看會員、修改會員資料、刪除會員
- **面試時：**「先寫一個 CRUD 出來。」→ 最基本的能力測試

## 為什麼要知道這個詞？

當工程師說「這就是 CRUD」，他的意思是「這個功能沒什麼特別的，就是基本的資料操作」。這是[後端](frontend-backend.md)開發最基礎的概念，聽懂這個詞，你就大概知道一個功能的複雜程度。
