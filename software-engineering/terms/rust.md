# Rust — 又快又安全的程式語言

> **白話說：** 就像一台既省油又安全的跑車——跑得很快，但內建各種防撞系統，不容易出事。

---

## 它到底是什麼？

Rust 是一種程式語言，特色是**效能極高**（跟 C/C++ 一樣快）但**安全性很強**（編譯器會在你寫程式時就幫你抓出很多常見的錯誤）。

傳統的高效能語言（C/C++）很快，但很容易寫出有記憶體問題的程式（當機、安全漏洞）。Rust 在設計時就把「安全」這件事內建在語言裡，讓你很難寫出不安全的程式碼。

常被用在：
- 系統底層開發（作業系統、瀏覽器引擎）
- 高效能後端服務
- 區塊鏈、加密貨幣相關系統
- 遊戲引擎
- CLI 工具

## 生活比喻 / 實際例子

想像不同的交通工具：

- **[Python](python.md)** = 腳踏車，好上手、慢慢騎很舒服
- **[JavaScript](javascript.md)** = 機車，到處都能騎，機動性高
- **Rust** = F1 賽車，速度極快，但駕駛座的安全裝置非常嚴格，連安全帶沒繫好車子都不讓你發動

實際例子：
- 工程師說：「這個服務對效能要求很高，我們考慮用 **Rust** 來寫」
- 工程師說：「**Rust** 的編譯器很嚴格，寫起來比較慢但 Bug 也比較少」

## 為什麼要知道這個詞？

- 近年來人氣快速上升的程式語言，常在技術討論中聽到
- 如果工程師說「用 Rust 寫」，通常代表這個東西對效能或安全性有很高的要求
- Discord、Dropbox、Cloudflare 等知名公司都有用 Rust
