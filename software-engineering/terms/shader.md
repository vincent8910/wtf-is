# Shader（著色器）— 電影打光師

> **白話說：** 控制遊戲裡光影、材質、特效的程式，讓畫面從「塑膠感」變成「電影感」。

---

## 它到底是什麼？

**Shader** 是一種專門處理畫面視覺效果的小程式。它決定畫面上每一個小點最終「長什麼顏色」。

想像一部電影的拍攝現場——

```
沒有打光師（沒有 Shader）：
  → 場景平平的，所有東西看起來像塑膠模型

有打光師（有 Shader）：
  → 金屬會反光、水面會折射、皮膚有質感、火焰會發光
```

**Shader 就是電影打光師。** 同一個場景，打光打得好看起來就像大片，打不好就像學生作品。

## 生活比喻 / 實際例子

你有沒有用過手機相機的「濾鏡」？把同一張照片套上不同濾鏡，整個感覺就不一樣。Shader 就是遊戲畫面的「即時濾鏡」，不過它更厲害——不只改顏色，還能算出光影、反射、陰影、透明度等效果。

### 常見的 Shader 效果

- **水面倒影**：水看起來真的像水，而不是一片藍色
- **角色皮膚質感**：有光澤、有陰影，不像塑膠人偶
- **爆炸特效**：火光、煙霧、衝擊波的視覺效果
- **卡通渲染**：讓 3D 模型看起來像手繪動畫

### 常見情境

- **遊戲客戶端工程師：**「這個 shader 太吃效能了，低階手機跑不動。」
- **美術：**「這個材質的 shader 能不能再亮一點？」
- **企劃：**「關掉一些 shader 效果，讓低配玩家也能玩。」

## 為什麼要知道這個詞？

如果你在遊戲公司工作，Shader 直接影響遊戲的「畫面品質」。當你看到遊戲截圖覺得「哇，好真實」或「哇，好有風格」，背後很大一部分是 shader 的功勞。但 shader 越精美，越吃硬體效能，這是企劃和工程師常需要取捨的地方。
