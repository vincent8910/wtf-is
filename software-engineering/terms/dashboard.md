# Dashboard（儀表板）— 汽車儀表板

> **白話說：** 用圖表呈現關鍵數據的視覺化頁面，讓你一眼就能掌握狀況。

---

## 它到底是什麼？

**Dashboard** 就是把重要的數字和趨勢，用圖表的方式整理在同一個頁面上。

就像汽車的儀表板——

```
汽車儀表板：一眼就能看到
  → 時速多少？油量剩多少？引擎溫度正常嗎？

軟體 Dashboard：一眼就能看到
  → 今天營收多少？用戶數成長了嗎？系統錯誤率高嗎？
```

**Dashboard 就是你的「駕駛座視角」——不用翻報告，重要數字都在眼前。**

## 生活比喻 / 實際例子

你的 iPhone「螢幕使用時間」報告就是一個 dashboard——用長條圖告訴你每天花多少時間在手機上、哪個 App 用最多。你不用自己算，一張圖就看完。

### 常見的 Dashboard 類型

| 類型 | 誰在看 | 看什麼 |
|------|--------|--------|
| 營運 Dashboard | PM、老闆 | DAU（每日活躍用戶數）、營收、轉換率（訪客變成付費用戶的比例） |
| 系統 Dashboard | 工程師 | CPU（電腦的大腦）、記憶體、錯誤率 |
| 行銷 Dashboard | 行銷 | 廣告成效、點擊率、花費 |

### 常見情境

- **老闆：**「把這些數字整理成一個 dashboard，我每天早上看一次。」
- **資料工程師：**「dashboard 的資料是每小時更新一次。」
- **PM：**「dashboard 上的轉換率突然掉了，發生什麼事？」

## 為什麼要知道這個詞？

做決策需要數據，但沒人想看一整份 Excel。Dashboard 讓複雜的數據變成直覺的圖表，是每天開會、追進度、做決策時最常用的工具。
