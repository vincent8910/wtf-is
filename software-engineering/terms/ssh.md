# SSH（安全遠端連線） — 從你的電腦遠端登入另一台電腦

> **白話說：** 就像遠端桌面，但不是看到畫面，而是用打字的方式操作對方的電腦，而且中間有加密保護。

---

## 它到底是什麼？

SSH 是 **Secure Shell** 的縮寫。它讓你坐在自己的電腦前，透過網路安全地連到另一台電腦（通常是[伺服器](server.md)），然後用打指令的方式操作它。

重點是「安全」：你跟伺服器之間的通訊是[加密](encryption.md)的，別人偷看也看不懂。

## 生活比喻 / 實際例子

想像公司的重要文件都鎖在辦公室的保險箱裡：

- **沒有 SSH** = 你只能親自跑去辦公室開保險箱
- **有 SSH** = 你有一把特製的遠端遙控鑰匙，在家就能安全地打開保險箱操作

實際例子：
- 工程師說：「我 **SSH** 進去看一下 [log](log.md)」（意思是遠端連到伺服器去查紀錄）
- 工程師說：「伺服器出問題了，我先 **SSH** 進去重開服務」

## 為什麼要知道這個詞？

- 工程師在處理伺服器問題時幾乎都會用 SSH，你會常聽到
- 設定部署環境、排查線上問題時，工程師會說「我 SSH 進去看一下」
- 有時候工程師請你提供伺服器的 SSH 帳號密碼或金鑰，你要知道他在說什麼
