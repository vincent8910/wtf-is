# Phishing（釣魚攻擊）— 詐騙電話假裝警察

> **白話說：** 壞人假裝成你信任的對象（銀行、公司、朋友），騙你點連結或輸入密碼。

---

## 它到底是什麼？

**Phishing**（釣魚攻擊）就是駭客假扮成你認識或信任的人，誘騙你交出帳號密碼或個人資料。

就像詐騙電話——

```
詐騙集團的套路（Phishing）：
  1. 假裝成某個你信任的身份（銀行客服、警察、同事）
  2. 製造緊急感（「你的帳戶異常！」「你被盜了！」）
  3. 誘導你採取行動（點連結、輸入密碼、匯款）
  4. 你上當了，他們拿到你的資料或錢
```

**Phishing 就是網路版的詐騙電話——假裝是你信任的人，騙你上鉤。**

## 生活比喻 / 實際例子

### 常見的釣魚手法

| 手法 | 範例 |
|------|------|
| 假 Email | 「您的 Apple ID 已被鎖定，請立即點此驗證」（連結是假的） |
| 假簡訊 | 「您有一筆未領包裹，請點連結查看」（連結連到假網站） |
| 假網站 | 長得跟真的銀行網站一模一樣，但網址不一樣 |
| 假同事 | 「我是 IT 部門，請提供你的密碼以進行系統更新」 |

### 怎麼辨別？

- 看寄件者的 Email 地址是否正確（不是 `apple@gmail.com` 之類的）
- 看網址有沒有奇怪的拼字（`gooogle.com`）
- 正規公司不會用 Email 或電話要你的密碼

### 常見情境

- **資安工程師：**「公司收到大量 phishing 信件，提醒大家不要亂點連結。」
- **HR：**「我們要做一次 phishing 演練，測試同仁的資安意識。」→ 故意發假的釣魚信看誰會上當
- **同事：**「這封信看起來怪怪的，是不是 phishing？」

## 為什麼要知道這個詞？

Phishing 是目前最常見的網路攻擊方式之一，而且不分職位——任何人都可能成為目標。你不需要是工程師也可能收到釣魚信件。知道這個詞、知道怎麼辨別，就是保護自己最好的方式。
