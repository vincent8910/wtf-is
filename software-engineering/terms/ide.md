# IDE — 全配的辦公室

> **白話說：** IDE 就像一間**全配的辦公室**——桌子（程式編輯器）、白板（預覽視窗）、助手（AI）、檔案櫃（專案管理），全部整合在同一個視窗裡，讓你不用切來切去。

---

## 它到底是什麼？

IDE 全名是 **Integrated Development Environment**（整合開發環境）。

聽起來很厲害，但其實就是一個**「什麼都包」的編輯軟體**。你打開它，就能在同一個畫面裡：
- 寫文件或程式碼
- 看到即時預覽
- 讓 AI 幫你改東西
- 管理專案裡的所有檔案

就像你不需要分別去文具店買筆、去傢俱行買桌子、去人力銀行找助手——IDE 一次全包。

## IDE vs [CLI](terminal-cli.md)：兩種使用 AI 的方式

| | IDE | CLI |
|---|---|---|
| **長什麼樣** | 有漂亮的視窗介面，按鈕、選單、圖示 | 一個黑色視窗，用打字下指令 |
| **比喻** | 去餐廳吃飯——菜單明確，看圖點菜 | 打電話叫外送——直接說你要什麼 |
| **優勢** | 看得到、點得到，上手快 | 速度快、彈性大、可以自動化 |
| **適合** | 需要寫程式、做前端開發的人 | 需要批次處理、自動化的人 |

> 💡 **兩者不衝突！** 很多人白天用 IDE 寫程式，晚上用 CLI 跑自動化任務。就像你可以去餐廳吃午餐、晚上自己煮飯。
