# Cache（快取）— 冰箱裡的常備菜

> **白話說：** Cache 就像把常用的資料放在隨手可拿的地方，不用每次都從頭找，所以更快。

---

## 它到底是什麼？

**Cache**（唸作「快許」，中文叫「快取」）是電腦用來加快速度的機制。

```
沒有 Cache：「今晚吃什麼？」→ 開車去超市 → 買菜 → 回家 → 料理（2 小時）
有 Cache：  「今晚吃什麼？」→ 打開冰箱拿常備菜 → 微波加熱（10 分鐘）
```

**冰箱就是你廚房的 Cache。** 把常吃的食材預先買好放冰箱（Cache），就不用每餐都跑超市（去遠端伺服器要資料）。

## 為什麼「清 Cache」能解決問題？

你最常聽到的一句話就是：**「清一下 Cache 試試看。」**

因為——便利貼上的電話號碼可能已經過期了：

1. 你把同事小明的電話寫在便利貼上（建立 Cache）
2. 小明換了新號碼（原始資料更新了）
3. 你照便利貼打，打不通（Cache 裡是舊資料）
4. 你撕掉便利貼，重新查通訊錄（清 Cache）
5. 找到新號碼，打通了！

**清 Cache = 撕掉舊便利貼，強制重新查一次最新資料。**

### 最常碰到：瀏覽器 Cache

> **行銷**：「我明明請工程師改好官網文案了，為什麼我看到的還是舊版？」
> **工程師**：「你清一下瀏覽器 Cache 就會看到新版了。」

| 瀏覽器 | 清 Cache 快捷鍵 |
|--------|----------------|
| Chrome（Mac） | `Cmd + Shift + Delete` |
| Chrome（Windows） | `Ctrl + Shift + Delete` |
| Safari | `Cmd + Option + E` |

或者最簡單：**用無痕模式開網頁**，就能看到最新版。

## 日常場景

- **「那個報表有做 cache，資料每 5 分鐘更新一次」**
  → 你看到的可能是 5 分鐘前的數字，重新整理就好
- **「清一下 cache 就好了」**
  → 把舊的暫存資料丟掉，重新取得最新版本

> 💡 **口訣：看到怪東西 → 先清 Cache → 80% 的機率問題就解決了。**
