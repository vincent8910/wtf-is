# AI Agent — 自主做事的 AI 員工

> **白話說：** 一般的 AI 像客服——你問一句它答一句。AI Agent 像正職員工——你交代一個任務，它會自己規劃步驟、自己執行、遇到問題自己想辦法解決。

---

## 它到底是什麼？

**Agent**（代理人）指的是一個能**自主行動**的 AI。

差別在哪裡？

| | 一般 AI（聊天機器人） | AI Agent（代理人） |
|---|---|---|
| 你說 | 「怎麼做報告？」 | 「幫我做好這份報告」 |
| AI 做 | 告訴你步驟 | 自己去讀資料、整理、產出報告 |
| 互動模式 | 你問一句、它答一句 | 你交代任務、它整個完成 |
| 像什麼 | 問路的路人 | 你僱的計程車司機 |

## Agent 為什麼厲害？

因為 Agent 能做到三件一般 AI 做不到的事：

1. **規劃**：把大任務自動拆成小步驟
2. **執行**：直接動手做（讀檔案、改文件、跑程式）
3. **反思**：做完檢查結果，不對就自己修正

這就是 Claude Code 的運作方式——它不只是聊天，它是一個 Agent。

## Agent 團隊（特遣隊）是什麼？

一個 Agent 已經很強了，但如果**多個 Agent 一起合作**，就更厲害了——這就是「特遣隊」概念：

```
🎖️ 指揮官（你）
├── 🔍 偵察兵 Agent → 負責搜集資料
├── 📊 分析師 Agent → 負責整理分析
├── 📝 撰稿 Agent   → 負責寫報告
└── ✅ 品管 Agent   → 負責最終審查
```

每個 Agent 各司其職，就像一個小型的專案團隊。

> 💡 這是進階用法，初學者不需要一開始就學。先熟悉單一 Agent（Claude Code 本身）就很夠用了。
