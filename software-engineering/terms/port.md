# Port（連接埠） — 電腦上不同服務的「門牌號碼」

> **白話說：** 就像一棟大樓有很多間辦公室，[IP 位址](ip-address.md)是大樓地址，Port 就是辦公室的門牌號碼。

---

## 它到底是什麼？

一台電腦上可能同時跑很多服務：網站、Email、資料庫、聊天軟體……這些服務都用同一個網路連線，那怎麼區分「這個請求要找誰」呢？

答案就是 Port。每個服務佔用一個 Port 號碼，就像同一棟大樓裡不同的辦公室。

常見的 Port 號碼：
- **80** — 一般網站（[HTTP](http-https.md)）
- **443** — 加密網站（[HTTPS](http-https.md)）
- **3000** — 很多開發工具預設用的（[localhost](localhost.md):3000）
- **3306** — MySQL [資料庫](database.md)
- **22** — [SSH](ssh.md) 遠端連線

## 生活比喻 / 實際例子

想像一棟大樓：

- **IP 位址** = 大樓地址（台北市信義路 100 號）
- **Port** = 大樓裡的門牌號碼（3 樓 301 室、5 樓 502 室）

你寄信時不能只寫大樓地址，還要寫清楚「幾樓幾號」，信才會送到對的辦公室。

實際例子：
- 工程師說：「**Port** 3000 被佔用了，我換一個 port 跑」
- 工程師說：「在 localhost:**8080** 就可以看到了」（8080 就是 port 號碼）
- PM 問：「為什麼我打不開測試環境？」工程師：「你的網址要加 **port** 號碼，是 `localhost:3000`」

## 為什麼要知道這個詞？

- 工程師給你測試連結時，常會帶 port 號碼（例如 `localhost:3000`）
- 聽到工程師說「port 被佔了」「換一個 port」，你知道他在說什麼
- 設定[防火牆](firewall.md)或伺服器時，要「開放」特定的 port 才能讓外面連進來
