# H5 — 不是一種遊戲，而是用網頁技術做出來的互動內容

> **白話說：** H5 就是「用網頁技術做的互動頁面」的簡稱。它可以是小遊戲、可以是活動頁、也可以是電子賀卡——重點不是「遊戲」，而是「用瀏覽器就能打開，不用下載 App」。

---

## 它到底是什麼？

H5 是中國和台灣業界對「用 HTML5 技術做出來的互動網頁內容」的簡稱。

先搞清楚幾個容易搞混的東西：

| 名稱 | 它是什麼 | 比喻 |
|------|---------|------|
| **[HTML](html.md)** | 網頁的骨架語言，告訴瀏覽器「頁面上有什麼東西」 | 建築的鋼筋結構 |
| **HTML5** | HTML 的第五代版本，加了很多新功能（影片、音樂、動畫、畫圖） | 鋼筋結構升級版，現在還內建水電管線 |
| **H5** | 業界俗稱，指「用 HTML5 技術做的互動內容」 | 用升級版建材蓋出來的各種體驗館 |

關鍵觀念：**H5 不等於遊戲。**

H5 可以是很多東西：
- 品牌活動頁（滑動、翻頁、動畫效果很酷的那種）
- 互動問卷或測驗（「測你是哪種動物」那種）
- 電子邀請函或賀卡
- 小遊戲（抽獎、接東西、答題闖關）
- 互動式故事或簡報

它們的共同點只有一個：**用瀏覽器就能打開，不用下載任何東西。** 通常在 Line 或微信裡面直接點連結就能玩。

## H5 vs 遊戲 vs HTML 到底差在哪？

```
HTML（語言）
  └── HTML5（HTML 的新版本，支援更多功能）
        └── H5（用 HTML5 做的互動內容）← 業界俗稱
              ├── 活動頁面（不是遊戲）
              ├── 互動問卷（不是遊戲）
              ├── 電子賀卡（不是遊戲）
              └── 小遊戲（是遊戲，但只是 H5 的其中一種）
```

所以：
- ❌「我們來做一個 H5 遊戲」→ 把 H5 當成遊戲的代名詞（不精確）
- ✅「我們來做一個 H5」→ 可能是遊戲，也可能是活動頁或互動內容
- ✅「我們來做一個 H5 小遊戲」→ 明確說是遊戲類型的 H5

## 生活比喻 / 實際例子

想像「影片」這個詞：
- 影片可以是電影、廣告、Vlog、教學、搞笑短片
- 你不會說「影片 = 電影」，因為影片只是一種格式，內容可以是任何東西

**H5 也一樣**——它是一種「用網頁技術做的互動格式」，內容可以是遊戲，但也可以是任何互動體驗。

實際例子：
- 行銷說：「週年慶我們做一個 **H5** 活動頁，使用者可以滑動看故事，最後抽獎」
- 老闆說：「聽說 **H5** 很紅，我們也來做一個」→ 他可能想要的是活動頁，不一定是遊戲
- PM 說：「這個 **H5** 要能在 Line 裡面直接打開」→ 這是 H5 的最大優勢：不用下載

## 為什麼要知道這個詞？

- 行銷企劃經常會提到「做一個 H5」，你要知道這不一定是做遊戲
- 跟工程師溝通時，說清楚你要的是「互動活動頁」還是「小遊戲」很重要，因為開發難度差很多
- H5 的最大賣點是「不用下載，瀏覽器直接開」，適合在社群（Line、Facebook）快速傳播
- 相比 [Native App](native-web-app.md)，H5 的開發成本低、速度快，但功能和效能上有限制
