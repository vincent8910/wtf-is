# Markdown — 最簡單的排版語言

> **白話說：** Markdown 就像「輕量版的 Word」。用幾個簡單的符號（`#`、`*`、`-`）就能讓純文字變成有標題、粗體、清單的漂亮文件。

---

## 它到底是什麼？

Markdown 是一種**文字格式**，副檔名是 `.md`。它讓你用純文字就能做出排版效果，不需要像 Word 那樣點按鈕。

例如：

| 你打的東西 | 顯示出來的效果 |
|-----------|--------------|
| `# 大標題` | **大標題**（跟 Word 的「標題一」一樣） |
| `## 小標題` | **小標題**（跟 Word 的「標題二」一樣） |
| `**粗體**` | **粗體** |
| `- 項目一` | 一個清單項目 |
| `` `程式碼` `` | 一段程式碼 |

## 為什麼我需要知道它？

因為 GitHub 上的文件（README.md 等）都是 Markdown 格式。你在 GitHub 上看到的漂亮排版，其實都是 Markdown 寫的。

當你用 Claude Code 產出文件時，它也經常輸出 Markdown 格式。認識它，你就能看懂並修改這些文件。

## 30 秒學會 Markdown

```markdown
# 標題（一個 # 是最大標題）
## 次標題（兩個 #）
### 小標題（三個 #）

**這是粗體**
*這是斜體*

- 清單第一項
- 清單第二項
- 清單第三項

[這是連結文字](https://example.com)

> 這是引用（像 Email 裡的引用）
```

就這些！只要會這幾個符號，你就能讀懂和編輯所有 `.md` 檔案了。

## 用什麼軟體打開 .md 檔？

- **任何文字編輯器**都可以（記事本、TextEdit）
- **VS Code**（免費）：支援即時預覽，推薦 → [下載](https://code.visualstudio.com/)
- **GitHub 上直接看**：上傳到 GitHub 後會自動排版顯示
- **Typora**（付費）：所見即所得的 Markdown 編輯器
