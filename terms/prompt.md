# Prompt — 你對 AI 說的話

> **白話說：** Prompt 就是「你打給 AI 的那段文字指令」。就像你跟同事交辦工作時說的話——說得越清楚，同事做得越好。

---

## 它到底是什麼？

**Prompt**（提示詞）就是你輸入給 AI 的文字。它可以是一個問題、一個指令、或一段描述。

```
「幫我把這份英文信翻譯成繁體中文」
```

↑ 這整句話就是一個 Prompt。

## 為什麼大家這麼重視 Prompt？

因為**同樣一件事，不同的說法會得到完全不同品質的結果**。

就像跟實習生交辦工作：

| Prompt 品質 | 範例 | 結果 |
|------------|------|------|
| 😟 模糊 | 「幫我處理一下」 | AI 不知道你要什麼，亂做一通 |
| 😐 普通 | 「幫我翻譯這個檔案」 | 能做，但可能格式跑掉、語氣不對 |
| 😄 清楚 | 「把 report.md 從英文翻譯成繁體中文，保持 Markdown 格式，語氣專業」 | 精準完成你的需求 |

## Prompt 的黃金公式

```
任務 + 對象 + 條件 + 輸出格式
```

| 元素 | 說明 | 範例 |
|------|------|------|
| 任務 | 你要 AI 做什麼 | 翻譯、摘要、分析、整理 |
| 對象 | 對什麼東西做 | report.csv、這個資料夾的所有 .md 檔 |
| 條件 | 特殊要求 | 用繁體中文、限 500 字、語氣活潑 |
| 輸出 | 結果格式 | 存成新檔案、用表格呈現、條列式 |

### 實際範例

```
❌ 「幫我整理資料」

✅ 「讀取 data/ 資料夾裡所有的 .csv 檔案，
    把每個檔案的前 5 欄整理成一份摘要表格，
    用繁體中文，
    存成 summary.md」
```

> 💡 **好的 Prompt 值得存起來重複使用！** 找到有效的說法就記下來，下次直接複製貼上改幾個字就好。
