# Terminal / CLI — 管家熱線（直撥電話）

> **白話說：** Terminal 就像一支直撥電話，讓你用「文字」直接對電腦下指令。不用點選圖示、不用拖拉檔案，打字就能叫電腦做事。

---

## 它到底是什麼？

你平常用電腦，是透過滑鼠點來點去（這叫**圖形介面**，GUI）。但其實還有另一種方式——**用文字打指令**，這就是 Terminal 做的事。

**CLI** 是 **Command Line Interface**（命令列介面）的縮寫，指的就是「用文字和電腦溝通」這種操作方式。Terminal 是那個讓你輸入文字的視窗程式。

想像一下：
- **圖形介面（GUI）**：你走到 7-11 櫃台，用手指著商品說「我要那個」
- **命令列（CLI）**：你打電話給管家說「幫我買一瓶鮮奶」——更快、更精確、更強大

## 為什麼我需要它？

因為 Claude Code 就是在 Terminal 裡面使用的。你打開 Terminal，輸入 `claude`，就啟動了你的 AI 員工。

## 怎麼打開 Terminal？

### macOS
- **方法一**：打開 Finder → 應用程式 → 工具程式 → **終端機**
- **方法二**：按 `Cmd + 空白鍵`，輸入「Terminal」，按 Enter
- 打開後你會看到一個黑色（或白色）的視窗，閃著游標等你輸入

### Windows
Windows 沒有內建像 macOS 那麼好用的 Terminal，有幾個選擇：

- **命令提示字元**：按 `Win + R`，輸入 `cmd`，按 Enter
- **PowerShell**：在開始選單搜尋「PowerShell」
- **Git Bash**（推薦）：安裝 [Git](git.md) 時會附帶，操作體驗更接近 macOS → [什麼是 Git Bash？](git-bash.md)

## 常用的幾個基本指令

| 指令 | 做什麼 | 比喻 |
|------|--------|------|
| `cd 資料夾名稱` | 進入某個資料夾 | 走進某個房間 |
| `cd ..` | 回到上一層資料夾 | 走出房間回到走廊 |
| `ls`（macOS）/ `dir`（Windows） | 列出資料夾裡有什麼 | 看看房間裡有什麼東西 |
| `claude` | 啟動 Claude Code | 拿起電話打給 AI 員工 |

> 💡 不用背這些！用到的時候再查就好。
