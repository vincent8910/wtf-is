# Cocos Creator — 輕量級的遊戲引擎，特別適合 2D 手遊和 H5 遊戲

> **白話說：** 跟 [Unity](unity.md) 一樣是做遊戲的工具箱，但更輕巧，特別擅長做 2D 遊戲和可以在瀏覽器裡玩的 [H5](h5.md) 遊戲。

---

## 它到底是什麼？

Cocos Creator 是由 Cocos 團隊（中國廈門的公司）開發的遊戲引擎。它跟 [Unity](unity.md) 一樣提供做遊戲的完整工具，但有自己的特色：

- **2D 特別強**：做 2D 遊戲的效率很高
- **H5 遊戲首選**：產出的遊戲可以直接在瀏覽器或 Line/微信裡面玩，不用下載
- **包體小**：做出來的遊戲檔案比較小，適合手機
- **免費開源**：不用付授權費

工程師用 TypeScript 或 [JavaScript](javascript.md) 在 Cocos Creator 裡寫遊戲邏輯。

### Cocos Creator vs Unity

| 比較 | Cocos Creator | Unity |
|------|:------------:|:-----:|
| 擅長 | 2D 遊戲、H5 遊戲 | 2D + 3D 通吃 |
| 語言 | TypeScript / JavaScript | C# |
| 費用 | 免費開源 | 有免費版，進階要付費 |
| 包體大小 | 小 | 相對大 |
| 3D 能力 | 可以但不是強項 | 非常強 |
| 適合 | 手遊、小遊戲、H5 | 各種規模的遊戲 |

## 生活比喻 / 實際例子

如果 Unity 是一輛什麼路都能跑的 SUV：

- **Cocos Creator** = 一輛省油的小車，在城市裡（2D、H5）開起來比 SUV 靈活又划算

實際例子：
- Game Client 工程師說：「這個 H5 小遊戲用 **Cocos Creator** 做就好，不用上 Unity」
- PM 說：「這個遊戲要能在 Line 裡面直接玩」工程師：「那用 **Cocos Creator** 比較適合」

## 為什麼要知道這個詞？

- 如果你的團隊做 2D 手遊或 H5 互動遊戲，很可能在用 Cocos Creator
- 在中國和東南亞市場特別流行
- 選 Cocos Creator 還是 Unity 是遊戲專案一開始就要做的重大決定
