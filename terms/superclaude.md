# SuperClaude — 員工培訓手冊 / 超級裝備包

> **白話說：** 如果 Claude Code 是你的員工，SuperClaude 就是一套**超級培訓手冊**——讓員工更懂你的需求、工作更有效率、報告更有結構。

---

## 它到底是什麼？

SuperClaude 是一個**開源的設定檔框架**，由社群開發維護。

裝好之後，你的 Claude Code 會獲得一系列「增強能力」：
- 更懂得拆解複雜任務
- 能使用各種專業指令（`/sc:analyze`、`/sc:research` 等）
- 自動選擇最合適的工具來處理問題
- 回報更有結構、更好懂

就像你的員工去上了進修課程，回來後工作能力大幅提升。

## 裝了跟沒裝的差別？

| 情境 | 沒有 SuperClaude | 有 SuperClaude |
|------|-----------------|----------------|
| 分析專案 | 你要一步步引導 AI | 打 `/sc:analyze` 就自動深度分析 |
| 複雜任務 | AI 可能會漏掉步驟 | 自動拆解成小步驟、追蹤進度 |
| 跨對話記憶 | 每次重新交代背景 | 能載入和儲存專案記憶 |
| 工具使用 | 手動指定要用什麼 | 自動判斷最佳工具組合 |

## 我一定要裝嗎？

**不一定。** Claude Code 本身就很好用了。

就像一台車原廠就能開，但加裝導航、倒車雷達、行車記錄器會更方便。SuperClaude 就是那些「加裝配備」。

**建議**：先用原廠（Claude Code），等你覺得「我好像需要更進階的功能」時，再來裝 SuperClaude。
