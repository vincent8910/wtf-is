# Go / Golang — Google 開發的高效能程式語言

> **白話說：** Google 自己造了一台車——簡單好開、跑得快、特別擅長同時載很多人（處理大量請求）。

---

## 它到底是什麼？

Go（又叫 Golang）是 Google 在 2009 年推出的程式語言。它的設計理念是：**簡單、快速、擅長處理大量同時進行的任務**。

Go 的特色：
- **語法很簡單**：比 [Rust](rust.md) 容易學很多，跟 [Python](python.md) 差不多好上手
- **執行很快**：編譯後的程式效能接近 C/C++
- **天生擅長[並發（Concurrency）](concurrency.md)**：處理大量同時湧入的請求特別拿手
- **編譯速度快**：改完程式碼，幾秒鐘就能跑起來

常被用在：
- 高流量的後端服務（API 伺服器）
- 微服務架構
- 雲端基礎設施工具（Docker、Kubernetes 都是用 Go 寫的）
- CLI 工具

## 生活比喻 / 實際例子

想像不同的交通工具：

- **Python** = 腳踏車，好騎、到處能去
- **Go** = 公車，跑得比腳踏車快，而且可以一次載很多人
- **Rust** = F1 賽車，最快但最難開

Go 特別適合「一次要服務很多人」的場景，像是大型網站的後端。

實際例子：
- 工程師說：「這個 API 我們用 **Go** 來寫，效能會比 Python 好很多」
- 工程師說：「Docker 跟 Kubernetes 都是用 **Go** 寫的」

## 為什麼要知道這個詞？

- 很多大型後端系統和雲端工具都用 Go 寫的
- 如果工程師說「用 Go 寫」，通常代表他們需要高效能和高併發能力
- 近年來後端工程師招聘常看到「熟悉 Golang」的要求
