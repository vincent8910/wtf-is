# VS Code — 免費的萬用編輯器

> **白話說：** VS Code 就像一本**超強的筆記本**——不只能寫字，還能裝各種外掛讓它變成計算機、翻譯機、甚至 AI 助手。而且它完全免費。

---

## 它到底是什麼？

VS Code 全名是 **Visual Studio Code**，是 Microsoft（微軟）推出的免費程式碼編輯器。

它是目前全世界最多人使用的編輯器，不管你是寫程式、寫文件、還是編輯 [Markdown](markdown.md) 檔案，都可以用它。

## 為什麼非工程師也會碰到它？

1. **編輯 .md 檔案**：這個專案的所有文件都是 Markdown 格式，用 VS Code 打開可以即時預覽排版效果
2. **AI IDE 的基底**：很多 AI [IDE](ide.md) 都是「基於 VS Code 改造的」——像 Cursor、Google Antigravity 都是。所以學會 VS Code 的基本操作，用這些工具就能快速上手
3. **搭配 GitHub Copilot**：在 VS Code 裡安裝 GitHub Copilot 外掛，就有 AI 輔助功能

## 我需要安裝它嗎？

**看你的需求：**

| 你的情況 | 建議 |
|---------|------|
| 只用 Claude Code（CLI） | 不一定需要，用任何文字編輯器都行 |
| 想要一個好用的 Markdown 編輯器 | 推薦安裝，免費又好用 |
| 想用 Cursor 或 Antigravity | 不用另外裝，它們已經內建 VS Code 的功能 |
| 想用 GitHub Copilot | 需要安裝 VS Code + Copilot 外掛 |

## 安裝

👉 **[VS Code 官方下載](https://code.visualstudio.com/)**（免費，macOS / Windows / Linux 都支援）

下載後直接打開安裝檔，一路點「下一步」就好。
