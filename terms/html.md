# HTML — 網頁的骨架結構

> **白話說：** 就像蓋房子的鋼筋水泥結構，決定「這裡放標題、那裡放圖片、下面放一段文字」。

---

## 它到底是什麼？

HTML 全名是 HyperText Markup Language（超文本標記語言）。

簡單來說，它就是**告訴瀏覽器「這個網頁上有哪些東西、怎麼排」的語言**。

你在網頁上看到的標題、文字、圖片、按鈕、連結——這些東西「存在」在頁面上，就是因為有人用 HTML 寫了出來。但 HTML 只管「有什麼」，不管「好不好看」——好不好看是 [CSS](css.md) 的工作。

## 生活比喻 / 實際例子

**蓋房子：**
- **HTML** = 鋼筋水泥結構（決定哪裡是房間、哪裡是廁所、哪裡是門）
- **[CSS](css.md)** = 室內設計（油漆顏色、地板材質、窗簾款式）
- **[JavaScript](javascript.md)** = 電器設備（開關按了燈會亮、門鈴按了會響）

三個合在一起，就是一個完整的網頁。

**你什麼時候會聽到：**
- 工程師：「這段 **HTML** 結構要改一下。」
- 設計師：「**HTML** 的架構先確認好，再來調 CSS。」
- PM：「請在 **HTML** 裡面加一個新的區塊。」

## 為什麼要知道這個詞？

- 當你看到副檔名是 `.html` 的檔案，就知道那是網頁檔
- 當工程師說「改 HTML」，就是在調整網頁上「有什麼東西」和「怎麼排」
- 當工程師說「改 CSS」，才是在調整「長怎樣」
- 很多行銷工具（電子報、Landing Page 編輯器）會讓你直接編輯 HTML，知道基本概念能幫你微調
