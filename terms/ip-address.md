# IP Address（IP 位址）— 網路上的門牌號碼

> **白話說：** 就像每棟房子都有門牌地址，每台連上網路的設備都有一個 IP 位址，這樣資料才知道要送去哪裡。

---

## 它到底是什麼？

IP 全名是 Internet Protocol（網際網路協定）。

**IP 位址就是網路上每台電腦的「地址」**。當你的電腦要跟某個網站的[伺服器](server.md)溝通，雙方都需要知道對方的 IP 位址，資料才能正確送達。

IP 位址長這樣：`192.168.1.1` 或比較新的格式像 `2001:0db8::1`。

通常你不需要記住這些數字，因為 [DNS](dns.md) 會幫你把網址翻譯成 IP 位址。但工程師在設定伺服器、[防火牆](firewall.md)、網路的時候，就會直接用 IP 位址。

## 生活比喻 / 實際例子

**門牌地址：**
- 你家的門牌是「台北市信義區信義路五段 7 號」
- 你電腦的 IP 位址可能是 `192.168.1.100`
- 快遞（資料封包）看地址才知道要送去哪裡

**你什麼時候會聽到：**
- 「把這個 **IP** 加到[防火牆](firewall.md)白名單裡。」
- 「這個 **IP** 一直在攻擊我們，封掉它。」
- 「你的 **IP** 是什麼？我開權限讓你連進來。」

## 為什麼要知道這個詞？

- 當工程師說要「封 IP」或「開 IP 白名單」，就是在控制誰能連到系統
- 在家工作要連公司內網時，有時需要提供你的 IP 位址
- 網站被攻擊時，工程師會說「某個 IP 一直在打我們」，就是某個來源地址在搗亂
- 「公有 IP」是全世界唯一的地址，「私有 IP」是你家或公司內部的地址（像社區內的門牌）
