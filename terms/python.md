# Python — 好學好讀、用途超廣的程式語言

> **白話說：** 程式語言界的「國民車」——好開、好上手、到哪都能用。從寫網站後端到做 AI、資料分析都行。

---

## 它到底是什麼？

Python 是一種程式語言，最大的特色是**語法簡潔好讀**，幾乎像在讀英文句子一樣。它是目前全世界最受歡迎的程式語言之一。

Python 的用途非常廣：
- **AI / 機器學習**：幾乎所有 AI 框架（TensorFlow、PyTorch）都用 Python
- **資料分析**：Data 資料工程師、資料科學家的首選
- **後端開發**：用 Django、Flask 等[框架](framework-library.md)做網站後端
- **自動化腳本**：把重複的工作自動化
- **爬蟲**：自動從網站抓取資料

### Python 2 vs Python 3

Python 有兩個主要版本：

- **Python 2**：舊版，2020 年已經正式停止維護。如果有人還在用 Python 2，代表那是很老的系統（[Legacy Code](legacy-code.md)）
- **Python 3**：現行版本，所有新專案都應該用 Python 3

兩個版本的語法有些不同，不完全相容。這就是為什麼有時候工程師會特別強調「我們用 Python 3」。

## 生活比喻 / 實際例子

想像程式語言是不同的工具：

- **Python** = 瑞士刀，什麼場合都能用，雖然每個功能不一定是最強，但夠用
- **[Rust](rust.md)** = 手術刀，專業、精準、適合特定場合
- **[JavaScript](javascript.md)** = 膠帶，到處都能黏，尤其是網頁世界

實際例子：
- 工程師說：「後端我們用 **Python** 的 Django 框架」
- 資料工程師說：「我寫了一段 **Python** 腳本去撈資料」
- PM 問：「我們的 AI 模型用什麼語言寫的？」工程師：「**Python**」

## 為什麼要知道這個詞？

- 幾乎所有 AI/ML 相關的討論都會提到 Python
- 很多後端系統和資料處理系統都用 Python 寫的
- 如果工程師說「Python 2 的專案要升級到 3」，你要知道這是一件有風險、需要時間的事
