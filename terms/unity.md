# Unity — 全球最多人用的遊戲引擎

> **白話說：** 做遊戲用的「超級工具箱」——提供蓋遊戲需要的所有零件和工具，從 2D 手遊到 3D 大作都能做。

---

## 它到底是什麼？

Unity 是一個遊戲引擎（Game Engine），也就是「做遊戲的平台工具」。它提供了做遊戲所需的各種功能：

- 畫面[渲染（Render）](render.md)
- 物理模擬（重力、碰撞）
- 動畫系統
- 音效管理
- UI 介面製作
- 跨平台發佈（手機、電腦、主機、網頁都能發佈）

工程師用 C# 語言在 Unity 裡寫遊戲邏輯，設計師可以用 Unity 的圖形介面拖拉元件來設計場景。

Unity 不只做遊戲——也被用在建築視覺化、VR/AR 應用、模擬訓練等領域。

## 生活比喻 / 實際例子

想像你要蓋一棟房子：

- **從零開始** = 自己燒磚、鑄鐵、做水泥（純手刻程式碼做遊戲，超慢）
- **Unity** = 建材行 + 設計軟體 + 施工工具，全部打包好給你，你專心設計房子就好

實際例子：
- Game Client 工程師說：「這個遊戲是用 **Unity** 開發的」
- PM 問：「手機和電腦都要出？」工程師：「沒問題，**Unity** 支援跨平台」
- 工程師說：「**Unity** 的新版[渲染](render.md)管線效能提升很多」

## 為什麼要知道這個詞？

- 全球超過一半的遊戲都是用 Unity 做的（尤其是手遊）
- 如果你的團隊在做遊戲，幾乎一定會聽到 Unity
- Unity 跟 [Cocos Creator](cocos-creator.md) 是手遊開發的兩大主流選擇
- Unity 使用 C# 語言，Cocos Creator 使用 TypeScript/[JavaScript](javascript.md)
