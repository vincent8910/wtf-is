# AWS（Amazon Web Services） — Amazon 的雲端服務平台

> **白話說：** Amazon 開的一家超級大的「電腦出租公司」——你要伺服器、資料庫、儲存空間、AI 工具，通通可以租，用多少付多少。

---

## 它到底是什麼？

AWS 是 **Amazon Web Services** 的縮寫，是目前全球市佔率最高的[雲端](cloud.md)服務平台。簡單說，它讓你不用自己買[伺服器](server.md)，直接租 Amazon 的來用。

AWS 提供超過 200 種服務，但最常聽到的幾個：

| 服務 | 在做什麼 | 白話說 |
|------|---------|--------|
| **EC2** | 虛擬伺服器，跑你的程式 | 租一台電腦 |
| **[S3](s3.md)** | 檔案儲存空間 | 租一個超大的雲端硬碟 |
| **RDS** | 託管的[資料庫](database.md) | 租一台幫你管好的資料庫主機 |
| **[EKS](eks.md)** | 託管的 [Kubernetes](k8s.md) 叢集 | 租一個幫你管好的容器管理平台 |
| **Lambda** | 不用管伺服器，只寫功能就好 | 按次計費的外包服務 |
| **CloudFront** | [CDN](cdn.md) 加速服務 | 在全球各地設分店加速 |
| **IAM** | 權限管理（Identity and Access Management） | 大樓的門禁系統，管誰能進哪裡 |

## 生活比喻 / 實際例子

想像 AWS 是一個超級大的商場：

- 你要辦公室？有（EC2）
- 你要倉庫存東西？有（[S3](s3.md)）
- 你要圖書館管資料？有（RDS）
- 你要快遞幫你全球配送？有（CloudFront）
- 全部按你用多少來收費，不用一次買斷

實際例子：
- 工程師說：「我們的服務跑在 **AWS** 上」
- 老闆看帳單：「這個月 **AWS** 費用怎麼又漲了？」
- [DevOps](devops.md) 說：「我在 **AWS** 上開了幾台 EC2 來跑新服務」

## 為什麼要知道這個詞？

- 全球市佔率第一的雲端平台，非常多公司都在用
- AWS 帳單是公司重要的支出項目之一，老闆和 PM 都會關心
- 跟 [GCP](gcp.md)（Google 的）和 Azure（微軟的）是三大雲端平台
