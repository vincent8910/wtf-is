# RWD（Responsive Web Design）— 變形金剛網頁

> **白話說：** 同一個網頁在手機、平板、電腦上自動調整版面，不用做三個不同的網站。

---

## 它到底是什麼？

**RWD** 全名 Responsive Web Design，中文叫「響應式網頁設計」。

想像你有一張海報，如果它能自動縮放——

```
電腦螢幕：完整展開，文字圖片並排
平板螢幕：稍微調整，圖片縮小
手機螢幕：全部變成單欄，往下滑就好
```

**就像變形金剛，到不同星球（裝置）自動變形成最適合的樣子。**

## 生活比喻 / 實際例子

你有沒有用手機開過一些老網站，文字超級小、要用手指一直放大縮小才看得到內容？那就是「沒有做 RWD」的網站。

反過來說，你用手機打開 YouTube、蝦皮、104 人力銀行，畫面都會自動變成適合手機看的版面——那就是有做 RWD。

### 常見情境

- **設計師：**「這個頁面有做 RWD 嗎？手機版的 banner 要怎麼排？」
- **PM：**「RWD 的斷點設幾個？手機、平板、桌機三種就好嗎？」
- **老闆：**「為什麼手機上看我們的網站這麼醜？」→ 因為沒做 RWD

## 為什麼要知道這個詞？

現在超過一半的使用者用手機上網。如果你的網站沒做 RWD，手機用戶體驗會很差，他們可能直接離開。所以你在討論任何網頁開發專案時，幾乎一定會聽到這個詞。
