# MCP — AI 的感官器官

> **白話說：** MCP 就像幫 AI 裝上眼睛、耳朵和手。原本的 AI 只會「想」，有了 MCP 就能「看」網頁、「記住」東西、「操作」瀏覽器。

---

## 它到底是什麼？

MCP 全名是 **Model Context Protocol**（模型上下文協議）。

名字很嚇人，但概念很簡單：它是一個**標準接口**，讓 AI 可以連接各種外部工具和服務。

想像 AI 是一個大腦。大腦很聰明，但沒有感官就什麼都做不了：
- 沒有眼睛 → 看不到網頁內容
- 沒有手 → 操作不了瀏覽器
- 沒有記憶卡 → 關掉對話就忘光光

MCP 就是幫大腦接上這些感官器官的「插座規格」。

## 有哪些 MCP 伺服器？

每個 MCP 伺服器就像一個專門的感官器官：

| 名稱 | 比喻 | 白話功能 |
|------|------|---------|
| **Context7** | 📚 專業圖書館員 | 查官方技術文件，確保做法正確 |
| **Sequential** | 🧠 深度思考引擎 | 把複雜問題拆成小步驟慢慢推理 |
| **Serena** | 🗂️ 長期記憶硬碟 | 記住專案結構，跨對話也不會忘 |
| **Magic** | 🎨 UI 設計師 | 快速生成漂亮的網頁元件 |
| **Morphllm** | ✂️ 批次編輯機器 | 一次修改大量檔案裡的相同內容 |
| **Playwright** | 👁️ 瀏覽器操控手 | 自動操作網頁、截圖、測試 |
| **Tavily** | 🔍 即時搜尋雷達 | 搜尋最新的網路資訊 |
| **Chrome DevTools** | 🔬 網頁顯微鏡 | 檢查網頁效能和問題 |

## 我需要自己設定 MCP 嗎？

**通常不用。** 如果你安裝了 [SuperClaude](superclaude.md)，它會幫你把常用的 MCP 都配置好。

就像你買了一台新手機，各種 App 已經預裝好了，你只需要打開來用。

## 它怎麼運作的？（很簡單的版本）

```
你對 Claude Code 說：「幫我查 React 的官方教學文件」
         ↓
Claude Code 心想：「這需要查文件，我來叫 Context7 幫忙」
         ↓
Context7（MCP 伺服器）去查官方文件，把結果回傳
         ↓
Claude Code 把整理好的資訊告訴你
```

**你完全不需要手動操作 MCP**——Claude Code 會自動判斷什麼時候該用哪個工具。
