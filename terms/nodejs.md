# Node.js — 辦公室的水電基礎建設

> **白話說：** Node.js 就像辦公室的水電系統。你的 AI 員工（Claude Code）需要有水有電才能上班，Node.js 就是提供這些基礎設施的東西。

---

## 它到底是什麼？

Node.js 是一個讓 JavaScript（一種程式語言）能在你電腦上執行的工具。

你不需要學 JavaScript，也不需要懂 Node.js 怎麼運作——你只需要**把它裝好**，就像你不需要懂電路原理，只要確認辦公室有插座能充電就好。

## 為什麼我需要它？

很多現代的開發工具（包括 Claude Code）都是用 JavaScript 寫的。安裝 Node.js 之後，你的電腦才有能力執行這些工具。

把它想成：
- 你要看 PDF → 需要先裝 Adobe Reader
- 你要用 Claude Code → 需要先裝 Node.js

## 安裝

👉 **[Node.js 官方網站](https://nodejs.org/)**

點擊綠色的 **LTS**（Long Term Support，長期支援版）按鈕下載安裝即可。

> 💡 **LTS 是什麼？** 就是「穩定版」的意思。像手機有正式版和測試版，你要選正式版（LTS），不要選測試版。

## 裝好後怎麼確認？

打開 [Terminal（終端機）](terminal-cli.md)，輸入：

```bash
node --version
```

看到 `v20.x.x` 或 `v22.x.x` 這樣的數字，就代表裝好了。
