# Push Notification（推播通知）— 便利商店的 Line 提醒

> **白話說：** App 主動發訊息到你手機的通知欄，不用你打開 App 才看得到。

---

## 它到底是什麼？

**Push Notification** 就是 App 主動「推」一則訊息到你的手機螢幕上，即使你沒有打開那個 App。

```
你沒開 App，手機突然震動：
  📱 「蝦皮：你的包裹已出貨！」
  📱 「Line：小明傳了一則訊息」
  📱 「Uber Eats：午餐時間到了，今天吃什麼？」
```

**就像便利商店的 Line 官方帳號——你沒去店裡，它也會主動傳訊息提醒你「集點快到期囉」。**

## 生活比喻 / 實際例子

Push Notification 就像有人主動按你家門鈴告訴你一件事。你不用自己出門去問，消息會自己送上門。

### 推播的幾種用途

| 類型 | 範例 |
|------|------|
| 交易通知 | 「你的訂單已出貨」 |
| 行銷推播 | 「限時特賣，全館 5 折！」 |
| 系統提醒 | 「你的密碼即將過期」 |
| 社交通知 | 「小明在貼文中標記了你」 |

### 常見情境

- **PM：**「這個功能完成後要發 push notification 通知用戶嗎？」
- **Mobile 工程師：**「iOS 的推播要先跟 Apple 申請推播許可證。」→ Apple 的推播有額外的申請手續
- **行銷：**「推播開太多用戶會關掉通知，要控制頻率。」

## 為什麼要知道這個詞？

推播通知是 App 留住用戶、提高回訪率的重要工具。但推得太頻繁用戶會覺得煩、直接關掉通知或刪 App。什麼時候推、推什麼內容，是行銷和 PM 常常需要決定的事。
