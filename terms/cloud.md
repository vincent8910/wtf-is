# Cloud（雲端）— 租辦公室，不用自己蓋大樓

> **白話說：** 不買自己的電腦和機房，租別人的來跑程式、存資料。

---

## 它到底是什麼？

**Cloud**（雲端）就是「用別人的電腦」。你的程式和資料不是放在自己買的機器上，而是放在 Amazon、Google、Microsoft 這些大公司的機房裡，用多少付多少。

```
自己買[伺服器](server.md)（地端，就是自己買機器放在自己公司）：
  → 自己蓋一棟辦公大樓 → 買地、蓋房、裝潢、請管理員
  → 花很多錢、要很久、要自己維護

用雲端：
  → 租辦公室 → 搬進去就能用、不夠再租一間、不用了退租
  → 彈性大、速度快、有人幫你維護
```

**雲端就是租辦公室——不用自己蓋大樓，按需求租就好。**

## 生活比喻 / 實際例子

你用過 Google Drive、iCloud 嗎？你的照片和文件不是存在手機裡（或者不只存在手機裡），而是存在 Google / Apple 的機房裡。這就是最日常的「雲端」體驗。

### 常見的雲端服務商

| 簡稱 | 全名 | 誰家的 |
|------|------|--------|
| AWS | Amazon Web Services | Amazon |
| GCP | Google Cloud Platform | Google |
| Azure | Microsoft Azure | Microsoft |

### 常見情境

- **DevOps（負責維運和自動化部署的）工程師：**「我們的服務都跑在 AWS 上。」→ 我們租 Amazon 的機房
- **老闆：**「雲端費用這個月又爆了。」→ 租金太高了
- **PM：**「這個要上雲嗎？還是放地端（自己公司的機器）？」→ 要租別人的還是用自己的？

## 為什麼要知道這個詞？

現在大多數的軟體服務都跑在雲端上。當你聽到「上雲」、「雲端服務」、「AWS 費用」，知道就是「租別人的電腦來跑我們的東西」就好。雲端費用也是公司的重要支出項目之一。
