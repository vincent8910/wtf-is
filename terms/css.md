# CSS — 網頁的化妝術

> **白話說：** 控制網頁「長怎樣」的語言——顏色、字型、排版、間距全靠它。

---

## 它到底是什麼？

**CSS** 全名 Cascading Style Sheets（層疊樣式表），但你不用記這個。

只要知道：**HTML 負責網頁的內容（骨架），CSS 負責網頁的外觀（化妝）。**

```
沒有 CSS 的網頁：像素顏出門，只有文字和連結，醜到不行
有 CSS 的網頁：上了妝、做了髮型、穿搭到位，美美的
```

**CSS 就是網頁的室內設計師。** 同一間毛坯屋（HTML），交給不同的設計師（CSS），可以變成北歐風、日式風、工業風。

## 生活比喻 / 實際例子

想像一份 Word 文件——你打完字之後，開始調字型大小、改顏色、設邊距、加底色。這些「美化」的動作，在網頁世界就是 CSS 在做的事。

### 常見情境

- **設計師：**「這個按鈕的顏色不對，CSS 改一下就好。」
- **PM：**「為什麼手機版排版跑掉了？」→ 通常是 CSS 的問題
- **前端工程師：**「CSS 跑版了，我修一下。」→ 就是畫面長得跟設計稿不一樣

## 為什麼要知道這個詞？

每次你對網頁的「外觀」有意見——字太小、顏色不對、間距太擠、手機上排版亂掉——這些都跟 CSS 有關。知道這個詞，你就知道該找前端工程師，跟他說「CSS 要調一下」。
