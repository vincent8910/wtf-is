# VPN — 走秘密地下通道

> **白話說：** 透過加密的秘密通道上網，別人看不到你在做什麼、你在哪裡。

---

## 它到底是什麼？

**VPN**（Virtual Private Network，虛擬私人網路）就是幫你的網路連線加一條「秘密地下通道」。

```
沒有 VPN：
  你的電腦 → 公開的馬路 → 網站
  → 路上的人（駭客、老闆、政府）都看得到你去哪、做什麼

有 VPN：
  你的電腦 → 秘密地下通道（加密隧道）→ VPN 出口 → 網站
  → 路上的人只看到你鑽進一條隧道，不知道你去哪做什麼
```

**VPN 就是一條秘密地下通道——你在裡面做什麼，外面的人看不到。**

## 生活比喻 / 實際例子

### VPN 常見的用途

| 用途 | 說明 |
|------|------|
| 公司遠端辦公 | 在家也能安全連進公司內網，像在辦公室一樣 |
| 保護隱私 | 在咖啡廳用 WiFi 時防止被竊聽 |
| 突破地區限制 | 假裝自己在美國，看美國的 Netflix 內容 |
| 連回公司系統 | PM 在外面也能連進公司的內部系統 |

### 常見情境

- **IT：**「在家工作要先連 VPN，才能用公司內網。」→ 要先走秘密通道才能進公司
- **同事：**「我 VPN 連不上，公司的文件打不開。」→ 秘密通道進不去
- **資安工程師：**「公用 WiFi 不安全，建議開 VPN。」→ 在外面上網要走安全通道

## 為什麼要知道這個詞？

很多公司的內部系統（文件、程式碼、管理後台）只有在公司網路才能連。遠端工作時，VPN 就是你的「遠端通行證」。另外，如果你在咖啡廳用公用 WiFi，開 VPN 可以保護你的資料不被偷看。
