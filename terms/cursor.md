# Cursor — 內建 AI 的程式碼編輯器

> **白話說：** 就像 [VS Code](vscode.md) 的加強版，編輯器裡面直接住了一個 AI 助手，可以幫你寫程式、改程式、解釋程式。

---

## 它到底是什麼？

Cursor 是一款 AI 程式碼編輯器（[IDE](ide.md)），基於 VS Code 改造而成。它跟 VS Code 長得幾乎一樣，操作方式也一樣，但最大的差別是：**AI 深度整合在每一個操作裡**。

你可以直接在編輯器裡跟 AI 對話：
- 「幫我把這個函式改成⋯⋯」
- 「這段程式碼在做什麼？」
- 「幫我寫一個 API 來處理⋯⋯」

AI 會直接修改你的程式碼，不用複製貼上來回切換。

Cursor 支援多種 AI 模型，包括 Claude、GPT 等，可以自己選擇要用哪一個。

## 生活比喻 / 實際例子

想像你在寫文章：

- **VS Code** = 一台很好用的打字機，功能齊全
- **Cursor** = 打字機旁邊坐了一個寫作教練，你打字的時候他會即時給建議、幫你改稿、甚至幫你寫段落

實際例子：
- 工程師說：「我改用 **Cursor** 了，寫 code 快很多，AI 直接幫我補完」
- PM 問：「你們用什麼工具寫程式？」工程師：「**Cursor**，有內建 AI 的編輯器」
