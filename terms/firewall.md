# Firewall（防火牆）— 社區大門警衛

> **白話說：** 擋在系統外面的守門員，決定誰可以進來、誰不能進來。

---

## 它到底是什麼？

**Firewall**（防火牆）就是系統的「大門警衛」，檢查每一個想進來的連線，決定放行還是擋掉。

想像你住的社區有門禁管理——

```
警衛（Firewall）的工作：
  → 住戶進出 → 放行 ✅
  → 預約的訪客 → 核對名單後放行 ✅
  → 可疑的陌生人 → 攔下，不准進入 ❌
  → 推銷員 → 直接拒絕 ❌
```

**Firewall 就是社區大門的警衛——只讓你信任的人進來。**

## 生活比喻 / 實際例子

你的 Windows 電腦裡也有內建的「Windows 防火牆」。有時候安裝軟體時會跳出「是否允許此應用程式通過防火牆」——那就是警衛在問你：「這個人你認識嗎？要讓他進來嗎？」

### 防火牆會做的事

- **擋掉可疑的連線**：不認識的 IP 想連進來，直接拒絕
- **限制特定的通訊埠**：只開放必要的門（例如只開 80 和 443 給網站用）
- **記錄可疑的行為**：有人一直想闖入，記下來報告

### 常見情境

- **資安工程師：**「防火牆規則要更新，把這個 IP 封掉。」→ 把可疑人物加到黑名單
- **工程師：**「你的電腦防火牆是不是擋掉了？連不上 server。」→ 警衛把你自己人也擋了
- **PM：**「我們有防火牆保護嗎？」→ 我們的大門有警衛嗎？

## 為什麼要知道這個詞？

防火牆是系統安全的第一道防線。就像社區一定要有門禁一樣，任何暴露在網路上的系統都需要防火牆。當你聽到「防火牆擋掉了」，通常是好事（擋掉壞人）或需要調整設定（不小心擋到自己人）。
