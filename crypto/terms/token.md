# Token（代幣）— 在某個區塊鏈上發行的數位資產，像是平台裡的點數

> **白話說：** 代幣就像**百貨公司禮券或遊樂園代幣**——有人在一個平台上發行的點數，在那個生態系裡可以拿來用。

---

## 它到底是什麼？

**Token（代幣）** 是在現有的[區塊鏈](blockchain.md)上面發行的數位資產。

這裡要分清楚兩個容易搞混的概念：

| 名稱 | 說明 | 例子 |
|------|------|------|
| **Coin（幣）** | 有自己的區塊鏈 | [BTC](bitcoin.md)（比特幣鏈）、[ETH](ethereum.md)（以太坊鏈） |
| **Token（代幣）** | 寄生在別人的區塊鏈上 | USDT、UNI、SHIB（都在以太坊上發行） |

打個比方：

- **Coin** 就像自己蓋了一棟百貨公司
- **Token** 就像在別人的百貨公司裡開了一個櫃位、發了自己的禮券

在[以太坊](ethereum.md)上，最常見的代幣標準叫 **ERC-20**。任何人都可以用這個標準發行自己的代幣——這也是為什麼市場上有幾千種代幣，因為發行門檻很低。

代幣的種類：

| 種類 | 用途 | 例子 |
|------|------|------|
| 支付型代幣 | 拿來付錢、轉帳 | 各種[加密貨幣](cryptocurrency.md) |
| 功能型代幣 | 在特定平台裡使用 | UNI（Uniswap 的治理投票） |
| 治理型代幣 | 持有者可以投票決定平台規則 | AAVE、COMP |
| [穩定幣](stablecoin.md) | 價格釘住法幣 | USDT、USDC |
| NFT | 獨一無二的數位收藏品 | CryptoPunks、Bored Ape |

## 生活比喻 / 實際例子

想像你去兒童新樂園：

1. 你在門口用台幣換了一堆**遊樂園代幣**
2. 這些代幣可以拿去玩各種遊樂設施
3. 代幣只在這個遊樂園有用
4. 你也可以把代幣轉給朋友

區塊鏈上的 Token 也是這樣——在某個生態系裡有特定功能，可以跟別人交換。

你會在這些地方聽到：

- 「這個項目發了自己的 **Token**，持有者可以投票」
- 「我買了一些 **代幣**，等它漲了再賣」
- 「這個 **Token** 是 ERC-20 的，可以存在以太坊錢包」
- 「小心那些亂發 **代幣** 的詐騙項目」

## 為什麼要知道這個詞？

Token 是幣圈裡每天都會用到的基本詞彙：

- **買賣加密貨幣時**，你買的很多東西其實是 Token 而不是 Coin
- **參與 DeFi 時**，各種協議都有自己的治理 Token
- **判斷項目時**，了解一個 Token 的用途和發行量，是基本的分析功課
- **避免詐騙時**，因為發行 Token 太容易了，所以假幣、空氣幣特別多

記住：有自己區塊鏈的叫 Coin，寄生在別人鏈上的叫 Token。但日常對話裡，大家常常混著用，不用太計較。
