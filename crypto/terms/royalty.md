# Royalty（版稅） — NFT 每次轉手時，原作者都能抽一筆

> **白話說：** 房子每次轉賣，建商都能抽 5%——你畫了一幅數位畫，不只第一次賣能賺錢，以後每次有人轉賣，你都能自動收到一筆「版稅」。

---

## 它到底是什麼？

**Royalty（版稅）** 在 [NFT](nft.md) 的世界裡，是指每當一個 NFT 在二手市場上被轉賣時，**原始創作者自動獲得的一筆分潤**。

在傳統的藝術市場裡，畫家把畫賣出去之後，不管那幅畫後來被轉賣了幾次、漲了多少價，畫家一毛都拿不到。但 NFT 的版稅機制改變了這件事：

```
第一次賣出：
  畫家賣給 A → 畫家收到 1 ETH

A 轉賣給 B（假設版稅 5%）：
  A 賣給 B → A 收到 9.5 ETH，畫家自動收到 0.5 ETH（版稅）

B 轉賣給 C：
  B 賣給 C → B 收到 19 ETH，畫家自動收到 1 ETH（版稅）
```

這個版稅比例是創作者在 [Mint（鑄造）](mint.md) NFT 時就設定好的，通常在 2.5% 到 10% 之間。每次有人轉賣這個 NFT，版稅就會自動從交易金額中扣除，直接打到創作者的[錢包](wallet.md)裡。

| 版稅比例 | 賣 10 ETH 時創作者拿到 | 常見於 |
|---------|---------------------|-------|
| 2.5% | 0.25 ETH | 大部分 PFP 項目 |
| 5% | 0.5 ETH | 藝術類 NFT |
| 10% | 1 ETH | 比較少見，買家會嫌太高 |

不過要注意，版稅制度正在經歷一些變化。有些交易平台為了吸引買家，開始允許買家選擇「不付版稅」或「自訂版稅比例」。這引發了很大的爭議——創作者認為這是他們應得的收入，但買家覺得版稅太高會降低交易意願。

## 生活比喻 / 實際例子

想像你是一個唱片歌手：

- **傳統模式**：你把歌賣給唱片公司，拿到一筆錢，之後不管這首歌被播了幾百萬次，你都只拿到最初那筆
- **NFT 版稅模式**：你每次被播放都能收到錢，而且是「自動收到」——不用跑去跟任何人要錢，區塊鏈幫你收

或者像房東收租的概念：
- 你蓋了一棟房子（創作了一個 NFT）
- 你賣掉了房子（第一次賣出 NFT）
- 但你設定了一個魔法條款：「以後每次這棟房子被轉賣，我都能抽 5%」
- 房子從 500 萬漲到 5,000 萬，每次轉手你都能分一杯羹

**在幣圈你會聽到：**
- 「這個藝術家靠 **Royalty** 月入百萬，他的 NFT 每天都在被交易。」
- 「這個平台不強制 **版稅** 了，創作者超不爽。」
- 「我設定 5% 的 **Royalty**，希望之後每次轉賣都能有被動收入。」

## 為什麼要知道這個詞？

Royalty 是 NFT 最革命性的特色之一，它改變了創作者的收入模式：

- **對創作者**：這是一個「持續被動收入」的機會。你不再只能靠第一次銷售賺錢，每次轉賣都能分潤
- **對買家**：版稅是你交易成本的一部分。買一個 NFT 時，除了 [Floor Price](floor-price.md) 和 [Gas Fee](gas-fee.md)，還要考慮版稅會從你的賣價裡扣掉多少
- **對整個生態**：版稅讓創作者有動力持續經營項目，因為項目越成功、交易越活絡，創作者賺越多。這讓創作者和持有者的利益一致
