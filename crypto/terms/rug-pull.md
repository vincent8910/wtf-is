# Rug Pull — 項目方捲款跑路

> **白話說：** 就像一間餐廳收了你的訂金和預約，結果隔天老闆帶著錢跑了，店門鎖上、電話打不通。

---

## 它到底是什麼？

Rug Pull（字面意思是「抽地毯」，像是你站在地毯上，有人突然把地毯抽走讓你摔倒）是幣圈最惡劣的詐騙手法之一。簡單來說，就是一個加密貨幣項目的開發團隊，在募集了大量投資人的資金後，突然把所有的錢提走、消失不見。

Rug Pull 的典型套路：

1. **包裝上線**：團隊創造一個新的代幣，做出漂亮的網站和白皮書，承諾各種了不起的功能和報酬。
2. **吸引投資**：透過社群行銷、找網紅推薦、甚至做出假的合作夥伴名單，吸引投資人買進。
3. **建立流動性**：在去中心化交易所（DEX）上建立[流動性池](liquidity-pool.md)，讓代幣可以交易。
4. **捲款跑路**：等到池子裡累積了足夠多的資金，團隊一口氣把流動性池裡的所有資金提走，代幣瞬間變成廢紙。

跟 [Pump and Dump](pump-and-dump.md) 不同的是，Pump and Dump 至少幣還在、只是暴跌；而 Rug Pull 是開發者直接帶走所有的錢，你的投資變成零，完全沒有任何回本的可能。

Rug Pull 在 2021 年的 [DeFi](defi.md) 熱潮中特別猖獗。因為任何人都可以在區塊鏈上創建新代幣（技術上非常簡單），加上 DeFi 平台不需要審核就可以上架交易，所以詐騙者的門檻非常低。

## 生活比喻 / 實際例子

想像有人在你家社區開了一間超高級的火鍋店，裝潢漂亮、菜單誘人、還推出「充值 5,000 送 2,000」的超划算儲值方案。很多鄰居都衝去儲值了。結果一個月後，某天你去吃飯發現店門拉下來了，老闆跑了，你的儲值金全部拿不回來。這就是 Rug Pull。

> **在幣圈你會聽到：**
> - 「那個 NFT 項目被 Rug Pull 了，開發者帶著 500 萬美元跑了。」
> - 「投資前一定要查團隊背景，不然小心被 Rug Pull。」
> - 「流動性沒有鎖定的項目，Rug Pull 的風險超高。」

## 為什麼要知道這個詞？

Rug Pull 是幣圈裡損失最慘重的風險之一，因為錢一旦被捲走就幾乎不可能拿回來。保護自己的方法包括：檢查項目的流動性有沒有「鎖定」（鎖定就是開發者不能隨意提走）、團隊成員有沒有公開真實身份、智能合約有沒有經過第三方審計。如果一個項目承諾的報酬高得離譜（年化 10,000% 之類的），又沒有合理的商業模式，那很可能就是 Rug Pull 的前奏。記住：如果好到不像真的，那通常就不是真的。
