# NFT（非同質化代幣） — 獨一無二的數位證書，證明你擁有某個數位物品

> **白話說：** 數位世界的房地產權狀——你手機裡的那張圖片誰都能截圖，但只有你有「官方認證的擁有證明」。

---

## 它到底是什麼？

**NFT** 是 **Non-Fungible Token** 的縮寫，中文叫「非同質化代幣」。聽起來超難懂，但拆開來看很簡單：

- **Non-Fungible（非同質化）**：意思是「獨一無二、不能互換」。一張千元鈔可以跟另一張千元鈔互換，因為它們價值相同（這叫「同質化」）。但《蒙娜麗莎》原畫只有一幅，你不能說「拿另一幅畫來換」——這就是「非同質化」。
- **Token（代幣）**：在[區塊鏈](blockchain.md)上的一筆數位記錄。

所以 NFT 就是：**寫在區塊鏈上的一張「獨一無二的數位證書」**，用來證明某個數位物品（圖片、音樂、影片、遊戲道具等）屬於你。

你可能會問：「圖片不是隨便都能下載嗎？」沒錯，任何人都能右鍵另存圖片。但 NFT 的重點不是圖片本身，而是**「誰是官方認證的擁有者」**。就像你可以印一張《蒙娜麗莎》的海報掛在家裡，但真正的原作收藏者是羅浮宮——NFT 就是那張「原作擁有證明」。

每個 NFT 都有自己的編號，記錄在[區塊鏈](blockchain.md)上，任何人都可以查到「這個 NFT 現在屬於誰」、「它過去的每一任主人是誰」。這些紀錄無法竄改、無法偽造。購買或[鑄造](mint.md) NFT 時，你需要支付 [Gas Fee](gas-fee.md)。

## 生活比喻 / 實際例子

想像你在網路上買了一幅數位畫作：

| 沒有 NFT 的時候 | 有 NFT 的時候 |
|---------------|-------------|
| 你買了一張 JPG 圖片 | 你買了一張 JPG 圖片 + 一張寫在區塊鏈上的「擁有權證書」 |
| 別人截圖就跟你一模一樣 | 別人可以截圖，但區塊鏈上寫的擁有者是你 |
| 你無法證明你是「正版買家」 | 全世界都能查到你是官方認證的擁有者 |
| 轉賣很麻煩，沒有公信力 | 轉賣就像轉帳一樣簡單，紀錄永久保存 |

**在幣圈你會聽到：**
- 「我昨天花了 0.5 ETH [Mint](mint.md) 了一個 **NFT**，現在 [Floor Price](floor-price.md) 已經漲到 2 ETH 了。」
- 「這個 **NFT** 項目有 [白名單](whitelist.md) 機制，搶到就等於穩賺。」
- 「我把我的猴子 **NFT** 設成推特頭像了，這就是所謂的 [PFP](pfp.md)。」

## 為什麼要知道這個詞？

NFT 在 2021-2022 年爆紅，雖然市場冷卻過，但它的底層概念——**用區塊鏈證明數位物品的擁有權**——已經被廣泛應用在：

- **數位藝術品**：藝術家直接賣作品給收藏者，還能透過 [Royalty（版稅）](royalty.md) 在每次轉手時持續獲利
- **遊戲道具**：遊戲裡的武器、角色變成 NFT，可以跨遊戲使用或在市場上交易
- **門票、會員卡**：演唱會門票做成 NFT，無法偽造
- **社群身份**：擁有某個 NFT 就像加入某個俱樂部，例如知名的 Bored Ape Yacht Club

即使你不打算買 NFT，了解這個概念也能幫你看懂新聞和社群討論。
