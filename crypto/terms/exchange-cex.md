# Exchange / CEX（交易所）— 買賣加密貨幣的平台，像是幣圈的證券商

> **白話說：** 交易所就像**加密貨幣的蝦皮 + 股票券商**——你在上面開帳戶、存錢、買賣幣，一切都由平台幫你處理。

---

## 它到底是什麼？

Exchange（交易所），又稱 CEX（Centralized Exchange，中心化交易所），是一個讓你**用台幣或其他貨幣買賣加密貨幣的平台**。就像你要買股票得開券商帳戶，要買加密貨幣也得先在交易所開帳戶。

全球知名的交易所包括：

| 交易所 | 總部 | 特色 |
|--------|------|------|
| Binance（幣安） | 全球 | 全球最大，幣種最多 |
| Coinbase | 美國 | 美國合規上市公司 |
| MAX | 台灣 | 台灣在地，支援台幣出入金 |
| ACE | 台灣 | 台灣在地，介面中文友善 |
| BitoPro（幣託） | 台灣 | 台灣在地，搭配 BitoEX |

在交易所上你可以做的事：

1. **入金**：把台幣匯進去（通常是銀行轉帳）
2. **買幣**：用台幣買比特幣、以太幣等等
3. **賣幣**：把幣賣掉換回台幣
4. **交易**：在不同加密貨幣之間互換（例如把比特幣換成以太幣）
5. **出金**：把台幣領回你的銀行帳戶

「中心化」的意思是交易所由一間公司營運，你的幣實際上是由交易所保管的。就像你把錢存在銀行，銀行幫你管——方便，但你得信任這間公司不會倒閉或被駭。這也是為什麼有人會把幣提領到自己的[錢包](wallet.md)裡自己保管。

## 生活比喻 / 實際例子

交易所就像你熟悉的蝦皮 + 銀行的綜合體：

- 蝦皮的部分：買賣雙方在上面交易，平台幫你撮合、抽手續費
- 銀行的部分：你的帳戶裡有錢和幣，平台幫你保管

你在全聯買東西，付錢就拿貨，不用管背後的供應鏈。在交易所買幣也一樣——點一下「買入」，你的帳戶就多了一筆比特幣。但這些幣的[私鑰](private-key.md)其實在交易所手上，不在你手上。

**在幣圈你會聽到：**

- 「我在幣安上買了一顆以太幣。」—— 最基本的交易行為。
- 「交易所倒了怎麼辦？所以我都會把幣提領到自己的[冷錢包](cold-wallet.md)。」—— FTX 倒閉事件後，很多人學到的教訓。
- 「你做 [KYC](kyc.md) 了嗎？交易所開帳戶要上傳身分證。」—— 合規交易所都要求身分驗證。

## 為什麼要知道這個詞？

交易所是大部分人進入幣圈的第一站：

- **第一次買幣時**，你需要選一個交易所來開帳戶（台灣人通常先從 MAX 或 ACE 開始）
- **比較手續費時**，不同交易所的費率和服務不一樣，了解之後才能選最適合的
- **安全意識上**，知道交易所跟[DEX](dex.md)（去中心化交易所）的差別，才能做出適合自己的選擇

建議搭配了解 [DEX](dex.md)（去中心化交易所）、[KYC](kyc.md)（身分驗證）和[現貨交易](spot.md)的概念。
