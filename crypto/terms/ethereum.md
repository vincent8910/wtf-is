# Ethereum（以太坊）— 不只能轉帳，還能在上面蓋各種應用程式的區塊鏈

> **白話說：** 如果 [Bitcoin](bitcoin.md) 是數位黃金，**Ethereum 就是能蓋房子的土地**——你不只能在上面存錢，還能在上面開店、玩遊戲、簽合約。

---

## 它到底是什麼？

**Ethereum**（以太坊）是 2015 年由 Vitalik Buterin（人稱「V 神」）推出的[區塊鏈](blockchain.md)平台，它的原生貨幣叫 **ETH（以太幣）**。

[比特幣](bitcoin.md)的功能比較單純——就是轉帳、存錢。以太坊則多了一個超強功能：**智能合約（Smart Contract）**。

白話說，智能合約就是：**寫在區塊鏈上的自動執行程式。** 條件達成就自動做事，不需要中間人。

| 比較 | Bitcoin | Ethereum |
|------|---------|----------|
| 主要功能 | 數位貨幣、價值儲存 | 數位貨幣 + 執行程式 |
| 比喻 | 黃金 | 能蓋房子的土地 |
| 貨幣名稱 | BTC | ETH |
| 共識機制 | [工作量證明 PoW](proof-of-work.md) | [權益證明 PoS](proof-of-stake.md) |
| 總量上限 | 2,100 萬顆 | 沒有硬上限 |

## 生活比喻 / 實際例子

把以太坊想像成一塊公共用地：

- **比特幣**像黃金——你可以買來放著、等它漲價、拿來交換
- **以太坊**像一塊土地——你可以在上面蓋 7-Eleven、蓋電影院、蓋遊樂園

在這塊「土地」上已經蓋了很多東西：

| 蓋了什麼 | 例子 |
|---------|------|
| 去中心化交易所 | Uniswap——不需要幣安，直接跟別人換幣 |
| 借貸平台 | Aave——不用銀行就能借錢 |
| NFT 市場 | 數位藝術品買賣 |
| 遊戲 | 邊玩邊賺的鏈遊 |
| [穩定幣](stablecoin.md) | USDT、USDC 很多都發行在以太坊上 |

智能合約的生活比喻：

想像你在蝦皮買東西，平台會幫你「先收錢、確認收貨後再付給賣家」。智能合約做的事情一樣，但不需要蝦皮當中間人——程式自動執行、寫在區塊鏈上、誰也改不了。

## 為什麼要知道這個詞？

以太坊是目前加密貨幣世界裡生態最豐富的平台：

- **買幣時**，ETH 是市值第二大的加密貨幣，僅次於比特幣
- **玩 DeFi（去中心化金融）時**，大部分都建在以太坊上
- **看 NFT 新聞時**，「鑄造在以太坊上」是最常見的說法
- **很多[代幣](token.md)都是 ERC-20 標準**，也就是在以太坊上發行的

簡單記：比特幣是加密貨幣的「黃金」，以太坊是加密貨幣的「作業系統」。
