# PFP（Profile Picture） — 拿來當社群頭像的 NFT

> **白話說：** 花錢買的 Line 大頭貼，但這不只是裝飾——它是你在幣圈的身分象徵，就像背 LV 包包一樣，一看就知道你是哪個圈子的人。

---

## 它到底是什麼？

**PFP** 是 **Profile Picture** 的縮寫，意思是「個人頭像」。在 [NFT](nft.md) 的世界裡，PFP 特指那些**被設計來當社群媒體頭像使用的 NFT 系列**。

最早的 PFP 系列就是大家可能聽過的「CryptoPunks（像素頭像）」和「Bored Ape Yacht Club（無聊猿）」。這些 NFT 的共同特徵是：

- **每一個都是獨一無二的頭像**：一個系列通常有 10,000 個，每個的表情、配件、背景都不同
- **設計適合當頭像**：正方形、臉部特寫、風格鮮明
- **買了之後拿來當推特/社群頭像**：讓別人一看就知道「這個人有這個 NFT」

你可能會問：「不就是一張圖嗎？為什麼有人願意花幾十萬甚至幾百萬買？」因為 PFP NFT 不只是圖片，它代表的是：

| 層面 | 說明 | 比喻 |
|------|------|------|
| **身分象徵** | 擁有某個 PFP = 加入某個社群 | 戴 Rolex = 有一定經濟實力 |
| **社群入場券** | 有些社群只有持有者才能進入 | 高爾夫球俱樂部的會員卡 |
| **投資品** | 稀有的 PFP 會增值 | 限量版球鞋，買了放著就漲價 |
| **文化認同** | 用什麼 PFP 代表你認同什麼文化 | 穿什麼品牌代表你的生活風格 |

推特（現在叫 X）甚至推出過「六角形頭像」功能——如果你的頭像是驗證過的 NFT，它會顯示成六角形而不是圓形，讓大家知道你是「真正擁有」這個頭像，而不是隨便截圖來的。

## 生活比喻 / 實際例子

想像你走進一個高級派對：

- **普通來賓**：穿普通衣服進場，大家不特別注意你
- **戴著無聊猿 PFP 的人**：就像穿著一件大家都認得的名牌外套走進去，馬上有人來跟你搭話：「嘿，你也是猿猴俱樂部的？」

或者像 LINE 的貼圖：
- 免費的預設頭像 = 不花錢，但看起來很普通
- 花 30 元買 LINE 貼圖當頭像 = 稍微有個人風格
- 花幾十萬買 PFP NFT 當頭像 = 頂級的數位身分象徵

**在幣圈你會聯到：**
- 「他的推特頭像是 Bored Ape，光那個 **PFP** 就值好幾百萬。」
- 「最近流行用 Azuki 當 **PFP**，那個日系畫風超好看。」
- 「我在 [Floor Price](floor-price.md) 很低的時候買了一個來當 **PFP**，現在漲了 5 倍。」

## 為什麼要知道這個詞？

PFP 是 NFT 世界裡最主流、最容易理解的應用。了解 PFP 可以幫你：

- **看懂社群文化**：當你在推特或社群上看到一堆人用同樣風格的頭像，那通常就是某個 PFP 系列的持有者
- **理解 NFT 的價值**：很多人質疑「一張圖為什麼值這麼多錢」，但 PFP 的價值不只是圖片本身，還包含了社群、身分、投資等多重意義
- **辨別「圈內人」**：幣圈人一看 PFP 就大概知道你的「等級」和立場，就像看人穿什麼衣服一樣
