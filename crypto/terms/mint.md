# Mint（鑄造） — 把一個數位物品變成 NFT 的過程

> **白話說：** 去戶政事務所幫新生兒辦出生證明——寶寶生出來了（作品做好了），但要去登記才算數（上鏈才變成 NFT）。

---

## 它到底是什麼？

**Mint** 的英文原意是「鑄造」，就像鑄幣廠把金屬鑄造成硬幣一樣。在加密貨幣的世界裡，Mint 的意思是：**把一個數位物品（圖片、音樂、影片等）正式登記到[區塊鏈](blockchain.md)上，讓它變成一個 [NFT](nft.md)**。

你可以把整個過程想像成這樣：

```
創作者畫好一張圖
      ↓
按下「Mint」按鈕
      ↓
支付 Gas Fee（手續費）
      ↓
區塊鏈上多了一筆記錄：「這個 NFT 存在了，擁有者是 XXX」
      ↓
恭喜！你的圖片現在是一個正式的 NFT 了
```

Mint 的過程需要支付 [Gas Fee](gas-fee.md)，因為你要請區塊鏈幫你「記帳」。就像去戶政事務所辦事要付規費一樣，區塊鏈上的每個動作都有成本。

Mint 分成兩種常見情境：

| 情境 | 說明 | 比喻 |
|------|------|------|
| **創作者 Mint** | 藝術家把自己的作品鑄造成 NFT | 作家出版新書，拿到 ISBN 編號 |
| **買家 Mint** | NFT 項目開賣時，買家付錢鑄造一個 | 新建案開賣，買家付錢登記取得所有權 |

很多 NFT 項目會設定一個「Mint 時間」，時間一到大家同時搶著 Mint。如果你在 [Whitelist（白名單）](whitelist.md) 上，通常可以提早 Mint，不用跟大家搶。

## 生活比喻 / 實際例子

Mint 就像辦出生證明的流程：

1. **寶寶出生** = 數位作品完成（畫好一張圖）
2. **去戶政事務所** = 連接區塊鏈平台（如 OpenSea）
3. **填寫資料** = 上傳作品、設定名稱和描述
4. **繳規費** = 支付 [Gas Fee](gas-fee.md)
5. **拿到出生證明** = 你的作品正式成為 [NFT](nft.md)，有了獨一無二的編號

**在幣圈你會聽到：**
- 「這個項目明天晚上 8 點開放 **Mint**，白名單可以提早一小時。」
- 「我剛 **Mint** 了一個，[Gas Fee](gas-fee.md) 花了我 50 美金，比 NFT 本身還貴…」
- 「Free Mint 的項目要小心，雖然免費鑄造，但 Gas Fee 還是要付。」

## 為什麼要知道這個詞？

如果你對 [NFT](nft.md) 有興趣，Mint 是你一定會碰到的第一步操作。不管是自己創作 NFT，還是參與別人的 NFT 項目，都離不開 Mint 這個動作。

搞懂 Mint，你就知道：
- NFT 不是憑空出現的，是有人「鑄造」出來的
- Mint 要花錢（[Gas Fee](gas-fee.md)），所以要算好成本
- 搶 Mint 是幣圈常見的活動，有些人靠搶到稀有 NFT 賺了不少
