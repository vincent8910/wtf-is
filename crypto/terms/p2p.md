# P2P（點對點）— 不透過中間人，直接彼此交易

> **白話說：** P2P 就像**直接跟同學借筆記，不用透過老師**——你跟對方直接互動，中間沒有任何人在「經手」。

---

## 它到底是什麼？

**P2P** 是 **Peer-to-Peer** 的縮寫，中文叫「點對點」或「對等式網路」。

核心概念很簡單：**兩個人直接交流，不需要透過中間人。**

在傳統世界裡，很多事情都需要中間人：

| 場景 | 中間人 | 如果用 P2P |
|------|--------|-----------|
| 轉帳給朋友 | 銀行 | 直接用[加密貨幣](cryptocurrency.md)轉給對方 |
| 買東西 | 蝦皮、momo（平台） | 直接跟賣家交易 |
| 傳訊息 | Line 伺服器 | 訊息直接傳到對方裝置 |
| 借錢 | 銀行（放貸中介） | DeFi 借貸平台直接撮合 |

[區塊鏈](blockchain.md)的核心精神就是 P2P——你的[比特幣](bitcoin.md)可以直接從你的錢包轉到對方的錢包，不需要經過任何銀行或支付公司。

P2P 的好處：
- **不用付中間人的手續費**（或大幅降低）
- **不用等中間人處理**（銀行匯款要等 1-3 天，P2P 幾分鐘到幾十分鐘）
- **不怕中間人出包**（銀行系統掛了你就轉不了帳）
- **隱私更好**（不需要跟第三方透露交易細節）

## 生活比喻 / 實際例子

**傳統方式（有中間人）：**

你要匯 1,000 元給在日本的朋友：
1. 你走進銀行 → 填匯款單 → 銀行收手續費
2. 你的銀行 → 中轉銀行 → 朋友的日本銀行
3. 等 1-3 個工作天
4. 被收了好幾層手續費

**P2P 方式：**

你用加密貨幣轉帳給在日本的朋友：
1. 你打開錢包 App → 輸入對方地址 → 按送出
2. 直接從你的錢包到對方的錢包
3. 幾分鐘就到
4. 手續費通常很低

在幣圈裡，P2P 也指一種交易方式——「P2P 交易」：

- 幣安等交易所有 **P2P 交易區**，讓你直接跟另一個人買賣加密貨幣
- 你用台幣轉給賣家，賣家把幣轉給你，交易所當中間公證人
- 有點像蝦皮的「面交」功能

你會在這些地方聽到：

- 「[比特幣](bitcoin.md)是 **P2P** 的電子現金系統」（比特幣白皮書的標題）
- 「我用 **P2P** 交易買了一些 USDT」
- 「區塊鏈就是一個 **P2P** 網路」
- 「**P2P** 轉帳不需要經過銀行」

## 為什麼要知道這個詞？

P2P 是理解區塊鏈和加密貨幣的基礎概念：

- **理解比特幣時**，它的[白皮書](whitepaper.md)標題就是「P2P 電子現金系統」
- **買賣加密貨幣時**，P2P 交易是入門常用的方式
- **理解[去中心化](decentralization.md)時**，P2P 就是去中心化的技術基礎
- **跨國轉帳時**，P2P 加密貨幣轉帳越來越受歡迎

簡單記：P2P = 不透過中間人，你跟我直接來。
