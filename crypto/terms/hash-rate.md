# Hash Rate（算力）— 挖礦電腦每秒能算多少次

> **白話說：** 算力就像**你的計算機每秒能按多少次**——按得越快，就越有機會搶先算出正確答案、贏得[區塊獎勵](block-reward.md)。

---

## 它到底是什麼？

**Hash Rate（算力）** 就是一台[挖礦](mining.md)電腦每秒能計算多少次 [Hash](hash.md)。

在 [PoW（工作量證明）](proof-of-work.md)的區塊鏈裡，挖礦就是不斷嘗試不同的 [Nonce](nonce.md)，看哪個能產生符合條件的 Hash 值。你的電腦每秒能試越多次，就越有機會先猜到正確答案。

算力的單位：

| 單位 | 代表 | 多少 |
|------|------|------|
| H/s | Hash per second | 每秒 1 次 |
| KH/s | Kilo Hash | 每秒 1,000 次 |
| MH/s | Mega Hash | 每秒 100 萬次 |
| GH/s | Giga Hash | 每秒 10 億次 |
| TH/s | Tera Hash | 每秒 1 兆次 |
| PH/s | Peta Hash | 每秒 1,000 兆次 |
| EH/s | Exa Hash | 每秒 100 萬兆次 |

目前[比特幣](bitcoin.md)全網算力大概在幾百 EH/s 左右——也就是全世界所有比特幣礦機加起來，每秒嘗試幾百萬兆次計算。這個數字大到很難想像。

不同設備的算力差距：

| 設備 | 大約算力 |
|------|---------|
| 一般筆電 CPU | 幾十 MH/s |
| 高階顯卡 GPU | 幾百 MH/s |
| 專業礦機 ASIC | 幾十到幾百 TH/s |

## 生活比喻 / 實際例子

想像一個猜密碼的比賽：

- 密碼鎖有 100 萬個可能的組合
- 每個參賽者用自己的速度猜
- A 選手每秒可以試 100 次（算力低）
- B 選手每秒可以試 10,000 次（算力高）
- B 先猜到的機率是 A 的 100 倍

算力就是你「猜密碼」的速度。速度越快，中獎機率越高。

在幣圈裡，算力有兩個層面：

**1. 個人算力**——你自己的礦機每秒能算多少次
- 算力越高，挖到幣的機率越大
- 但因為全網算力太高了，個人礦機幾乎不可能單獨挖到
- 所以大部分人會加入「礦池」，合作挖礦、按算力比例分獎勵

**2. 全網算力**——全世界所有礦機加起來的總算力
- 全網算力越高，代表網路越安全（攻擊成本越高）
- 全網算力突然下降，可能表示有大量礦機關機（電費太貴、幣價太低）

你會在這些地方聽到：

- 「比特幣全網 **算力** 又創新高了」
- 「這台礦機的 **Hash Rate** 是 100 TH/s」
- 「全網 **算力** 下降，可能是有大礦場關機了」
- 「**算力** 越高，網路越安全」

## 為什麼要知道這個詞？

算力是衡量 PoW 區塊鏈健康程度的關鍵指標：

- **判斷網路安全性時**，全網算力越高越安全
- **評估挖礦收益時**，你的算力佔全網比例決定了你能分到多少
- **看新聞時**，「算力遷移」「算力暴跌」都是重要消息
- **理解挖礦成本時**，更高的算力意味著更多的電費和設備投資

簡單記：算力 = 猜密碼的速度。速度越快，越有機會挖到幣。
