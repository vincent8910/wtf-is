# NDA（保密協議） — 簽了就不能把聽到的事說出去

> **白話說：** NDA 就像**朋友跟你說「我跟你講一個秘密，你不能跟別人說喔」**——只是這次是白紙黑字寫下來，說出去要負法律責任。

---

## 它到底是什麼？

**NDA** 是 **Non-Disclosure Agreement** 的縮寫，中文叫「保密協議」或「保密條款」。

白話說就是：**一份合約，簽了之後你不能把知道的機密資訊告訴別人。**

NDA 通常會寫清楚：

```
NDA 裡面會有的內容：
├─ 什麼是「機密資訊」？（定義範圍）
├─ 誰不能說？（簽署的人）
├─ 不能跟誰說？（通常是除了工作需要以外的所有人）
├─ 保密多久？（通常 1-5 年，有些是永久）
└─ 違反了怎麼辦？（賠償金額、法律責任）
```

NDA 有分：

| 類型 | 說明 | 常見場景 |
|------|------|---------|
| 單方 NDA | 只有一方要保密 | 員工入職、聽取簡報 |
| 雙方 NDA | 雙方都要保密 | 兩家公司談合作 |

## 生活比喻 / 實際例子

想像你要去一家公司面試高階職位：

```
面試前：
├─ 公司：「等等會跟你分享我們還沒公開的新產品計畫」
├─ 公司：「但你要先簽這份 NDA」
├─ 你：（看了看，簽名）
├─ 面試中：聽到了很多未公開的資訊
└─ 面試後：不管有沒有錄取，你都不能把聽到的告訴任何人
```

常見需要簽 NDA 的場景：

- **新人到職**：入職第一天就會簽，保護公司的營業秘密
- **合作洽談**：兩家公司要談合作前，先簽 NDA 才開始分享各自的資訊
- **投資會議**：[VC](vc.md) 在聽你的商業計畫前，可能會先簽 NDA
- **外包合作**：把工作外包給別人做之前，先讓對方簽 NDA

在公司裡你會聽到：

- 「先簽 **NDA** 再談」→ 先簽保密協議才能分享機密資訊
- 「這個受 **NDA** 保護，不能說」→ 這個資訊是機密，有保密義務
- 「入職要簽 **NDA**」→ 新人報到時要簽的保密合約

## 為什麼要知道這個詞？

NDA 是職場上最常遇到的法律文件之一：

- **入職時**：幾乎所有公司都會要你簽 NDA（或是競業禁止條款）
- **跳槽時**：你在前公司知道的機密，受 NDA 保護，不能帶到新公司
- **談合作時**：NDA 是商業洽談的第一步
- **日常工作**：很多公司內部的資訊（財報、產品計畫、客戶資料）都受 NDA 保護

一句話提醒：**簽 NDA 前要看清楚內容，特別是保密範圍和期限。** 不確定的話，可以問公司或找律師確認。
