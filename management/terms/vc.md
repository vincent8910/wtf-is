# VC（創投） — 專門投資新創公司的機構

> **白話說：** VC 就像**星探**——他們到處找有潛力的新人（新創公司），投入資源培養（投資），賭他們未來會大紅大紫（上市或被收購），然後賺好幾倍回來。

---

## 它到底是什麼？

**VC** 是 **Venture Capital** 的縮寫，中文叫「創業投資」或簡稱「創投」。

VC 的工作就是：**用錢投資有潛力的新創公司，等公司變值錢之後再賣掉股份來賺錢。**

VC 的運作方式：

```
VC 的賺錢邏輯：
1. 募集一筆基金（通常來自有錢人和機構）
2. 用這筆錢投資很多新創公司
3. 大部分會失敗（賠錢）
4. 但只要有幾家大成功（上市或被收購）
5. 賺回來的足以覆蓋所有虧損，還有得賺
```

新創公司的融資通常有幾個階段：

| 階段 | 通常金額 | 通常由誰投 |
|------|---------|-----------|
| 種子輪（Seed） | 幾百萬到幾千萬台幣 | 天使投資人、早期 VC |
| A 輪 | 幾千萬到上億台幣 | VC |
| B 輪 | 上億台幣 | VC、成長型基金 |
| C 輪以後 | 數億以上 | 大型 VC、私募基金 |
| IPO（上市） | 公開市場 | 所有投資人 |

## 生活比喻 / 實際例子

想像你有一個很棒的滷味配方，想開一間滷味連鎖店：

```
你的狀況：
├─ 有很棒的配方（產品）
├─ 有開過一間店的經驗（初步驗證）
├─ 但你沒有錢一次開 20 間店（缺資金）
└─ 你也不想去銀行借錢（怕還不起）

VC 出場：
├─ 「我投你 1,000 萬，幫你開 20 間店」
├─ 「但我要你公司 20% 的股份」
├─ 如果你成功開到 100 間店，公司很值錢 → VC 賺很多
└─ 如果你失敗了，VC 認賠 → 這就是「風險投資」
```

台灣知名的 VC 機構：

- 國發基金、中華開發、AppWorks（之初創投）
- 國際知名：紅杉資本（Sequoia）、a16z、Y Combinator

在公司裡你會聽到：

- 「我們在跟幾家 **VC** 談」→ 我們在跟創投公司談投資
- 「**VC** 最看重成長速度」→ 創投公司最在意公司成長有多快
- 「這輪是某某 **VC** 領投的」→ 這輪融資是某家創投主導的

## 為什麼要知道這個詞？

如果你在新創公司工作，VC 跟你的關係比你想的更近：

- **公司能不能活下去**：很多新創的 [現金流](cash-flow.md) 就是靠 VC 的投資撐著的
- **你的薪水**：公司拿到的投資，一部分會用來付薪水和擴大團隊
- **你的股票期權**：VC 投資時的 [估值](valuation.md) 會影響你的股票值多少
- **公司方向**：VC 可能會影響公司的策略（因為他們有董事席位）

一句話總結：**VC 是拿錢換股份的人，他們賭你會成功。**
