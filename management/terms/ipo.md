# IPO（首次公開發行） — 公司第一次把股票拿到股票市場上賣

> **白話說：** 就像**自家釀的酒第一次上架到全聯賣**——以前只有親朋好友（早期投資人）喝得到，現在任何人都可以去全聯（股票市場）買到。IPO 就是公司「從私人變公開」的那一刻。

---

## 它到底是什麼？

IPO 是 **Initial Public Offering（首次公開發行）** 的縮寫。

簡單來說：**一間公司第一次把自己的股票放到股票市場上，讓一般人都能買賣。** 在 IPO 之前，公司的股票只有創辦人、員工、和早期投資人（[VC](vc.md)）持有，一般人買不到。

```
公司上市前 vs 上市後：

IPO 前（私人公司）：
├─ 股票只有內部人持有
├─ 要買股票得私下談（很難）
├─ 公司不需要公開財務資訊
└─ 像是私人廚房，只有受邀的人能吃

IPO 後（上市公司）：
├─ 任何人都可以在股票市場買賣
├─ 股價由市場供需決定
├─ 公司必須定期公開財報
└─ 像是開了一間餐廳，誰都能來吃
```

公司為什麼要 IPO？

1. **募集大量資金**：上市可以一次賣出大量股票，拿到大筆錢來擴張
2. **早期投資人出場**：VC 和員工持有的股票終於可以賣掉，把錢拿回來
3. **提高知名度**：上市公司的品牌信任度更高
4. **用股票吸引人才**：上市後的股票比較有價值，更容易招人

但 IPO 也有代價：
- 要定期公開財報，接受公眾和媒體監督
- 要遵守很多法規
- 股價會受市場情緒影響，壓力很大

## 生活比喻 / 實際例子

想像你從小攤做到有三間分店的滷味老闆：

```
階段 1：路邊攤
├─ 只有你自己出資
├─ 所有權 100% 是你的
└─ 像是私人小公司

階段 2：找投資人
├─ 某個朋友投了 100 萬（VC 投資）
├─ 他拿到 20% 股份
└─ 但股票還是不能在市場上買賣

階段 3：IPO 上市
├─ 你把一部分股票拿到股票市場上賣
├─ 任何人都可以買你的「滷味公司」股票
├─ 你一口氣募到 5,000 萬
├─ 當初投 100 萬的朋友，他的股份現在值 1,000 萬
└─ 你也從「滷味攤老闆」變成「上市公司董事長」
```

台灣近年知名的 IPO：
- **路易莎咖啡**：從小咖啡店做到上市
- **瓦城泰統**：泰式餐廳集團上市
- **91APP**：電商平台上市

**造句**：
- 「聽說那間公司明年要 **IPO** 了，他們的員工股票一定很值錢」
- 「**IPO** 第一天股價就漲了 50%，認購的人都賺翻了」
- 「公司還沒 **IPO** 之前，你手上的股票是賣不掉的」
- 「這間公司準備了三年才終於 **IPO** 成功」

## 為什麼要知道這個詞？

- IPO 是財經新聞的常客——每次有大公司上市都是大新聞
- 如果你在新創公司工作且持有股票期權，IPO 是你的股票能變現金的重要時刻
- [VC](vc.md) 投資的終極目標之一就是讓公司 IPO，這是他們賺錢的方式
- 了解 IPO，你才能理解為什麼「上市公司」和「未上市公司」的規則差這麼多
