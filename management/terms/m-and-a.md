# M&A（併購） — 一家公司買下或合併另一家公司

> **白話說：** 就像**大餐廳把隔壁小吃店買下來，變成自己的分店**——或者兩間差不多大的餐廳合併，變成一間更大的餐廳。M&A 就是公司之間的「買賣」或「結婚」。

---

## 它到底是什麼？

M&A 是 **Mergers and Acquisitions（合併與收購）** 的縮寫。

這兩個字其實是兩件不同的事：

```
Merger（合併）：
├─ 兩間公司合在一起，變成一間新公司
├─ 通常是兩間差不多大的公司
├─ 像兩個人結婚，變成一個家庭
└─ 例如：A 公司 + B 公司 → C 公司（新的）

Acquisition（收購）：
├─ 一間大公司把另一間公司買下來
├─ 被買的公司變成買家的一部分
├─ 像大魚吃小魚
└─ 例如：A 公司買了 B 公司 → B 變成 A 的子公司
```

公司為什麼要做 M&A？

| 目的 | 舉例 |
|------|------|
| 消滅競爭對手 | Facebook 買下 Instagram（怕被搶走用戶） |
| 取得技術或人才 | Google 買 AI 新創公司（要他們的技術和工程師） |
| 擴大市場 | 某銀行併購另一家銀行（拿到更多客戶和分行） |
| 進入新領域 | 鴻海買夏普（從代工跨入品牌） |
| 垂直整合 | 食品公司買下農場（自己控制原料來源） |

M&A 的金額通常非常大，動輒幾十億、幾百億。這也是 [VC](vc.md) 投資的另一個出場方式——如果你投資的新創被大公司收購了，你的股份就能賣掉賺錢。

## 生活比喻 / 實際例子

想像你們社區有兩間早餐店：

- **收購（Acquisition）**：隔壁的「阿明早餐店」生意不好，你把它買下來。招牌換成你的、菜單換成你的、員工可能留也可能走。阿明拿到一筆錢，從此不用再操心
- **合併（Merger）**：你跟隔壁的「阿華早餐店」決定合作，兩間店合成一間「明華早餐店」，菜單整合、資源共享、一起打拼

但現實中，很多號稱「合併」的其實是「收購」——就像結婚說好了要平等，結果一方說了算。

台灣經典的 M&A 案例：
- **鴻海收購夏普**：花了幾千億台幣買下日本夏普
- **日月光收購矽品**：台灣半導體封裝兩大巨頭合併
- **富邦併購日盛金**：金融業整併

**造句**：
- 「聽說 A 公司要 **併購** B 公司，股價馬上漲了」
- 「這個 **M&A** 案花了三年才通過公平會審查」
- 「被 **收購** 之後，很多老員工都離職了」
- 「**M&A** 之後的整合才是最困難的——兩間公司的文化要融合」

## 為什麼要知道這個詞？

- M&A 是財經新聞的大事件——每次有大型併購案，相關公司的股價都會劇烈波動
- 如果你工作的公司被收購了，你的工作內容、主管、甚至工作本身都可能改變
- 了解 M&A，你才能看懂為什麼大公司要一直買其他公司（不是因為有錢任性，是有策略的）
- 跟 [IPO](ipo.md)、[VC](vc.md)、[估值](valuation.md) 都有密切關係
