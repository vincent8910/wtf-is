# 商業模式（Business Model） — 你的公司靠什麼方式賺錢

> **白話說：** 商業模式就像**夜市攤販的經營方式**——你賣什麼？賣給誰？怎麼讓客人願意掏錢？成本多少？最後能賺多少？這一整套加起來就是你的商業模式。

---

## 它到底是什麼？

**商業模式**就是回答這個問題：**「你的公司到底是怎麼賺錢的？」**

聽起來很簡單，但背後其實包含很多層面：

1. **你提供什麼價值？**（你的產品或服務解決了什麼問題）
2. **賣給誰？**（你的目標客戶是 [B2B 還是 B2C](b2b-b2c.md)）
3. **怎麼收錢？**（一次買斷？月費訂閱？抽成？免費但靠廣告？）
4. **成本多少？**（[成本結構](cost-structure.md)：做這件事要花多少錢）
5. **能持續嗎？**（這個賺錢方式長期可行嗎）

## 生活比喻 / 實際例子

用台灣常見的例子來看不同的商業模式：

| 商業模式 | 台灣例子 | 怎麼賺錢 |
|---------|---------|---------|
| 直接銷售 | 鼎泰豐 | 你吃飯、你付錢 |
| 訂閱制 | Netflix、Spotify | 每月付固定費用 |
| 平台抽成 | Uber Eats、蝦皮 | 免費讓你用，但跟商家抽成 |
| 免費增值 | LINE | 免費通訊，但賣貼圖和廣告 |
| 廣告模式 | Google、Facebook | 免費讓你用，賣廣告位給企業 |
| 授權金 | 台積電的 IP | 讓別人用你的技術，收授權費 |

```
便利商店的商業模式：
├─ 賣什麼？日常用品、鮮食、咖啡
├─ 賣給誰？附近的居民和上班族
├─ 怎麼賺？商品加價賣（賺差價）
├─ 成本？店租、人力、進貨成本
└─ 關鍵？開在人多的地方、24 小時營業
```

在公司裡你會聽到：

- 「我們的**商業模式**是訂閱制」→ 客戶每月付費使用
- 「這個**商業模式**可行嗎？」→ 這樣賺錢的方式行得通嗎？
- 「要調整**商業模式**了」→ 公司賺錢的方式要改變

## 為什麼要知道這個詞？

理解商業模式，你就能看懂一家公司的「遊戲規則」：

- **面試時**：知道這家公司靠什麼賺錢，回答問題才會到位
- **日常工作**：理解為什麼某些決策是這樣做的（例如免費增值模式的公司，免費功能越好用越重要）
- **跨部門溝通**：知道商業模式，才能理解業務、行銷、產品各自在乎什麼
- **投資人場合**：[VC](vc.md) 最愛問「你的商業模式是什麼」
