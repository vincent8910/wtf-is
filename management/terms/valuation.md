# 估值（Valuation） — 你的公司值多少錢

> **白話說：** 估值就像**二手車鑑價**——你的車（公司）到底值多少錢？不是你說了算，也不完全是買家說了算，而是看車況（營收）、里程數（成長性）、品牌（市場地位）等各種因素來判斷。

---

## 它到底是什麼？

**Valuation**（估值）就是**用一個數字來表示你的公司值多少錢**。

但跟二手車不同的是，公司的估值不是一個絕對的數字，而是雙方**談出來的**。尤其是新創公司還沒有穩定獲利的時候，估值更像是一種「對未來的期待」。

有兩個重要的概念：

```
Pre-money Valuation（投前估值）：
投資人投錢進來「之前」，公司值多少

Post-money Valuation（投後估值）：
投資人投錢進來「之後」，公司值多少

關係：投後估值 = 投前估值 + 投資金額
```

舉個例子：

```
你的公司：
├─ 投前估值：8,000 萬
├─ 投資人投入：2,000 萬
├─ 投後估值：8,000 萬 + 2,000 萬 = 1 億
└─ 投資人持股比例：2,000 萬 ÷ 1 億 = 20%
```

## 生活比喻 / 實際例子

想像你開了一間手搖飲料店，有人想要入股：

```
你的飲料店估值取決於：
├─ 現在每月營收多少？（現況）
├─ 客人越來越多還是越來越少？（成長趨勢）
├─ 附近有沒有競爭者？（市場環境）
├─ 你的獨家配方值錢嗎？（獨特優勢）
├─ 未來能不能開分店？（成長潛力）
└─ 類似的飲料店之前賣多少？（市場行情）
```

新創公司估值更「玄」，因為很多都還沒賺錢：

| 估值依據 | 白話說 |
|---------|--------|
| 市場規模 | 這個市場有多大的餅可以吃 |
| 成長速度 | 用戶或營收長得有多快 |
| 團隊 | 創辦人和團隊有多強 |
| 競爭優勢 | 有什麼別人學不來的 |
| 可比公司 | 類似的公司估值多少 |

在公司裡你會聽到：

- 「我們的**估值**是 3 億」→ 公司目前被認為值 3 億
- 「這輪**估值**拉太高了」→ 這次融資的估值定太高，可能有風險
- 「**估值**翻倍了」→ 公司的價值是上一輪的兩倍

## 為什麼要知道這個詞？

估值影響的事情很多：

- **創辦人**：估值決定你融資時要讓出多少 [股權](dilution.md)
- **員工**：如果你有員工股票期權，估值越高你的股票越值錢
- **投資人**：[VC](vc.md) 用估值來決定投資條件，寫在 [Term Sheet](term-sheet.md) 上
- **看新聞**：「獨角獸」就是估值超過 10 億美元的新創公司

一句話總結：**估值不是科學，是藝術——它反映的是大家對公司未來的信心。**
