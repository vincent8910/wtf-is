# 現金流（Cash Flow） — 公司帳上實際有多少現金在流進流出

> **白話說：** 現金流就像你的**銀行帳戶餘額變化**——不管你這個月理論上賺了多少，重要的是帳戶裡「實際有多少錢可以用」。錢不夠付下個月房租，理論上賺再多都沒用。

---

## 它到底是什麼？

**現金流**指的是**實際上有多少現金流進公司、流出公司**。

聽起來跟 [P&L（損益表）](pnl.md) 很像，但有一個關鍵差異：

```
P&L（損益表）：記錄「應該」賺了多少
現金流：記錄「實際上」手上有多少現金

舉例：
你今天賣了一批貨，客戶下個月才付款
→ P&L 上：這個月營收 +100 萬 ✅
→ 現金流：這個月進帳 0 元 ❌（錢還沒到）
```

這就是為什麼有些公司「帳面上賺錢，但實際上倒閉了」——因為現金流斷了，付不出薪水和帳單。

## 生活比喻 / 實際例子

想像你是一個接案的自由工作者：

```
本月狀況：
├─ 你完成了 3 個案子，總報價 15 萬
├─ 但客戶說「30 天後才匯款」
├─ 現在帳戶裡只剩 2 萬
├─ 這個月的房租 + 生活費要 3 萬
└─ 結果：案子接很多、很忙，但錢不夠用 💀
```

這就是「收入帳面很好，但現金流有問題」的典型狀況。

在商業世界裡，這種事天天發生：

| 狀況 | P&L 看起來 | 現金流實際上 |
|------|-----------|-------------|
| 客戶下單但還沒付款 | 有營收 | 沒收到現金 |
| 預付了半年的租金 | 每月分攤一點成本 | 這個月一次付出去一大筆 |
| 客戶先付年費 | 分 12 個月認列營收 | 這個月收到一大筆現金 |

在公司裡你會聽到：

- 「**現金流**還撐得住嗎？」→ 帳上的錢夠不夠公司運作
- 「**現金流**是正的」→ 流進來的錢比流出去的多，是好事
- 「注意**現金流**，別燒太快」→ 小心不要花錢的速度比賺錢快
- 「**Cash is King**」→ 現金為王，有現金才能活下去

## 為什麼要知道這個詞？

現金流是公司能不能「活著」的最直接指標：

- **新創公司**：最怕的不是不賺錢，是**現金燒完了**。[VC](vc.md) 投的錢就是在補現金流
- **營運主管**：每天都要關注現金流，確保薪水、供應商、租金付得出來
- **PM**：你的專案如果要大量前期投資，要考慮對公司現金流的影響
- **全公司**：當老闆說「要注意現金流」，通常代表公司的錢比較緊

一句話總結：**P&L 告訴你公司賺不賺錢，現金流告訴你公司活不活得下去。**
