# P&L（損益表） — 一段時間內賺了多少、花了多少、最後賺還是賠

> **白話說：** P&L 就像你的**記帳 App 月報表**——這個月薪水進來多少、房租水電吃飯花了多少、最後剩多少或透支多少，一目了然。

---

## 它到底是什麼？

**P&L** 是 **Profit and Loss Statement** 的縮寫，中文叫「損益表」或「損益報表」。

白話說就是一張**公司的成績單**，告訴你：

```
營業額（Revenue） — 總共賺了多少錢
  - 成本（Cost）   — 做產品花了多少錢
  = 毛利（Gross Profit） — 扣掉成本剩多少
  - 營業費用（Operating Expenses） — 租金、薪水、行銷等
  = 營業利益（Operating Income） — 本業到底賺不賺
  - 稅金和其他
  = 淨利（Net Income） — 最後真正賺到口袋裡的
```

P&L 通常會看某一段時間的：月報、季報、年報。

## 生活比喻 / 實際例子

假設你在夜市擺攤賣雞排，一個月的 P&L 長這樣：

| 項目 | 金額 |
|------|------|
| 🟢 這個月雞排總共賣了（[營業額](revenue.md)） | +150,000 |
| 🔴 雞肉、油、粉的成本 | -60,000 |
| = [毛利](gross-margin.md)（賣價扣成本） | 90,000 |
| 🔴 攤位租金 | -15,000 |
| 🔴 請工讀生的薪水 | -30,000 |
| 🔴 瓦斯、電費 | -5,000 |
| = **淨利（最後賺到的）** | **40,000** |

看到淨利是正的，代表這個月有賺錢。如果是負的，就代表在「燒錢」。

在公司裡你會聽到：

- 「看一下這個月的 **P&L**」→ 看看這個月賺了還是賠了
- 「**P&L** 上面成本太高了」→ 花太多錢，要想辦法降低
- 「這個部門的 **P&L** 是負的」→ 這個部門目前是賠錢的

## 為什麼要知道這個詞？

P&L 是財務報表裡最基本也最重要的一張：

- **主管**：每個月都會看 P&L 來決定資源分配
- **PM**：如果你負責的產品有獨立的 P&L，你要知道它賺不賺錢
- **全公司**：公司賺不賺錢，P&L 上寫得清清楚楚
- **投資人**：[VC](vc.md) 看 P&L 來判斷公司的財務健康狀況

跟 [現金流](cash-flow.md) 的差別：P&L 告訴你「賺了還是賠了」，現金流告訴你「帳上還有沒有錢」。有時候 P&L 顯示賺錢，但現金流卻是負的（因為客戶還沒付款），所以兩個都要看。
