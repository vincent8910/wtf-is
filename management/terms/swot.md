# SWOT — 分析優勢、劣勢、機會、威脅的四格框架

> **白話說：** SWOT 就像**看醫生做全身健檢**——先搞清楚你身體哪裡好（優勢）、哪裡弱（劣勢）、有什麼保健機會（機會）、有什麼潛在風險（威脅），然後才能對症下藥。

---

## 它到底是什麼？

**SWOT** 是四個英文字的縮寫，是一種用來分析局勢的簡單框架：

| | 正面的 👍 | 負面的 👎 |
|---|----------|----------|
| **內部**（自己能控制的） | **S**trengths 優勢 | **W**eaknesses 劣勢 |
| **外部**（自己不能控制的） | **O**pportunities 機會 | **T**hreats 威脅 |

用一張經典的四格表來看：

```
┌─────────────────┬─────────────────┐
│ S 優勢           │ W 劣勢           │
│ 我們哪裡比別人強？│ 我們哪裡比別人弱？│
│                  │                  │
├─────────────────┼─────────────────┤
│ O 機會           │ T 威脅           │
│ 外面有什麼好機會？│ 外面有什麼危險？  │
│                  │                  │
└─────────────────┴─────────────────┘
```

## 生活比喻 / 實際例子

假設你在考慮要不要開一間早午餐店：

```
┌──────────────────────┬──────────────────────┐
│ S 優勢                │ W 劣勢                │
│ • 你很會做蛋料理       │ • 沒有開店經驗         │
│ • 有朋友是設計師能幫設計│ • 資金不多             │
│ • 住在年輕人多的區域    │ • 不懂行銷            │
├──────────────────────┼──────────────────────┤
│ O 機會                │ T 威脅                │
│ • 附近沒有早午餐店      │ • 附近即將開一間連鎖早餐│
│ • 健康飲食趨勢正夯      │ • 房租可能漲價         │
│ • 網紅帶貨效應強        │ • 食材成本一直上升      │
└──────────────────────┴──────────────────────┘
```

看完這張表你就有方向了：

- **用優勢抓機會**：用你的蛋料理特色搭上健康飲食趨勢
- **補劣勢防威脅**：先學基本行銷，並存夠半年的租金才開店

在公司裡的應用：

| 場景 | 用 SWOT 分析什麼 |
|------|-----------------|
| 新產品上市 | 產品的優劣勢、市場機會和競爭威脅 |
| 競爭分析 | 我們 vs 競爭對手的相對位置 |
| 年度策略會 | 公司整體的內外部環境 |
| 個人職涯 | 你自己的能力和市場機會 |

在公司裡你會聽到：

- 「做一個 **SWOT** 分析」→ 用四格表分析一下優劣勢和機會威脅
- 「**SWOT** 看起來，我們的威脅比較大」→ 外部環境的危險因素比較多
- 「從 **SWOT** 來看，這是一個好機會」→ 分析顯示這個機會值得把握

## 為什麼要知道這個詞？

SWOT 是最常見的商業分析工具之一，幾乎每個管理課程的第一堂課都會教：

- **簡單好用**：不需要專業知識，任何人都能上手
- **應用範圍廣**：分析公司、產品、競爭、甚至個人職涯都適用
- **開會常用**：主管很愛在策略會議上說「來做個 SWOT」
- **面試會考**：「請做一下我們公司的 SWOT 分析」是很常見的面試題

不過要注意：SWOT 是一個**起點**，不是終點。分析完之後，重要的是「接下來要做什麼」。
