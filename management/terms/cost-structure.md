# 成本結構（Cost Structure） — 公司的錢都花在哪些地方

> **白話說：** 成本結構就像你的**每月支出明細**——房租佔多少、吃飯佔多少、交通佔多少。知道錢花在哪裡，才知道哪裡可以省、哪裡不能省。

---

## 它到底是什麼？

**成本結構**就是把公司所有的花費，分門別類列出來，看看**錢到底都花去哪了**。

一般公司的成本主要分兩大類：

```
固定成本（不管賣多少都要付的）：
├─ 辦公室租金
├─ 員工薪水
├─ 軟體訂閱費
└─ 設備折舊

變動成本（賣越多花越多的）：
├─ 原料 / 進貨成本
├─ 物流 / 運費
├─ 交易手續費
└─ 業務獎金
```

不同類型的公司，成本結構天差地遠：

| 公司類型 | 最大成本 | 為什麼 |
|---------|---------|--------|
| 軟體公司 | 人事成本（工程師薪水） | 程式寫好就能複製，但人才很貴 |
| 餐廳 | 食材 + 店租 + 人事 | 三大成本幾乎不能省 |
| 電商 | 進貨 + 物流 + 行銷 | 要買貨、要出貨、要打廣告 |
| 製造業 | 原料 + 設備 + 人工 | 生產線的投入很大 |

## 生活比喻 / 實際例子

想像你每個月的生活成本結構：

```
月收入：50,000
├─ 🏠 房租：15,000（30%）← 固定成本，每月都要付
├─ 🍱 伙食：10,000（20%）← 半固定，可以稍微調整
├─ 🚗 交通：3,000（6%）← 變動成本
├─ 📱 手機網路：1,000（2%）← 固定成本
├─ 🎮 娛樂：5,000（10%）← 變動成本，可以砍
└─ 💰 剩餘：16,000（32%）
```

如果你想多存錢，就會分析成本結構：「房租佔最多，但短期不能搬。娛樂費可以先砍一點。」公司的思路也是一樣的。

在公司裡你會聽到：

- 「我們的**成本結構**太重了」→ 花費太高，利潤空間被壓縮
- 「要優化**成本結構**」→ 要重新調整花費比例，讓公司更賺錢
- 「人事成本佔**成本結構**的 60%」→ 大部分的錢都花在人的薪水上

## 為什麼要知道這個詞？

成本結構決定了公司的「體質」：

- **賺錢能力**：成本結構決定了 [毛利率](gross-margin.md) 和最終利潤
- **抗風險能力**：固定成本高的公司，營收一下滑就很危險（因為開銷省不掉）
- **定價策略**：你要知道成本結構，才能合理定價
- **[商業模式](business-model.md) 設計**：好的商業模式會設計出健康的成本結構

舉例：為什麼軟體公司估值通常比較高？因為成本結構好——程式寫一次可以賣給無數人，邊際成本幾乎為零。
