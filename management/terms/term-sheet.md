# Term Sheet — 投資人給你的「報價單」，列出投資條件

> **白話說：** Term Sheet 就像**買房子的議價單**——買家列出「我願意出多少錢、什麼條件、什麼時候交屋」，你看了可以接受、也可以再談。但在正式簽約前都還能反悔。

---

## 它到底是什麼？

**Term Sheet**（條款清單）是投資人在決定要投資你之後，列出來的**一份投資條件摘要**。

上面會寫：

```
Term Sheet 通常包含：
├─ 💰 投多少錢？（投資金額）
├─ 📊 公司值多少？（投前/投後估值）→ 參考 [估值](valuation.md)
├─ 📉 投資人拿多少股？（持股比例）→ 參考 [股權稀釋](dilution.md)
├─ 🪑 要不要董事席位？（治理權）
├─ 🛡️ 有什麼保護條款？（反稀釋、優先清算等）
└─ 📅 什麼時候完成？（時程）
```

Term Sheet 通常**不具法律約束力**（除了保密和排他條款），但它是正式投資合約的基礎。

## 生活比喻 / 實際例子

想像你要賣一間店面：

```
買家的 Term Sheet：
├─ 「我願意出 500 萬買你的店」
├─ 「但我要確認你沒有欠債」
├─ 「簽約前你不能再跟別人談」
├─ 「一個月內完成所有手續」
└─ 你可以接受、議價、或拒絕
```

在新創公司的融資過程中：

```
融資流程：
1. 你跟 VC 簡報公司的狀況
2. VC 有興趣 → 做盡職調查（DD）
3. VC 決定要投 → 發 Term Sheet 📄
4. 雙方針對 Term Sheet 談判條件
5. 談好了 → 簽正式投資合約
6. 錢入帳 ✅
```

常見的 Term Sheet 重要條款：

| 條款 | 白話說 |
|------|--------|
| Pre-money Valuation | 投錢進來之前，你的公司值多少 |
| Investment Amount | 投資人要投多少錢 |
| Liquidation Preference | 公司賣掉或倒了，投資人先拿回多少 |
| Anti-dilution | 如果下一輪估值變低，保護投資人的條款 |
| Board Seat | 投資人要不要在董事會有一個位子 |
| No-shop Clause | 簽了之後，一段時間內你不能再找其他投資人 |

在公司裡你會聽到：

- 「收到 **Term Sheet** 了！」→ 投資人正式表達投資意願和條件了
- 「**Term Sheet** 的條件要再談」→ 投資條件還有討價還價的空間
- 「拿到 **Term Sheet** 不代表錢就到了」→ 從 Term Sheet 到錢入帳還有很多步

## 為什麼要知道這個詞？

如果你在新創公司或跟投資有關的環境工作：

- **創辦人**：收到 Term Sheet 是融資的重要里程碑
- **早期員工**：Term Sheet 上的 [估值](valuation.md) 會影響你的員工股票期權值多少
- **財務/法務**：你需要理解 Term Sheet 上的每個條款
- **所有人**：知道 Term Sheet 代表什麼，你就能理解公司融資進度

一句話總結：**Term Sheet 不是合約，但它是投資談判最關鍵的文件。**
