# Dashboard（儀表板） — 用圖表一眼看到公司關鍵數字的頁面

> **白話說：** Dashboard 就像**汽車儀表板**——時速、油量、水溫都在同一個面板上，你一眼就知道車子的狀態，不用打開引擎蓋檢查。

---

## 它到底是什麼？

**Dashboard** 就是把公司重要的數字，用圖表的方式呈現在一個畫面上，讓你**一眼就看到全貌**。

不用翻一堆 Excel、不用等人寄報表、不用開會才知道數字。打開 Dashboard，所有關鍵資訊都在那裡。

一個典型的 Dashboard 可能長這樣：

```
┌─────────────────────────────────────────┐
│  📊 本月營收          📈 用戶成長        │
│  $1,200 萬           +15% ↑            │
│  ████████░░  (80%)   ▁▃▅▆█            │
│                                         │
│  👥 活躍用戶          ⭐ 滿意度          │
│  52,000 人           4.3 / 5.0         │
│  ▁▂▃▅▆▇█▇▆          ██████████░       │
└─────────────────────────────────────────┘
```

常見的 Dashboard 工具有 Google Data Studio、Tableau、Power BI，或是公司自己開發的後台。

## 生活比喻 / 實際例子

你的手機就是一個 Dashboard：

- 鎖定畫面上一眼就能看到：時間、日期、電量、有幾則未讀訊息
- 不用打開每個 App 去看，重要資訊都在一個畫面上

又像是夜市攤販老闆：

- 如果有一個螢幕能同時顯示「今天賣了幾份」「剩多少材料」「哪道菜最熱銷」，那就是他的 Dashboard

在公司裡你會聽到：

- 「拉一下 **Dashboard** 看一下數字」→ 打開數據面板看最新狀況
- 「老闆要看 **Dashboard**」→ 主管想看關鍵指標的即時報表
- 「這個 **Dashboard** 上要加毛利率」→ 在報表頁面上多加一個 [毛利率](gross-margin.md) 的圖表

## 為什麼要知道這個詞？

現在幾乎每家公司都有某種形式的 Dashboard：

- **PM** 會看產品數據 Dashboard（日活用戶、留存率等）
- **業務** 會看銷售 Dashboard（成交數、業績達成率等）
- **老闆** 會看公司營運 Dashboard（[營業額](revenue.md)、[現金流](cash-flow.md)、[P&L](pnl.md) 等）

Dashboard 上面顯示的數字，很多就是你的 [KPI](kpi.md)。所以看懂 Dashboard，就等於看懂公司在意什麼。
