# GTM（Go-to-Market） — 產品做好了，怎麼推出去賣的策略

> **白話說：** GTM 就像**新餐廳的開幕計畫**——菜單設計好了、廚師請好了，但你還要決定：怎麼讓大家知道你開了？開幕要辦什麼活動？定價多少？先打哪個客群？這整套計畫就是 GTM。

---

## 它到底是什麼？

**GTM** 是 **Go-to-Market** 的縮寫，中文叫「上市策略」或「市場進入策略」。

白話說就是：**產品做好了，然後呢？怎麼賣出去？**

一個 GTM 策略通常要回答這些問題：

```
GTM 計畫：
├─ 賣給誰？→ 目標客群是哪些人
├─ 為什麼買？→ 我們解決了什麼痛點
├─ 怎麼定價？→ 收多少錢、怎麼收
├─ 在哪裡賣？→ 通路是什麼（網路？門市？經銷商？）
├─ 怎麼推廣？→ 行銷策略是什麼
└─ 什麼時候？→ 上線時程和里程碑
```

GTM 不只是「打廣告」這麼簡單，而是一整套從產品到客戶手上的完整規劃。

## 生活比喻 / 實際例子

假設你做了一款很好喝的手工果醬，GTM 策略可能是：

```
路線 A（B2C 直接賣給消費者）：
├─ 先在 Instagram 建立品牌
├─ 透過蝦皮、Pinkoi 開店
├─ 找美食部落客業配
└─ 定價一罐 280 元

路線 B（B2B 先跟企業合作）：
├─ 先找早午餐店和咖啡廳試用
├─ 談批發價，一次訂 50 罐以上打折
├─ 透過餐廳的客人認識品牌
└─ 再回頭做消費者市場
```

兩條路都能賣果醬，但策略完全不同。選哪條路，就是 GTM 策略在決定的事。

在公司裡你會聽到：

- 「新產品的 **GTM** 策略定了嗎？」→ 新產品要怎麼推出去，計畫好了嗎？
- 「**GTM** 團隊開個會對齊一下」→ 負責產品上市的人一起討論
- 「這次 **GTM** 先打企業市場」→ 這次新產品先賣給企業客戶

## 為什麼要知道這個詞？

不管你在公司的哪個位置，GTM 都跟你有關：

- **產品經理**：你做完產品，還要規劃 GTM（或跟行銷一起規劃）
- **行銷**：GTM 裡的推廣策略就是你要執行的
- **業務**：GTM 決定了你要先打哪個客群、用什麼話術
- **工程師**：GTM 的時程會影響你的開發時程
- **新創團隊**：[VC](vc.md) 會問你的 GTM 策略，來判斷你能不能把產品賣出去

簡單說：**有了好產品 + 好的 GTM = 才有可能成功。** 只有好產品但沒有 GTM，就像開了一家超好吃的餐廳，但藏在巷子裡沒人知道。
