# PMF（Product-Market Fit） — 你的產品真的是市場需要的

> **白話說：** PMF 就像**夜市擺攤找到對的位置**——你賣的臭豆腐剛好擺在人潮最多的地方，而且大家真的愛吃，每天排隊排到巷口。產品和市場完美對上了。

---

## 它到底是什麼？

**PMF** 是 **Product-Market Fit** 的縮寫，中文叫「產品市場適配」。

白話說就是：**你做的東西，市場真的買單。**

很多產品做出來之後，發現沒人要用。PMF 就是那個「啊，大家真的需要這個！」的時刻。

怎麼知道有沒有達到 PMF？看這些訊號：

| 有 PMF 的徵兆 | 沒有 PMF 的徵兆 |
|--------------|---------------|
| 用戶主動推薦給朋友 | 砸廣告才有人來，廣告停了人就走 |
| 產品被搶著用，供不應求 | 用了一次就不回來了 |
| 用戶會抱怨「為什麼不加這個功能」 | 用戶根本不在乎你更新了什麼 |
| 營收自然成長 | 怎麼推都推不動 |

## 生活比喻 / 實際例子

想像你在夜市賣東西：

```
❌ 沒有 PMF：
你在夜市賣高級法式甜點，一份 300 元
→ 路過的人看一眼就走了，偶爾才賣一份

✅ 有 PMF：
你改賣 50 元的雞排
→ 每天排隊排到馬路上，材料準備多少賣多少
```

不是雞排比較「好」，而是雞排跟夜市的客群**更對味**。這就是 PMF。

台灣的經典 PMF 案例：

- **LINE**：在台灣剛好遇上大家想從 MSN 換到手機通訊軟體的時期，加上免費貼圖，爆紅
- **Gogoro**：解決了台灣機車族對電動車「充電麻煩」的痛點，用換電站打中市場

在公司裡你會聽到：

- 「我們還沒找到 **PMF**」→ 產品還沒被市場接受，還在摸索
- 「先做 [MVP](mvp.md) 來驗證 **PMF**」→ 先做最小版本測試市場反應
- 「找到 **PMF** 之後再大規模投資」→ 確定市場買單了，再加碼投入

## 為什麼要知道這個詞？

PMF 是新創公司和產品團隊最常講的概念之一：

- **產品經理**：PMF 是產品開發的第一個里程碑
- **新創公司**：[VC](vc.md) 投資前會評估你有沒有達到 PMF
- **行銷**：有 PMF 的產品推起來事半功倍，沒有的話砸再多錢也沒用
- **全公司**：找到 PMF 之前，公司的重心是探索；找到之後，重心轉向成長

一句話總結：**沒有 PMF，其他都不重要。**
