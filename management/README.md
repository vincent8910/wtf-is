# WTF is...? 🏢 企業管理與營運篇

> **老闆又在講管理黑話了？這裡翻譯給你聽。**
>
> 每個詞都有白話解釋、生活化比喻、實際造句。點進去就懂，不用 Google 半天。

---

## 這是什麼？

這是一份給**沒有管理或商業背景的人**看的術語翻譯手冊。

如果你是工程師、設計師、剛入職場的新人，或任何聽到 KPI、OKR、P&L 就一頭霧水的人——這裡就是你的救星。

每個術語都有：
- **白話解釋**：不用任何管理背景就能懂
- **生活比喻**：用你熟悉的事物來對照
- **造句示範**：讓你知道這個詞在什麼情境下會出現

## 績效管理類

> 公司怎麼衡量做得好不好？怎麼設目標？怎麼持續改善？

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| KPI | 用數字衡量你做得好不好的指標 | 學校的成績單 | 「你這季的 KPI 達成率怎樣？」 | [→ 詳細](terms/kpi.md) |
| OKR | 先定大目標，再拆成可衡量的小結果 | 旅行計畫：目的地 + 準備清單 | 「這季的 OKR 大家對齊一下」 | [→ 詳細](terms/okr.md) |
| PDCA | 計畫→執行→檢查→改進，不斷循環 | 學騎腳踏車：試了摔了調整再試 | 「用 PDCA 的方式來跑這個專案」 | [→ 詳細](terms/pdca.md) |
| SOP | 一步一步寫好的標準作業流程 | 食譜：照著做就不會出錯 | 「這個流程有 SOP 嗎？」 | [→ 詳細](terms/sop.md) |
| Dashboard | 用圖表一眼看到關鍵數字的頁面 | 汽車儀表板：時速油量一目了然 | 「拉一下 Dashboard 看數字」 | [→ 詳細](terms/dashboard.md) |

## 比喻對照總表

> 每個管理術語，用一句生活比喻讓你秒懂。

### 績效管理類

| 術語 | 比喻 |
|------|------|
| KPI | 學校的**成績單**——老師定好考試科目和及格標準 |
| OKR | **旅行計畫**——先決定目的地，再列出準備清單 |
| PDCA | **學騎腳踏車**——試了、摔了、調整、再試，一直循環 |
| SOP | **食譜**——照著步驟做，誰來煮都差不多 |
| Dashboard | **汽車儀表板**——時速、油量、水溫一目了然 |

### 商業模式類

| 術語 | 比喻 |
|------|------|
| B2B / B2C | **批發商** vs **全聯**——客戶是企業還是消費者 |
| 商業模式 | **夜市攤販的經營方式**——賣什麼、賣誰、怎麼賺 |
| PMF | **夜市擺攤找到對的位置**——產品和市場完美對上 |
| GTM | **新餐廳的開幕計畫**——怎麼讓大家知道你開了 |
| MVP | **夜市先擺一天試賣**——最小成本測試市場反應 |

### 財務基礎類

| 術語 | 比喻 |
|------|------|
| P&L（損益表） | **記帳 App 月報表**——收入、支出、結餘一目了然 |
| 毛利率 | **一杯珍奶的賺頭比例**——賣 60 元成本 20 元，賺 67% |
| 營業額 | **收銀機裡的總金額**——總共賣了多少，還沒扣成本 |
| 成本結構 | **每月支出明細**——房租、伙食、交通各佔多少 |
| 現金流 | **銀行帳戶餘額變化**——不管帳面賺多少，重要的是實際有多少現金 |

### 組織管理類

| 術語 | 比喻 |
|------|------|
| BU（事業單位） | **百貨公司各樓層專櫃**——各自負責不同的業務 |
| Stakeholder | **辦婚禮要顧到的所有人**——新人、家長、賓客、攝影師⋯⋯ |
| 跨部門協作 | **拍一部電影**——導演、編劇、演員、攝影各有專業 |
| 組織架構 | **家族族譜**——誰是誰的上級，一看就懂 |

### 法務合約類

| 術語 | 比喻 |
|------|------|
| NDA（保密協議） | **「我跟你講秘密，不能說喔」**——白紙黑字的正式版 |
| MoU（合作備忘錄） | **訂婚**——表示有意結婚，但還沒正式登記 |
| Term Sheet | **買房子的議價單**——列出條件，但還不是正式合約 |

### 投資類

| 術語 | 比喻 |
|------|------|
| 估值 | **二手車鑑價**——看車況、里程、品牌來判斷值多少 |
| VC（創投） | **星探**——找有潛力的新人投資，賭他會大紅 |
| 股權稀釋 | **披薩多切幾塊**——你的塊數沒變，但比例縮水了 |
| SWOT | **全身健檢**——好的壞的、內部外部都查一遍 |

### 創業與資本類

| 術語 | 比喻 |
|------|------|
| ROI（投資報酬率） | 花 100 塊擺攤，賺了 150 回來，ROI 50% |
| Pitch / Pitch Deck | 選秀節目上台表演，短時間征服評審 |
| Burn Rate | 油箱的耗油速度——每月燒多少現金 |
| Runway | 飛機起飛前的跑道長度——錢還能撐多久 |
| IPO | 從私人小店變成連鎖品牌上架百貨公司 |
| M&A（併購） | 大魚吃小魚 |
