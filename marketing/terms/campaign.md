# Campaign（活動企劃） — 有目標、有期限的一整套行銷活動

> **白話說：** 就像**百貨公司的週年慶**——從廣告、折扣、來店禮到抽獎，每一件事都是事先規劃好的，有開始日期也有結束日期，目標就是要讓你來花錢。

---

## 它到底是什麼？

Campaign（行銷活動企劃）就是**一個有明確目標、有時間限制、有預算的完整行銷計畫**。

它不是單一的廣告，也不是一篇貼文，而是一整套策略。從「為什麼要做」→「做給誰看」→「透過什麼管道」→「用什麼內容」→「花多少錢」→「怎麼衡量成效」，全部都規劃好的一系列行動。

```
一個 Campaign 的組成：

📋 Campaign：雙 11 購物節活動
├─ 目標：營業額提升 50%
├─ 時間：11/1 ~ 11/11
├─ 預算：100 萬
├─ 受眾：25-35 歲女性（TA）
│
├─ 管道：
│  ├─ Facebook / IG 廣告
│  ├─ LINE 官方帳號推播
│  ├─ KOL 合作貼文
│  └─ Google 搜尋廣告（SEM）
│
├─ 內容：
│  ├─ 倒數預熱貼文
│  ├─ 限時優惠碼
│  ├─ 直播抽獎
│  └─ 會員獨享折扣
│
└─ 成效衡量：
   ├─ 營業額（最重要）
   ├─ ROAS（廣告花 1 元賺回多少）
   ├─ 新會員數
   └─ 社群互動數（Engagement）
```

大公司一年通常會有好幾個 Campaign：年初的新年活動、母親節檔期、暑期促銷、雙 11、聖誕節......每個 Campaign 都是一場精心策劃的「戰役」。

## 生活比喻 / 實際例子

想像你要辦一場同學會：

- 你不是只丟一則訊息到群組就結束
- 你要：選餐廳、訂日期、做一張海報、私訊那些很久沒聯絡的同學、提醒大家報名、最後統計人數
- 從頭到尾有一個明確的目標（讓大家來聚餐）、有時間限制（下個月的某天）、有一整套要做的事
- **這整件事就是一個 Campaign**

**造句**：
- 「這次母親節的 **Campaign** 預算是 50 萬，目標是會員成長 20%」
- 「**Campaign** 的素材都準備好了嗎？下週一要上線了」
- 「上一波 **Campaign** 的 [ROAS](roas.md) 表現不錯，這次可以加碼」
- 「我們需要一個全新的 **Campaign** 來推這個新品」

## 為什麼要知道這個詞？

- 只要在行銷部門或廣告代理商上班，Campaign 是你每天都會聽到的詞
- 它是行銷工作的基本單位——你的工作就是規劃和執行一個又一個 Campaign
- 理解 Campaign 的結構，才能看懂行銷預算怎麼分配、成效怎麼衡量
- 跟 [CPC](cpc.md)、[CTR](ctr.md)、[ROAS](roas.md) 這些指標都是用來評估 Campaign 表現的
