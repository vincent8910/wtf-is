# KOL / KOC — KOL 是大網紅帶貨，KOC 是素人真心推薦

> **白話說：** KOL 就像百貨公司請明星代言，排場大、價格高；KOC 就像你的朋友在 Line 群說「這個超好用欸」——沒有業配感，反而更讓人想買。

---

## 它到底是什麼？

**KOL（Key Opinion Leader）** 是「關鍵意見領袖」，也就是我們常說的「網紅」「部落客」「YouTuber」。他們有大量粉絲，品牌付錢請他們推薦產品。

**KOC（Key Opinion Consumer）** 是「關鍵意見消費者」，就是「有影響力的一般消費者」。他們粉絲可能不多（幾百到幾千人），但因為是「真的在用」的素人，推薦起來更有說服力。

兩者的差別：

| | KOL | KOC |
|---|---|---|
| 粉絲數 | 通常萬人以上 | 幾百到幾千人 |
| 費用 | 高（幾萬到幾十萬） | 低（免費試用或小額費用） |
| 內容風格 | 專業、有製作感 | 隨性、像朋友分享 |
| 信任度 | 大家知道是業配 | 感覺是真心推薦 |
| 適合目標 | 品牌曝光、快速擴散 | 口碑累積、促進購買 |

近年的趨勢是：消費者越來越不信「明顯的業配」，反而更相信素人的真實體驗，所以 KOC 的行銷策略越來越受重視。

## 生活比喻 / 實際例子

想像你想買一台新的空氣清淨機：

- **KOL 路線**：你看到某個科技 YouTuber 拍了一支 10 分鐘的開箱影片，畫面精美、數據詳細，但影片開頭寫了「本片與品牌合作」——你知道他收了錢
- **KOC 路線**：你在 PTT 或 Dcard 看到一個普通人寫了使用心得「用了三個月真心覺得不錯，分享一下」——他沒收錢，就是自己覺得好

兩種你都會參考，但那個「素人真心推薦」可能更讓你動心。

實際例子：
- 行銷說：「這次新品上市先找 3 個 **KOL** 做開箱，再發 50 份試用品給 **KOC** 做口碑」
- 老闆問：「網紅合作要多少錢？」行銷：「大的 **KOL** 一支影片 10 萬，**KOC** 的話送產品就好」
- 行銷報告寫：「**KOC** 帶來的[轉換率](conversion-rate.md)比 **KOL** 高 2 倍，建議擴大 KOC 合作規模」

## 為什麼要知道這個詞？

- 網紅行銷是現在品牌推廣的主流方式之一，KOL 和 KOC 是最常聽到的分類
- 不同預算有不同做法：有錢找 KOL 衝曝光，預算小找 KOC 做口碑
- 跟 [UGC](ugc.md)（使用者原創內容）密切相關——KOC 產出的內容很多就是 UGC
- [互動率（Engagement）](engagement.md)通常 KOC 比 KOL 高，因為粉絲黏性更強
