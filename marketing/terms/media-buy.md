# Media Buy（媒體採購） — 花錢買廣告版位的過程

> **白話說：** 就像**在報紙上買一格廣告欄位，或在公車車身貼廣告**——你要決定在哪裡買、買多大、買多久、花多少錢。Media Buy 就是把廣告「放到對的地方」的工作。

---

## 它到底是什麼？

Media Buy（媒體採購）就是**購買廣告版位或廣告時段的整個過程**，包含選擇在哪個媒體上投放廣告、談價錢、買版位、追蹤成效。

簡單來說就是：你的廣告做好了，但要放在哪裡讓人看到？這個「找地方放廣告」的工作就是 Media Buy。

```
Media Buy 的工作流程：

1. 了解目標（TA 是誰？要達成什麼？）
2. 選擇媒體：
   ├─ 數位媒體：Facebook、Google、YouTube、LINE
   ├─ 傳統媒體：電視、報紙、雜誌、廣播
   └─ 戶外媒體：公車廣告、捷運燈箱、看板
3. 談價錢和版位
4. 下單購買
5. 廣告上線
6. 追蹤成效、優化調整
```

數位時代的 Media Buy 跟以前很不一樣：

| | 傳統 Media Buy | 數位 Media Buy |
|--|---|---|
| 買什麼 | 電視時段、報紙版面 | Facebook 版位、Google 關鍵字 |
| 怎麼計費 | 固定價格（買一個月多少錢） | 按效果計費（[CPC](cpc.md)、[CPM](cpm.md)、[CPA](cpa.md)） |
| 能調整嗎 | 買了就不能改 | 隨時可以調整出價、受眾、素材 |
| 能追蹤嗎 | 很難知道有多少人看 | 精確知道多少人看到、點擊、購買 |

現在大部分公司的 Media Buy 都以數位為主，因為可以精準追蹤每一塊錢花在哪裡、效果如何。

## 生活比喻 / 實際例子

想像你開了一間新的手搖飲料店，想讓附近的人知道：

- **在學校門口發傳單** = 買學校附近的「版位」（觸及學生族群）
- **在公車站牌貼海報** = 買戶外媒體（觸及通勤族群）
- **在 IG 投廣告** = 買數位媒體（觸及滑手機的年輕人）
- **在電台買廣告** = 買廣播媒體（觸及開車族群）

每個管道觸及的人不同、價錢不同、效果也不同。Media Buy 的專業就是幫你選出「CP 值最高的組合」。

**造句**：
- 「這波 [Campaign](campaign.md) 的 **Media Buy** 預算有 200 萬，怎麼分配？」
- 「**媒體採購**那邊說 YouTube 的版位價格漲了」
- 「我們的 **Media Buy** 策略是 70% 數位、30% 戶外」
- 「**媒體採購**要談好折扣，量大可以壓低 [CPM](cpm.md)」

## 為什麼要知道這個詞？

- 只要公司有投廣告，就一定牽涉到 Media Buy
- 了解媒體採購，你才知道為什麼到處都看得到某個品牌的廣告——因為他們買了很多版位
- 廣告代理商裡有一整個部門在做這件事，叫做「媒體部」或「媒體採購部」
- 跟 [CPM](cpm.md)、[CPC](cpc.md)、[Impression](impression.md)、[Reach](reach.md) 這些指標直接相關
