# Content Marketing（內容行銷） — 用有價值的內容吸引客人，而不是直接打廣告

> **白話說：** 就像**料理教室教你做菜，順便推薦他們家的鍋子**——你先給別人有用的東西（食譜、教學），他覺得你很專業、很值得信任，自然就會想買你的產品。

---

## 它到底是什麼？

Content Marketing（內容行銷）就是**透過產出有價值、有幫助的內容（文章、影片、圖文、Podcast），來吸引潛在客戶，建立信任感，最終讓他們願意購買你的產品或服務**。

跟傳統廣告最大的差別：

```
傳統廣告：
├─ 「快來買我們的產品！超好用！打五折！」
├─ 直接推銷，消費者看了會想跳過
└─ 效果短暫，不投錢就沒人看

內容行銷：
├─ 「教你 5 個讓皮膚變好的方法」（其中一個用到我們的產品）
├─ 先給有用的資訊，消費者自願來看
├─ 看完覺得你專業、值得信任 → 自然想買
└─ 內容會持續被搜尋到，長期有效
```

台灣常見的內容行銷形式：

| 形式 | 舉例 |
|------|------|
| 部落格文章 | IKEA 教你「小坪數收納術」→ 推薦他們的收納產品 |
| YouTube 影片 | 化妝品牌拍「日常妝容教學」→ 用的都是自家產品 |
| IG 圖文 | 健身品牌分享「每日訓練菜單」→ 推薦自家高蛋白 |
| Podcast | 理財平台聊「新手投資入門」→ 推廣自家 App |
| 電子報 | 訂閱制電商每週寄「生活風格提案」→ 附上商品連結 |

## 生活比喻 / 實際例子

想像你搬到一個新社區，附近有兩間水電行：

- **A 水電行**：只會發傳單寫「水電維修找我們，便宜又快」
- **B 水電行**：在 Facebook 上分享「冬天水管怎麼防凍」「馬桶不通自己先試這招」各種實用教學

你會信任誰？下次水管真的壞了，你會打給誰？

B 水電行做的就是內容行銷——他先用有價值的內容讓你覺得他很專業、很可靠，等你真的有需求的時候，自然第一個想到他。

**造句**：
- 「我們要靠**內容行銷**建立品牌的專業形象」
- 「這篇文章的**內容行銷**做得很好，自然帶到產品但不會讓人覺得在推銷」
- 「**內容行銷**的重點是先提供價值，不是一直叫人買東西」
- 「我們的**內容行銷**策略是每週出兩篇文章、一支影片」

## 為什麼要知道這個詞？

- 內容行銷是現在最主流的行銷策略之一，大品牌幾乎都在做
- 它跟 [SEO](sem.md) 密切相關——好的內容會被 Google 搜尋到，帶來免費的長期流量
- 了解內容行銷，你就知道為什麼品牌要經營部落格、YouTube、Podcast——不是為了好玩，是為了行銷
- 跟 [UGC](ugc.md)、[KOL](kol-koc.md) 合作也是內容行銷的一環
