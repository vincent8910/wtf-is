# WTF is...? 行銷企劃篇

> **行銷部又在講黑話了？這裡翻譯給你聽。**
>
> 每個詞都有白話解釋、生活化比喻、實際造句。點進去就懂，不用 Google 半天。

---

## 這是什麼？

這是一份給**沒有行銷背景的人**看的術語翻譯手冊。

如果你是工程師、設計師、剛入行的行銷新人，或任何聽到 CPC、ROAS、漏斗就一頭霧水的人——這裡就是你的救星。

每個術語都有：
- **白話解釋**：不用任何行銷背景就能懂
- **生活比喻**：用你熟悉的事物來對照
- **造句示範**：讓你知道這個詞在什麼情境下會出現

## 數位廣告類

> 只要公司有投廣告，這些詞就會天天出現在報表和會議裡。

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| CPC | 每次有人點廣告，你付多少錢 | 夜市發傳單，有人接你才付錢 | 廣告報表、投放會議 | [詳細](terms/cpc.md) |
| CPM | 廣告被看 1000 次要花多少錢 | 捷運站燈箱廣告，被路人看到就算一次 | 品牌廣告、曝光型活動 | [詳細](terms/cpm.md) |
| CPA | 有人做了你要的動作才付錢 | 房仲成交才收佣金 | 廣告報表、成效檢討 | [詳細](terms/cpa.md) |
| CTR（點擊率） | 看到的人有多少比例點進去 | Line 群裡丟連結，多少人點開 | 廣告成效、Email 行銷 | [詳細](terms/ctr.md) |
| ROAS | 每花 1 元廣告費，賺回多少 | 花 100 買彩券中 500，回報 5 倍 | 廣告預算會議、月報 | [詳細](terms/roas.md) |
| 曝光（Impression） | 廣告出現在螢幕上的次數 | 名片放櫃檯上被人看到幾次 | 所有廣告報表 | [詳細](terms/impression.md) |
| 觸及率（Reach） | 有多少不同的人看到廣告 | Line 群訊息有多少人「看到」 | 社群報表、廣告後台 | [詳細](terms/reach.md) |
| 再行銷（Retargeting） | 對看過但沒買的人再投廣告 | 試穿沒買，回家一直被那件衣服追 | 電商行銷、廣告策略 | [詳細](terms/retargeting.md) |
| 歸因模型（Attribution） | 追蹤客人看了哪個廣告才買的 | 球賽進球了，功勞算誰的？ | 進階廣告分析、GA | [詳細](terms/attribution.md) |

## 社群行銷類

> 經營 IG、Facebook、找網紅合作時，這些詞是基本功。

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| KOL / KOC | KOL 是大網紅帶貨，KOC 是素人真心推薦 | KOL 像明星代言，KOC 像朋友推薦 | 網紅合作、行銷企劃 | [詳細](terms/kol-koc.md) |
| UGC | 使用者自己產出的內容 | 你去餐廳自己拍照打卡就是 UGC | 社群企劃、內容策略 | [詳細](terms/ugc.md) |
| 互動率（Engagement） | 粉絲按讚、留言、分享的比例 | 上課舉手回答的學生佔多少 | 社群報表、KOL 評估 | [詳細](terms/engagement.md) |
| 粉絲經營（Community） | 長期經營跟粉絲的關係 | 巷口早餐店記住你的口味 | 社群策略、品牌會議 | [詳細](terms/community.md) |

## 企劃提案類

> 寫企劃、提案、跟老闆報告時，這些是基本術語。

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| TA（目標客群） | 你最想把東西賣給誰 | 夜市擺攤，你的招牌寫給誰看 | 幾乎所有行銷會議 | [詳細](terms/ta.md) |
| Persona（人物誌） | 用一個虛構角色代表目標客群 | 把「25-35歲女性」變成「小美，28歲行銷助理」 | 企劃提案、產品設計 | [詳細](terms/persona.md) |
| 品牌定位 | 在消費者心中你代表什麼 | 提到平價時尚就想到 UNIQLO | 品牌策略、年度計畫 | [詳細](terms/brand-positioning.md) |
| Slogan / Tagline | 品牌的招牌口號 | 「I'm Lovin' It」就是麥當勞的 Slogan | 品牌設計、廣告製作 | [詳細](terms/slogan-tagline.md) |
| Brief（需求簡報） | 把需求整理成文件給執行方 | 點餐單：大薯、可樂去冰 | 委外合作、活動企劃 | [詳細](terms/brief.md) |

## 數據分析類

> 看報表、開檢討會議時，這些詞你不能不認識。

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| GA（Google Analytics） | 免費看網站多少人來、從哪來 | 店門口的超強監視器 | 行銷部日常、產品會議 | [詳細](terms/ga.md) |
| Pixel（追蹤像素） | 埋在網頁上追蹤使用者行為 | 店裡每個角落的隱形偵探 | 廣告設定、跨部門溝通 | [詳細](terms/pixel.md) |
| 漏斗（Funnel） | 從看到→買單，每一步都流失人 | 夜市路過 1000 人，最後 10 人掏錢 | 行銷策略、轉換優化 | [詳細](terms/funnel.md) |
| 轉換率（Conversion Rate） | 來的人有多少比例做了你要的事 | 夜市攤位路過的人有幾個買 | 所有成效報表 | [詳細](terms/conversion-rate.md) |
| 跳出率（Bounce Rate） | 只看一頁就走的人佔多少 | 推開店門看一眼就走 | GA 報表、網站優化 | [詳細](terms/bounce-rate.md) |
| 熱力圖（Heatmap） | 用顏色顯示使用者最常點哪裡 | 天氣預報的溫度圖，紅色最熱 | UX 優化、頁面改版 | [詳細](terms/heatmap.md) |
| A/B Testing | 同時試兩個版本，數據決定哪個好 | 飲料店推兩種口味看誰賣更好 | 行銷優化、產品迭代 | [詳細](terms/ab-testing.md) |

## 行銷策略類

> 行銷的大方向怎麼規劃？用什麼方式觸及消費者？

| 詞彙 | 一句話解釋 | 比喻 | 你會在哪聽到 | 詳細說明 |
|------|-----------|------|-------------|---------|
| Campaign（活動企劃） | 為了特定目標規劃的一整套行銷活動 | 一場有主題的園遊會 | 「這波 Campaign 的 KPI 是什麼？」 | [詳細](terms/campaign.md) |
| SEM（搜尋引擎行銷） | 花錢在 Google 搜尋結果上買廣告 | 在黃頁上買版面 | 「SEM 的關鍵字出價太高了」 | [詳細](terms/sem.md) |
| Content Marketing（內容行銷） | 用有價值的內容吸引消費者，而不是硬推廣告 | 開烹飪教室賣廚具 | 「我們的 Content Marketing 策略要調整」 | [詳細](terms/content-marketing.md) |
| Viral Marketing（病毒式行銷） | 讓人主動分享、像病毒一樣擴散的行銷手法 | 同學間傳紙條，一個傳一個 | 「這支影片做到了 Viral」 | [詳細](terms/viral-marketing.md) |
| Media Buy（媒體採購） | 花錢買廣告版位或曝光機會 | 買電視廣告時段 | 「這個月 Media Buy 預算多少？」 | [詳細](terms/media-buy.md) |

### 數位廣告類

| 術語 | 比喻 |
|------|------|
| CPC | 夜市發傳單，有人接過去你才付錢 |
| CPM | 捷運站燈箱廣告，路人經過就算一次 |
| CPA | 房仲幫你賣房子，成交了才付佣金 |
| CTR | Line 群丟連結，看多少人真的點開 |
| ROAS | 花 100 買彩券中 500，回報 5 倍 |
| 曝光 | 名片放在櫃檯上，被人看到幾次 |
| 觸及率 | Line 群發訊息，有多少人看到（不重複） |
| 再行銷 | 百貨試穿沒買，回家被那件衣服的廣告追殺 |
| 歸因模型 | 球賽進球了，功勞到底算傳球的還是射門的？ |

### 社群行銷類

| 術語 | 比喻 |
|------|------|
| KOL / KOC | KOL 像百貨明星代言，KOC 像朋友在 Line 群推薦 |
| UGC | 去餐廳自己拍食物照打卡——不是店家請你的 |
| 互動率 | 老師上課問問題，多少學生舉手回答 |
| 粉絲經營 | 巷口早餐店記住你的口味、偶爾多給一片吐司 |

### 企劃提案類

| 術語 | 比喻 |
|------|------|
| TA | 夜市擺攤，招牌到底寫給誰看 |
| Persona | 把「25-35歲女性」變成「小美，28歲，住板橋，愛逛蝦皮」 |
| 品牌定位 | 提到平價時尚就想到 UNIQLO，那個腦中的位置 |
| Slogan / Tagline | 「I'm Lovin' It」讓你馬上想到麥當勞 |
| Brief | 點餐單——「大薯、可樂去冰」，不要只說「給我好吃的」 |

### 數據分析類

| 術語 | 比喻 |
|------|------|
| GA | 店門口裝的超強監視器，記錄客人的一切行為 |
| Pixel | 店裡每個角落放的隱形偵探，追蹤誰來過、看了什麼 |
| 漏斗 | 夜市路過 1000 人 → 停下來 200 → 試吃 50 → 買單 10 |
| 轉換率 | 經過攤位 100 人，3 人買東西，轉換率 3% |
| 跳出率 | 推開店門看一眼就走的人佔多少 |
| 熱力圖 | 天氣預報的溫度圖，紅色是最多人看的地方 |
| A/B Testing | 飲料店同時推兩種口味，哪個賣更好就上架哪個 |

### 行銷策略類

| 術語 | 比喻 |
|------|------|
| Campaign | 一場有主題的園遊會，所有攤位都圍繞同一個主題 |
| SEM | 在黃頁上買版面，讓找東西的人第一眼看到你 |
| Content Marketing | 開免費烹飪教室，順便賣廚具 |
| Viral Marketing | 同學間傳紙條，一個傳一個根本停不下來 |
| Media Buy | 買電視廣告時段，用錢換曝光機會 |
